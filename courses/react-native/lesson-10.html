.1 أنواع الأزرار في RN Button Types

في RN لا يوجد عنصر <button> واحد كـ HTML — بدلاً منه عندك 4 خيارات، كل منها لموقف مختلف:

المكوّنالاستخدامالتخصيصالحداثة
Button نماذج أولية سريعة محدود جداً قديم
TouchableOpacity الاستخدام اليومي (90%) مرن جداً شائع
TouchableHighlight تأثير تظليل عند الضغط متوسط أقل شيوعاً
Pressable تحكم كامل في حالات الضغط الأكثر مرونة ⭐ الأحدث

.2 Button — الزر البسيط Button Component

مكوّن Button المدمج بسيط لكن محدود — مفيد فقط للاختبار السريع:

🔘 Button البسيط
import { Button, View } from 'react-native';

<Button
  title="اضغط هنا"
  onPress={() => alert('مرحباً!')}
  color="#0ea5e9"      // لون الزر
  disabled={false}    // تعطيل الزر
/>

// ❌ لا يمكن تخصيص: الحجم، الشكل، الخط، الأيقونة...
// ✅ مناسب فقط للاختبار السريع
تجنّب Button في الإنتاج — يظهر بشكل مختلف كلياً على iOS وAndroid، ولا يمكن تنسيقه. استخدم TouchableOpacity أو Pressable دائماً.

.3 TouchableOpacity — الأشيع TouchableOpacity

هو المكوّن الأساسي لبناء الأزرار — يُلف أي محتوى ويُضيف تأثير تلاشٍ عند الضغط:

👆 TouchableOpacity — الاستخدامات
import { TouchableOpacity, Text, View, Image } from 'react-native';

// 1. زر نصي بسيط
<TouchableOpacity
  style={styles.btn}
  onPress={() => console.log('ضُغط')}
  activeOpacity={0.7}   // درجة الشفافية عند الضغط (0-1)
>
  <Text style={styles.btnText}>إرسال</Text>
</TouchableOpacity>

// 2. زر بأيقونة
<TouchableOpacity style={styles.iconBtn} onPress={handleSearch}>
  <Text>🔍</Text>
  <Text style={styles.iconBtnText}>بحث</Text>
</TouchableOpacity>

// 3. بطاقة قابلة للضغط
<TouchableOpacity
  style={styles.card}
  onPress={() => navigation.navigate('Details', { id })}
  onLongPress={() => showOptions()}
  delayLongPress={500}   // ms قبل تفعيل onLongPress
>
  <Image source={img} style={styles.cardImg} />
  <Text style={styles.cardTitle}>{title}</Text>
</TouchableOpacity>

// 4. زر معطّل
<TouchableOpacity
  style={[styles.btn, loading && styles.btnDisabled]}
  onPress={handleSubmit}
  disabled={loading}
>
  <Text>{loading ? 'جاري الإرسال...' : 'إرسال'}</Text>
</TouchableOpacity>

.5 Pressable — الأحدث والأقوى Pressable

أُضيف في RN 0.63 — يُعطيك تحكماً كاملاً في كل حالات الضغط:

TouchableOpacity
pressedopacity: 0.7
onPress✓
onLongPress✓
hover state✗
Pressable ⭐
pressedكامل التحكم
onPress✓
onLongPress✓
onPressIn/Out✓
⭐ Pressable — القوة الكاملة
import { Pressable, Text } from 'react-native';

// الطريقة الأساسية — style كـ function
<Pressable
  style={({ pressed }) => [
    styles.btn,
    pressed && styles.btnPressed,
  ]}
  onPress={() => console.log('ضُغط!')}
  onPressIn={() => console.log('بدأ الضغط')}
  onPressOut={() => console.log('انتهى الضغط')}
  onLongPress={() => console.log('ضغط طويل')}
>
  {({ pressed }) => (
    <Text style={[styles.text, pressed && styles.textPressed]}>
      {pressed ? '🔥 مضغوط!' : 'اضغط هنا'}
    </Text>
  )}
</Pressable>

// الأنماط
const styles = StyleSheet.create({
  btn: {
    backgroundColor: '#0ea5e9',
    padding: 14, borderRadius: 12,
    alignItems: 'center',
  },
  btnPressed: {
    backgroundColor: '#0284c7',   // أغمق عند الضغط
    transform: [{ scale: 0.97 }],    // يصغر قليلاً
  },
  text:        { color: 'white', fontWeight: '700' },
  textPressed: { opacity: 0.8 },
});
متى تستخدم Pressable بدل TouchableOpacity؟ عندما تحتاج تغيير نمط أو محتوى المكوّن بحسب حالة الضغط. مثلاً: الزر يصغر عند الضغط، أو النص يتغير، أو اللون يتحول.

.6 مكوّن زر قابل لإعادة الاستخدام Reusable Button

بدل كتابة نمط زر جديد في كل شاشة — ابنِ مكوّن AppButton مرة واحدة واستخدمه في كل مكان:

🏗️ AppButton — مكوّن قابل لإعادة الاستخدام
// components/AppButton.js
import { Pressable, Text, StyleSheet, ActivityIndicator } from 'react-native';

export function AppButton({
  title,
  onPress,
  variant = 'primary',  // 'primary' | 'outline' | 'ghost' | 'danger'
  size = 'md',         // 'sm' | 'md' | 'lg'
  loading = false,
  disabled = false,
  icon,
  fullWidth = false,
}) {
  return (
    <Pressable
      style={({ pressed }) => [
        s.base,
        s[variant],
        s[`size_${size}`],
        fullWidth && s.fullWidth,
        (disabled || loading) && s.disabled,
        pressed && !disabled && s.pressed,
      ]}
      onPress={onPress}
      disabled={disabled || loading}
    >
      {({ pressed }) => (
        <>
          {loading && (
            <ActivityIndicator
              size="small"
              color={variant === 'primary' ? 'white' : '#0ea5e9'}
            />
          )}
          {icon && !loading && <Text>{icon}</Text>}
          <Text style={[s.text, s[`text_${variant}`]]}>
            {loading ? 'جاري...' : title}
          </Text>
        </>
      )}
    </Pressable>
  );
}

const s = StyleSheet.create({
  base: {
    flexDirection: 'row', alignItems: 'center',
    justifyContent: 'center', gap: 8,
    borderRadius: 12,
  },
  // Variants
  primary: { backgroundColor: '#0ea5e9' },
  outline:  { backgroundColor: 'transparent', borderWidth: 2, borderColor: '#0ea5e9' },
  ghost:    { backgroundColor: 'transparent' },
  danger:   { backgroundColor: '#f43f5e' },
  // Sizes
  size_sm: { paddingVertical: 8,  paddingHorizontal: 14 },
  size_md: { paddingVertical: 12, paddingHorizontal: 20 },
  size_lg: { paddingVertical: 16, paddingHorizontal: 28 },
  // States
  disabled: { opacity: 0.5 },
  pressed:  { transform: [{ scale: 0.97 }] },
  fullWidth: { width: '100%' },
  // Text
  text:          { fontWeight: '700', fontSize: 15 },
  text_primary:  { color: 'white' },
  text_outline:  { color: '#0ea5e9' },
  text_ghost:    { color: '#64748b' },
  text_danger:   { color: 'white' },
});

.7 أنماط الأزرار الشائعة Common Patterns

🎨 أنماط تصميم الأزرار
const styles = StyleSheet.create({

  // 1. زر ممتلئ (Filled)
  filled: {
    backgroundColor: '#0ea5e9', borderRadius: 12,
    padding: 14, alignItems: 'center',
  },

  // 2. زر محدود (Outlined)
  outlined: {
    borderWidth: 2, borderColor: '#0ea5e9',
    borderRadius: 12, padding: 14, alignItems: 'center',
  },

  // 3. زر شفاف (Text/Ghost)
  ghost: {
    padding: 14, alignItems: 'center',
    // بدون خلفية أو حد
  },

  // 4. زر مستدير (FAB)
  fab: {
    width: 56, height: 56, borderRadius: 28,
    backgroundColor: '#0ea5e9',
    alignItems: 'center', justifyContent: 'center',
    position: 'absolute', bottom: 24, right: 24,
    elevation: 8,
  },

  // 5. زر أيقونة نصية (Icon + Text)
  iconBtn: {
    flexDirection: 'row', alignItems: 'center',
    gap: 8, padding: 12, borderRadius: 10,
    backgroundColor: '#f0f9ff',
  },
});

📝 التمارين العملية Exercises

تمرين 1: اختيار المكوّن

أي مكوّن تستخدم لكل حالة؟
أ) بطاقة منتج قابلة للضغط مع تأثير fade
ب) إغلاق لوحة المفاتيح عند الضغط خارج حقل النص
ج) زر يتغير نصه ولونه أثناء الضغط فعلاً

الحل:

أ) TouchableOpacity — تأثير fade الافتراضي
ب) TouchableWithoutFeedback — بدون أي تأثير بصري
ج) Pressable — التحكم الكامل في حالة الضغط

تمرين 2: زر Outlined

اكتب TouchableOpacity بنمط Outlined — حد أزرق 2px، لون خلفية شفاف، نص أزرق "تعرّف أكثر".

الحل:

  style={styles.outlineBtn}
  onPress={handleLearnMore}
  activeOpacity={0.7}
>
  <Text style={styles.outlineBtnText}>تعرّف أكثر</Text>
</TouchableOpacity>

تمرين 3: زر Loading

اكتب زراً يُظهر ActivityIndicator و"جاري الإرسال..." عند loading = true، ونص "إرسال" عادياً.

الحل: 
import { ActivityIndicator } from 'react-native';

function SubmitBtn({ loading, onPress }) {
  return (
    <TouchableOpacity
      style={[styles.btn, loading && { opacity: 0.7 }]}
      onPress={onPress}
      disabled={loading}
    >
      {loading && (
        <ActivityIndicator color="white" size="small" />
      )}
      <Text style={styles.btnText}>
        {loading ? 'جاري الإرسال...' : 'إرسال'}
      </Text>
    </TouchableOpacity>
  );
}
تمرين 4: Pressable مع تأثير

اكتب Pressable يُقلّص حجمه إلى 95% عند الضغط ويعود للحجم الطبيعي عند الرفع.

الحل: 
<Pressable
  style={({ pressed }) => ({
    backgroundColor: '#0ea5e9',
    padding: 14,
    borderRadius: 12,
    alignItems: 'center',
    transform: [{ scale: pressed ? 0.95 : 1 }],
  })}
  onPress={handlePress}
>
  <Text style={{ color: 'white', fontWeight: '700' }}>
    اضغط
  </Text>
</Pressable>

اختبر فهمك للأزرار في React Native!

1مكوّن Button المدمج في RN مرن ويمكن تخصيصه بسهولة.
2Pressable يقبل style كـ function تأخذ حالة pressed.
3activeOpacity في TouchableOpacity تتحكم في درجة شفافية الزر عند الضغط.
4TouchableWithoutFeedback يعطي تأثير تلاشٍ افتراضياً عند الضغط.
5disabled={true} في TouchableOpacity يمنع تفعيل onPress.
