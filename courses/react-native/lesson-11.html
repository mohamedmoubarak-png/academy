.1 TextInput — المدخل الأساسي TextInput Intro

TextInput هو المكوّن الوحيد في React Native لاستقبال النصوص من المستخدم — يقابل عنصر <input> و <textarea> في HTML معاً.

مثال على حقل نصي عادي
✓ صيغة صحيحة
النص مخفي مع secureTextEntry
في HTML عندك <input type="text">، <input type="email">، <textarea>... أما في RN فمكوّن واحد فقط: TextInput — الفرق يكون عبر الخصائص (props).

.2 الخصائص الأساسية Core Props

هذه أهم الخصائص التي ستستخدمها يومياً:

الخاصيةالنوعالوصفمثال

⌨️ TextInput — الاستخدام الأساسي
import { TextInput, View, Text } from 'react-native';

<TextInput
  style={styles.input}
  placeholder="أدخل اسمك..."
  placeholderTextColor="#94a3b8"
  value={name}
  onChangeText={setName}
  maxLength={50}
  returnKeyType="next"        // زر التالي في الكيبورد
  onSubmitEditing={() => emailRef.current?.focus()}
  autoCapitalize="words"      // تكبير أول حرف من كل كلمة
  autoCorrect={false}         // تعطيل التصحيح التلقائي
/>
placeholderTextColor — لا يتوفر في CSS عادي بل هو prop مستقل في RN. دائماً حدّد لون واضحاً مثل #94a3b8 بدل ترك اللون الافتراضي الرمادي الفاتح جداً.

.3 Controlled vs Uncontrolled Input Modes

أهم مفهوم في TextInput — هل React يتحكم في قيمة الحقل أم لا؟

🔴 Uncontrolled — لا تُوصى به في RN
// ❌ Uncontrolled — بدون value و onChangeText
// React لا يعرف ما كتبه المستخدم
<TextInput
  style={styles.input}
  placeholder="اكتب هنا..."
  defaultValue="قيمة افتراضية"  // للقراءة الأولى فقط
/>

// لا تعرف القيمة إلا بـ ref:
const inputRef = useRef(null);
const getValue = () => {
  // لا يوجد طريقة مباشرة كـ inputRef.current.value
  // يجب تتبّع القيمة يدوياً
};
✅ Controlled — المنهج الصحيح في RN
import { useState } from 'react';
import { TextInput, Text } from 'react-native';

function SearchBar() {
  const [query, setQuery] = useState('');

  return (
    <>
      <TextInput
        style={styles.input}
        placeholder="ابحث..."
        value={query}              // ✅ React يتحكم في القيمة
        onChangeText={setQuery}    // ✅ يُحدّث الـ state
      />
      <Text>كتبت: {query}</Text>  // ✅ دائماً محدّث
    </>
  );
}

// ✅ يمكنك الآن:
// - مسح الحقل: setQuery('')
// - ملء الحقل مسبقاً: setQuery('React Native')
// - التحقق من القيمة في أي وقت: query
// - تحريك التركيز بين حقول متعددة
🤔
تشبيه بسيط

المدخل Uncontrolled كورقة تكتب عليها — أنت لا تعرف ما فيها إلا لما تقرأها.
المدخل Controlled كشخص يملي عليك وأنت تكتب — أنت دائماً تعرف ما قاله.

خاصية keyboardType تُخبر نظام التشغيل أي لوحة مفاتيح تُظهر — يُحسّن تجربة المستخدم كثيراً:

🔢

📞

💰

🌐

<TextInput
  keyboardType="email-address"
  autoCapitalize="none"       // لا تكبير تلقائي
  autoCorrect={false}
  placeholder="example@email.com"
/>

// 2. حقل رقم الهاتف

  keyboardType="phone-pad"
  maxLength={10}
  placeholder="05XXXXXXXX"
/>

// 3. حقل السعر
<TextInput
  keyboardType="decimal-pad"
  placeholder="0.00"
/>

// returnKeyType — نص زر الإرسال
<TextInput returnKeyType="search"  />  // 🔍 بحث
<TextInput returnKeyType="next"    />  // التالي
<TextInput returnKeyType="done"    />  // تم
<TextInput returnKeyType="send"    />  // إرسال
<TextInput returnKeyType="go"      />  // اذهب
autoCapitalize له 4 قيم: "none" (بدون تكبير)، "sentences" (أول كل جملة)، "words" (أول كل كلمة)، "characters" (كل الحروف). للبريد والرابط استخدم دائماً "none".

.6 تنسيق TextInput Styling Inputs

التنسيق في RN يختلف عن HTML — لا outline ولا :focus pseudo-class. نُحاكي التركيز بـ onFocus و onBlur مع state:

🎨 حقل مع تأثير التركيز — الطريقة الاحترافية
import { useState } from 'react';
import { TextInput, View, Text, StyleSheet } from 'react-native';

function AppInput({ label, error, ...props }) {
  const [focused, setFocused] = useState(false);

  return (
    <View style={s.wrapper}>
      {label && <Text style={s.label}>{label}</Text>}
      <TextInput
        style={[
          s.input,
          focused && s.focused,   // حدود زرقاء عند التركيز
          error  && s.error,      // حدود حمراء عند الخطأ
        ]}
        onFocus={() => setFocused(true)}
        onBlur={() => setFocused(false)}
        placeholderTextColor="#94a3b8"
        {...props}
      />
      {error && <Text style={s.errorText}>{error}</Text>}
    </View>
  );
}

const s = StyleSheet.create({
  wrapper: { marginBottom: 16 },
  label:   { fontSize: 14, fontWeight: '700', color: '#475569', marginBottom: 6 },
  input: {
    borderWidth: 2,
    borderColor: '#e2e8f0',
    borderRadius: 12,
    padding: 14,
    fontSize: 15,
    backgroundColor: 'white',
    color: '#1e293b',
  },
  focused: {
    borderColor: '#0ea5e9',
    shadowColor: '#0ea5e9',   // iOS
    shadowOpacity: 0.2,
    shadowRadius: 6,
    elevation: 2,             // Android
  },
  error:     { borderColor: '#f43f5e' },
  errorText: { color: '#f43f5e', fontSize: 12, marginTop: 4, fontWeight: '600' },
});

.7 التحقق من المدخلات Form Validation

التحقق من المدخلات يحمي بياناتك ويُحسّن تجربة المستخدم — النمط الأكثر شيوعاً هو التحقق عند onBlur:

✅ نمط التحقق عند مغادرة الحقل
function LoginForm() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');
  const [errors, setErrors] = useState({});

  // دوال التحقق
  const validateEmail = (val) => {
    const re = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!val) return 'البريد مطلوب';
    if (!re.test(val)) return 'صيغة البريد غير صحيحة';
    return '';
  };

  const validatePassword = (val) => {
    if (!val) return 'كلمة المرور مطلوبة';
    if (val.length < 8) return '8 أحرف على الأقل';
    return '';
  };

  // التحقق عند مغادرة الحقل
  const handleBlur = (field) => {
    const validators = { email: validateEmail, password: validatePassword };
    const values = { email, password };
    const err = validators[field](values[field]);
    setErrors(prev => ({ ...prev, [field]: err }));
  };

  // التحقق الكامل عند الإرسال
  const handleSubmit = () => {
    const emailErr = validateEmail(email);
    const passErr  = validatePassword(password);
    if (emailErr || passErr) {
      setErrors({ email: emailErr, password: passErr });
      return;
    }
    // ✅ نجح التحقق — أرسل البيانات
  };

  return (
    <View>
      <AppInput
        label="البريد الإلكتروني"
        value={email}
        onChangeText={(t) => { setEmail(t); setErrors(e => ({ ...e, email: '' })); }}
        onBlur={() => handleBlur('email')}
        error={errors.email}
        keyboardType="email-address"
        autoCapitalize="none"
      />
      <AppInput
        label="كلمة المرور"
        value={password}
        onChangeText={(t) => { setPassword(t); setErrors(e => ({ ...e, password: '' })); }}
        onBlur={() => handleBlur('password')}
        error={errors.password}
        secureTextEntry
      />
    </View>
  );
}
نمط مفيد: امسح الخطأ فور بدء الكتابة (setErrors(e => ({...e, field: ''}))) — هذا يُشعر المستخدم بأن التطبيق يستجيب لتصحيحه، ويُعيد التحقق فقط عند مغادرة الحقل.

.8 مثال كامل: شاشة تسجيل Registration Screen

نجمع كل ما تعلمناه في شاشة تسجيل كاملة مع refs للانتقال بين الحقول:

🏗️ RegisterScreen — شاشة تسجيل كاملة
import { useRef, useState } from 'react';
import {
  View, Text, TextInput, TouchableOpacity,
  ScrollView, KeyboardAvoidingView, Platform, StyleSheet,
} from 'react-native';

export function RegisterScreen() {
  const [form, setForm] = useState({
    name: '', email: '', phone: '',
    password: '', confirm: '', bio: '',
  });
  const [errors, setErrors] = useState({});
  const [showPass, setShowPass] = useState(false);

  // refs للتنقل بين الحقول بزر "التالي"
  const emailRef   = useRef(null);
  const phoneRef   = useRef(null);
  const passRef    = useRef(null);
  const confirmRef = useRef(null);
  const bioRef     = useRef(null);

  const update = (field, val) =>
    setForm(f => ({ ...f, [field]: val }));

  const validate = () => {
    const e = {};
    if (!form.name.trim())          e.name = 'الاسم مطلوب';
    if (!/\S+@\S+\.\S+/.test(form.email)) e.email = 'بريد غير صحيح';
    if (form.password.length < 8)  e.password = '8 أحرف على الأقل';
    if (form.password !== form.confirm) e.confirm = 'كلمتا المرور غير متطابقتين';
    setErrors(e);
    return Object.keys(e).length === 0;
  };

  const handleRegister = () => {
    if (validate()) console.log('✅ تسجيل ناجح', form);
  };

  return (
    // KeyboardAvoidingView يمنع الكيبورد من إخفاء الحقول
    <KeyboardAvoidingView
      behavior={Platform.OS === 'ios' ? 'padding' : 'height'}
      style={{ flex: 1 }}
    >
      <ScrollView contentContainerStyle={s.container}>

        <Text style={s.title}>إنشاء حساب</Text>

        // الاسم
        <TextInput
          style={[s.input, errors.name && s.inputErr]}
          placeholder="الاسم الكامل"
          value={form.name} onChangeText={v => update('name', v)}
          returnKeyType="next"
          onSubmitEditing={() => emailRef.current?.focus()}
          autoCapitalize="words"
        />
        {errors.name && <Text style={s.err}>{errors.name}</Text>}

        // البريد
        <TextInput
          ref={emailRef} style={[s.input, errors.email && s.inputErr]}
          placeholder="البريد الإلكتروني"
          value={form.email} onChangeText={v => update('email', v)}
          keyboardType="email-address"
          autoCapitalize="none" autoCorrect={false}
          returnKeyType="next"
          onSubmitEditing={() => phoneRef.current?.focus()}
        />
        {errors.email && <Text style={s.err}>{errors.email}</Text>}

        // الهاتف
        <TextInput
          ref={phoneRef} style={s.input}
          placeholder="رقم الهاتف (اختياري)"
          value={form.phone} onChangeText={v => update('phone', v)}
          keyboardType="phone-pad" maxLength={10}
          returnKeyType="next"
          onSubmitEditing={() => passRef.current?.focus()}
        />

        // كلمة المرور
        <View style={s.passWrap}>
          <TextInput
            ref={passRef}
            style={[s.input, s.flex1, errors.password && s.inputErr]}
            placeholder="كلمة المرور"
            value={form.password} onChangeText={v => update('password', v)}
            secureTextEntry={!showPass}
            autoCapitalize="none"
            returnKeyType="next"
            onSubmitEditing={() => confirmRef.current?.focus()}
          />
          <TouchableOpacity
            style={s.eyeBtn} onPress={() => setShowPass(p => !p)}>
            <Text>{showPass ? '🙈' : '👁️'}</Text>
          </TouchableOpacity>
        </View>
        {errors.password && <Text style={s.err}>{errors.password}</Text>}

        // نبذة
        <TextInput
          ref={bioRef} style={[s.input, s.textArea]}
          placeholder="نبذة عنك (اختياري)"
          value={form.bio} onChangeText={v => update('bio', v)}
          multiline numberOfLines={4}
          textAlignVertical="top"
          maxLength={200}
        />
        <Text style={s.counter}>{form.bio.length}/200</Text>

        <TouchableOpacity style={s.btn} onPress={handleRegister}>
          <Text style={s.btnText}>إنشاء الحساب</Text>
        </TouchableOpacity>

      </ScrollView>
    </KeyboardAvoidingView>
  );
}
KeyboardAvoidingView — أهم مكوّن مع النماذج! يرفع المحتوى عندما يظهر الكيبورد حتى لا يُخفي الحقول. على iOS استخدم behavior="padding"، وعلى Android behavior="height" أو لا شيء.

تمرين 1: حقل البحث

اكتب حقل بحث Controlled مع زر "X" لمسح النص. الحقل يقبل أي نص، وزر المسح يظهر فقط عندما يكون هناك نص.

الحل: 
function SearchInput() {
  const [query, setQuery] = useState('');

  return (
    <View style={{ flexDirection: 'row', alignItems: 'center' }}>
      <TextInput
        style={[styles.input, { flex: 1 }]}

          <Text>✕</Text>
        </TouchableOpacity>

تمرين 2: حقل رقم الهاتف

اكتب TextInput لرقم الهاتف السعودي: لوحة مفاتيح phone-pad، حد أقصى 10 أرقام، يتحقق عند onBlur أن الرقم يبدأ بـ 05 وطوله 10 أرقام.

الحل: 
function PhoneInput() {
  const [phone, setPhone] = useState('');
  const [error, setError] = useState('');

  const validate = () => {
    if (!/^05\d{8}$/.test(phone)) {
      setError('رقم غير صحيح — يجب أن يبدأ بـ 05 ويكون 10 أرقام');
    } else {
      setError('');
    }
  };

  return (
    <View>
      <TextInput
        style={[styles.input, error && { borderColor: '#f43f5e' }]}
        placeholder="05XXXXXXXX"
        value={phone}
        onChangeText={(t) => { setPhone(t); setError(''); }}
        onBlur={validate}
        keyboardType="phone-pad"
        maxLength={10}
      />
      {error ? <Text style={{ color: '#f43f5e', fontSize: 12 }}>{error}</Text> : null}
    </View>
  );
}
تمرين 3: حقل كلمة مرور مع قوة

اكتب حقل كلمة مرور مع مؤشر القوة: أقل من 6 أحرف (ضعيف)، 6-10 (متوسط)، أكثر من 10 (قوي).

الحل: 
function getStrength(pass) {
  if (pass.length === 0)  return null;
  if (pass.length < 6)   return { label: 'ضعيف',   color: '#f43f5e', width: '33%' };
  if (pass.length < 10)  return { label: 'متوسط',  color: '#f59e0b', width: '66%' };
  return                  { label: 'قوي',    color: '#16a34a', width: '100%' };
}

function PasswordStrength() {
  const [pass, setPass] = useState('');
  const strength = getStrength(pass);

  return (
    <View>
      <TextInput
        style={styles.input}
        secureTextEntry
        value={pass}
        onChangeText={setPass}
        placeholder="كلمة المرور"
      />
      {strength && (
        <>
          <View style={{ height: 4, backgroundColor: '#e2e8f0', borderRadius: 2, marginTop: 6 }}>
            <View style={{ height: 4, width: strength.width, backgroundColor: strength.color, borderRadius: 2 }} />
          </View>
          <Text style={{ color: strength.color, fontSize: 12, marginTop: 4 }}>{strength.label}</Text>
        </>
      )}
    </View>
  );
}
تمرين 4: حقل متعدد الأسطر مع عداد

اكتب TextInput متعدد الأسطر لكتابة تعليق، بحد أقصى 200 حرف، مع عداد يظهر الأحرف المتبقية ويتحول للأحمر عند أقل من 20 حرفاً.

الحل: 
function CommentInput() {
  const [comment, setComment] = useState('');
  const MAX = 200;
  const remaining = MAX - comment.length;
  const isLow = remaining < 20;

  return (
    <View>
      <TextInput
        style={[styles.input, { height: 100, textAlignVertical: 'top' }]}
        placeholder="اكتب تعليقك..."
        value={comment}
        onChangeText={setComment}
        multiline
        maxLength={MAX}
        textAlignVertical="top"
      />
      <Text
        style={{
          color: isLow ? '#f43f5e' : '#94a3b8',
          fontSize: 12, textAlign: 'left', marginTop: 4,
        }}
      >
        متبقٍ: {remaining}
      </Text>
    </View>
  );
}
تمرين 5: التنقل بين الحقول

اكتب نموذج بـ 3 حقول (اسم، بريد، كلمة مرور) بحيث يُنقل التركيز تلقائياً للحقل التالي عند الضغط على زر "التالي" في الكيبورد، وعند آخر حقل يُرسل النموذج.

function MultiFieldForm() {
  const [form, setForm] = useState({ name: '', email: '', password: '' });
  const emailRef = useRef(null);
  const passRef  = useRef(null);

  return (
    <View>
      <TextInput
        style={styles.input}
        placeholder="الاسم"
        value={form.name}
        onChangeText={v => setForm(f => ({ ...f, name: v }))}
        returnKeyType="next"
        onSubmitEditing={() => emailRef.current?.focus()}
        autoCapitalize="words"
      />
      <TextInput
        ref={emailRef} style={styles.input}
        placeholder="البريد"
        value={form.email}
        onChangeText={v => setForm(f => ({ ...f, email: v }))}
        keyboardType="email-address"
        autoCapitalize="none"
        returnKeyType="next"
        onSubmitEditing={() => passRef.current?.focus()}
      />
      <TextInput
        ref={passRef} style={styles.input}
        placeholder="كلمة المرور"
        value={form.password}
        onChangeText={v => setForm(f => ({ ...f, password: v }))}
        secureTextEntry
        autoCapitalize="none"
        returnKeyType="done"
        onSubmitEditing={() => console.log('إرسال', form)}
      />
    </View>
  );
}
