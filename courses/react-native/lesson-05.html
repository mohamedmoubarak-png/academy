.1 نظرة عامة — المكونات الخمسة Core Components

كل واجهة في React Native مهما كانت معقدة، تتكون من هذه المكونات الخمسة الأساسية. تخيّلها كأدوات النجار الأساسية — منشار، مطرقة، مفك — تبني بها أي شيء.

📦
<View>
الحاوية الأساسية. تجمع المكونات وتُطبّق Flexbox. كل واجهة تبدأ بـ View.
<div> في HTML
📝
<Text>
أي نص في التطبيق يجب أن يكون داخل Text. تدعم تنسيق الخط واللون والحجم.
<p> <h1> <span>
🖼️
<Image>
لعرض الصور سواء من الجهاز أو من الإنترنت. تحتاج تحديد حجم دائماً.
<img> في HTML
📜
<ScrollView>
تُضيف إمكانية التمرير. للمحتوى الأطول من الشاشة. تدعم التمرير الرأسي والأفقي.
overflow: scroll
👆
<TouchableOpacity>
يجعل أي مكوّن قابلاً للضغط مع تأثير تلاشي بصري. الأساس لبناء الأزرار.
<button> <a>
قاعدة 90/10: 90% من الواجهات في RN تبنيها بهذه المكونات الخمسة فقط. باقي المكونات تأتي لاحقاً عند الحاجة.

.2 View — الحاوية الأساسية View Component

View هي الطوبة الأساسية في بناء واجهات RN. وحدها لا تعرض شيئاً مرئياً — لكنها تُنظّم وتُرتّب كل ما بداخلها.

📦 View الأساسية
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    {/* View خارجية — تملأ الشاشة كلها */}
    <View style={styles.screen}>

      {/* View داخلية — بطاقة */}
      <View style={styles.card}>
        <Text>محتوى البطاقة</Text>
      </View>

      {/* View أفقية — صف */}
      <View style={styles.row}>
        <Text>يسار</Text>
        <Text>يمين</Text>
      </View>

    </View>
  );
}

const styles = StyleSheet.create({
  screen: { flex: 1, backgroundColor: '#f0f9ff', padding: 16 },
  card: {
    backgroundColor: 'white', borderRadius: 12,
    padding: 16, marginBottom: 12,
    elevation: 3,                      // Android shadow
    shadowColor: '#000',               // iOS shadow
    shadowOffset: { width: 0, height: 2 },
    shadowOpacity: 0.1,
  },
  row: { flexDirection: 'row', justifyContent: 'space-between' },
});

الخصائص المهمة لـ View:

الخاصيةالنوعالاستخدام
styleobject | arrayالأنماط (مطلوب في الغالب)
onLayoutfunctionيُنفَّذ عند حساب أبعاد العنصر
accessiblebooleanلدعم إمكانية الوصول
testIDstringللاختبارات التلقائية
pointerEventsstringالتحكم في استقبال أحداث الضغط
Flexbox في View: View تستخدم Flexbox تلقائياً بـ flexDirection: 'column'. لصف أفقي: flexDirection: 'row'. للتوسيط: alignItems: 'center' و justifyContent: 'center'.

.3 Text — عرض النصوص Text Component

كل نص في RN — عنوان، فقرة، رقم، رمز — يجب أن يكون داخل <Text>. لا استثناءات!

📝 Text والخصائص المهمة
import { Text } from 'react-native';

// 1. نص عادي
<Text>مرحباً بالعالم</Text>

// 2. مع تنسيق
<Text style={{
  fontSize: 24,
  fontWeight: 'bold',
  color: '#0369a1',
  textAlign: 'center',
  letterSpacing: 1,
  lineHeight: 32,
}}>
  عنوان رئيسي
</Text>

// 3. تقليص النص الطويل (مفيد للبطاقات)
<Text
  numberOfLines={2}
  ellipsizeMode="tail"
>
  هذا نص طويل جداً سيتم اقتطاعه بعد سطرين...
</Text>

// 4. نص قابل للضغط
<Text onPress={() => alert('ضُغط!')}>
  اضغط هنا
</Text>

// 5. نصوص متداخلة (مزج أنماط)
<Text style={{ fontSize: 16 }}>
  مرحباً <Text style={{ color: '#0ea5e9', fontWeight: 'bold' }}>
    أحمد
  </Text>، كيف حالك؟
</Text>
الخاصيةالنوعالوظيفة
numberOfLinesnumberالحد الأقصى لعدد الأسطر
ellipsizeMode'tail' | 'head' | 'middle'موضع ... عند الاقتطاع
selectablebooleanالسماح بتحديد النص ونسخه
onPressfunctionحدث الضغط على النص
onLongPressfunctionحدث الضغط المطوّل
خطأ شائع جداً:
❌ <View>مرحباً</View> — تعطل التطبيق!
✅ <View><Text>مرحباً</Text></View> — صحيح!

.4 Image — عرض الصور Image Component

مكوّن Image يعرض الصور من مصدرين: صور محلية (داخل مشروعك) أو صور من الإنترنت (URL).

🖼️ طرق تحميل الصور
import { Image } from 'react-native';

// 1. صورة محلية (داخل المشروع)
import logo from './assets/logo.png';

<Image
  source={logo}                  // مصدر محلي
  style={{ width: 100, height: 100 }}
/>

// 2. صورة من الإنترنت
<Image
  source={{ uri: 'https://picsum.photos/200' }}
  style={{ width: 200, height: 200, borderRadius: 100 }}
/>

// 3. مع resizeMode (كيفية ملء المساحة)
<Image
  source={{ uri: 'https://picsum.photos/400/200' }}
  style={{ width: '100%', height: 150 }}
  resizeMode="cover"             // cover | contain | stretch | center
/>

// 4. مع صورة بديلة عند الخطأ
<Image
  source={{ uri: imageUrl }}
  defaultSource={require('./assets/placeholder.png')}
  style={{ width: 80, height: 80 }}
/>

resizeMode — كيفية ملء الصورة للمساحة:

القيمةالوصفمتى تستخدمها
coverتملأ المساحة وتقطع الزائدصور الغلاف والخلفيات
containتظهر كاملة داخل المساحةالشعارات والأيقونات
stretchتمتد لتملأ المساحة بالضبطنادراً — تشوّه الصورة
centerتتوسط دون تكبير أو تقليصالصور الصغيرة
مهم: Image في RN دائماً تحتاج width و height محددين في الـ style. بدونهما لن تظهر الصورة!
مثال — صور دائرية (الشائع في الصور الشخصية):
👤 صورة دائرية
<Image
  source={{ uri: 'https://picsum.photos/seed/user/200' }}
  style={{
    width: 80,
    height: 80,
    borderRadius: 40,          // نصف العرض/الارتفاع = دائرة
    borderWidth: 3,
    borderColor: '#0ea5e9',
  }}
  resizeMode="cover"
/>

.5 ScrollView — التمرير ScrollView Component

عندما يكون المحتوى أطول من ارتفاع الشاشة، تحتاج ScrollView. بدونها سيُقطع المحتوى الزائد ولن يرى المستخدم باقيه.

📜 ScrollView الأساسية
import { ScrollView, View, Text } from 'react-native';

// تمرير رأسي (الافتراضي)
<ScrollView
  contentContainerStyle={{ padding: 16 }}
  showsVerticalScrollIndicator={false}    // إخفاء شريط التمرير
>
  {/* أي عدد من العناصر */}
  <View style={{ height: 200, backgroundColor: '#dbeafe' }} />
  <View style={{ height: 200, backgroundColor: '#fce7f3' }} />
  <View style={{ height: 200, backgroundColor: '#d1fae5' }} />
</ScrollView>

// تمرير أفقي (مثل قوائم Netflix)
<ScrollView
  horizontal                                  // أفقي
  showsHorizontalScrollIndicator={false}
  contentContainerStyle={{ gap: 10, padding: 8 }}
>
  <View style={{ width: 140, height: 100, backgroundColor: '#dbeafe', borderRadius: 10 }} />
  <View style={{ width: 140, height: 100, backgroundColor: '#fce7f3', borderRadius: 10 }} />
  <View style={{ width: 140, height: 100, backgroundColor: '#d1fae5', borderRadius: 10 }} />
</ScrollView>
الخاصيةالنوعالوظيفة
horizontalbooleanتمرير أفقي بدل الرأسي
contentContainerStyleobjectنمط محتوى التمرير (للـ padding مثلاً)
showsVerticalScrollIndicatorbooleanإظهار/إخفاء شريط التمرير
onScrollfunctionحدث التمرير (للأنيميشن)
refreshControlelementتحديث بسحب الصفحة للأسفل
keyboardDismissModestringإغلاق لوحة المفاتيح عند التمرير
ScrollView vs FlatList: ScrollView تُحمّل كل المحتوى دفعة واحدة. لو عندك قائمة طويلة (50+ عنصر) استخدم FlatList (درس 12) لأداء أفضل.

.6 TouchableOpacity — الضغط والتفاعل TouchableOpacity

في React Native لا يوجد onClick. بدلاً منه نلف العناصر بـ TouchableOpacity الذي يعطي تأثير تلاشي جميل عند الضغط.

👆 TouchableOpacity — الأساس
import { TouchableOpacity, Text, View } from 'react-native';

// زر بسيط
<TouchableOpacity
  onPress={() => alert('تم الضغط!')}
  activeOpacity={0.7}           // 0-1: درجة الشفافية عند الضغط
  style={styles.button}
>
  <Text style={styles.buttonText}>اضغط هنا</Text>
</TouchableOpacity>

// بطاقة قابلة للضغط
<TouchableOpacity
  onPress={() => navigateTo('Details')}
  onLongPress={() => showOptions()}       // ضغط مطوّل
  style={styles.card}
>
  <Image source={...} style={styles.cardImg} />
  <Text style={styles.cardTitle}>عنوان</Text>
</TouchableOpacity>

// زر معطل (disabled)
<TouchableOpacity
  onPress={handleSubmit}
  disabled={isLoading}
  style={[styles.button, isLoading && styles.disabled]}
>
  <Text>{isLoading ? 'جاري الإرسال...' : 'إرسال'}</Text>
</TouchableOpacity>
🎨 تصميم زر احترافي
const styles = StyleSheet.create({
  button: {
    backgroundColor: '#0ea5e9',
    paddingVertical: 14,
    paddingHorizontal: 28,
    borderRadius: 999,            // حواف دائرية كاملة
    alignItems: 'center',
    shadowColor: '#0ea5e9',
    shadowOffset: { width: 0, height: 4 },
    shadowOpacity: 0.3,
    elevation: 6,
  },
  buttonText: {
    color: 'white', fontSize: 16, fontWeight: '700',
  },
  disabled: {
    backgroundColor: '#94a3b8', shadowOpacity: 0, elevation: 0,
  },
});
متى تستخدم ماذا؟
• TouchableOpacity — للأزرار، البطاقات، القوائم (أشيع خيار)
• Pressable — للتحكم الدقيق في حالات الضغط (أحدث)
• TouchableHighlight — لو تريد تغيير اللون بدل الشفافية

.7 مقارنة مع HTML HTML vs RN

لو قادم من HTML/CSS، هذه المقارنة ستوضح الصورة كاملاً دفعة واحدة:

🌐 HTML 📱 React Native
<div><View>
<p>, <h1>...<h6>, <span><Text>
<img><Image>
overflow: scroll<ScrollView>
<button>, <a><TouchableOpacity>
<input type="text"><TextInput>
<ul> <li><FlatList>
className="..."style={styles.name}
onClickonPress
font-size: 16pxfontSize: 16
background-colorbackgroundColor
margin-top: 10pxmarginTop: 10
display: flexافتراضي دائماً
flex-direction: rowflexDirection: 'row'
اختصار: كل خاصية CSS تتحول من kebab-case (font-size) إلى camelCase (fontSize) في RN. الأرقام بدون px. الأسماء كـ strings.

.8 محرر تفاعلي — UI Builder Live Editor

ابنِ واجهة كاملة باستخدام المكونات الخمسة. عدّل الكود وشاهد النتيجة مباشرة على شاشة الجوال:

📱 معاينة حية — المكونات الخمسة معاً View+Text+Image+Scroll+Touch 
import {
  View, Text, Image,
  ScrollView, TouchableOpacity,
} from 'react-native';

export default function App() {
  return (
    <ScrollView style={s.screen}>

      {/* View — الترويسة */}
      <View style={s.header}>
        <Text style={s.title}>
          🎓 أكاديمية البرمجة
        </Text>
      </View>

      {/* Image — صورة الغلاف */}
      <Image
        source={{ uri: bannerUrl }}
        style={s.banner}
        resizeMode="cover"
      />

      {/* Text — قسم */}
      <Text style={s.sectionTitle}>
        📚 آخر الدروس
      </Text>

      {/* TouchableOpacity — بطاقة قابلة للضغط */}
      <TouchableOpacity
        style={s.card}
        onPress={openLesson}
      >
        <Text style={s.cardTitle}>
          المكونات الأساسية
        </Text>
        <Text style={s.cardSub}>
          ⏱️ 45 دقيقة
        </Text>
      </TouchableOpacity>

    </ScrollView>
  );
}

📝 التمارين العملية Exercises

تمرين 1: المكوّن الصحيح

ما المكوّن المناسب لكل حالة؟
أ) قائمة تحتاج تمرير عمودي   ب) زر "أضف للسلة"   ج) صورة شخصية   د) نص "الإجمالي: 150 ريال"

الحل:

أ) <ScrollView>
ب) <TouchableOpacity>
ج) <Image> مع borderRadius دائري
د) <Text>

تمرين 2: إصلاح الكود

الكود التالي لن يُظهر الصورة. ما السبب وكيف تصلحه؟

<Image source={{ uri: 'https://picsum.photos/200' }} />
الحل:

Image لا تظهر بدون width و height!

<Image
  source={{ uri: 'https://picsum.photos/200' }}
  style={{ width: 200, height: 200 }}
/>
تمرين 3: ScrollView أفقي

اكتب ScrollView أفقي يعرض 4 بطاقات ملونة كل واحدة عرضها 120 وارتفاعها 80 بألوان مختلفة.

الحل: 
<ScrollView
  horizontal
  showsHorizontalScrollIndicator={false}
  contentContainerStyle={{ gap: 10, padding: 10 }}
>
  {['#dbeafe', '#fce7f3', '#d1fae5', '#fef3c7'].map((color, i) => (
    <View
      key={i}
      style={{
        width: 120, height: 80,
        backgroundColor: color,
        borderRadius: 12,
      }}
    />
  ))}
</ScrollView>
تمرين 4: زر متغيّر

اكتب TouchableOpacity يُغيّر نصه من "إضافة ❤️" إلى "تم الإضافة ✓" عند الضغط. (تلميح: useState)

الحل: 
import { useState } from 'react';
import { TouchableOpacity, Text } from 'react-native';

function FavoriteButton() {
  const [added, setAdded] = useState(false);

  return (
    <TouchableOpacity
      onPress={() => setAdded(!added)}
      style={{
        backgroundColor: added ? '#22c55e' : '#0ea5e9',
        padding: 12, borderRadius: 999,
      }}
    >
      <Text style={{ color: 'white', fontWeight: 'bold' }}>
        {added ? 'تم الإضافة ✓' : 'إضافة ❤️'}
      </Text>
    </TouchableOpacity>
  );
}
تمرين 5: بطاقة كاملة

اكتب مكوّن CourseCard يعرض: صورة الدورة، اسمها، عدد الدروس، وزر "ابدأ الآن" قابل للضغط.

الحل: 
function CourseCard({ title, lessons, imageUri, onPress }) {
  return (
    <View style={styles.card}>
      <Image
        source={{ uri: imageUri }}
        style={styles.image}
        resizeMode="cover"
      />
      <View style={styles.info}>
        <Text style={styles.title}
          numberOfLines={1}
        >{title}</Text>
        <Text style={styles.sub}>{lessons} درس</Text>
        <TouchableOpacity
          onPress={onPress}
          style={styles.btn}
        >
          <Text style={styles.btnText}>ابدأ الآن ▶️</Text>
        </TouchableOpacity>
      </View>
    </View>
  );
}
