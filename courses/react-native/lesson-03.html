.1 هيكل ملفات Expo Project Structure

بعد npx create-expo-app MyApp، ستجد هذه الملفات:

MyApp/ ├── App.js ← نقطة البداية (ابدأ هنا دائماً) ├── assets/ │ ├── adaptive-icon.png ← أيقونة Android │ ├── favicon.png ← أيقونة الويب │ ├── icon.png ← أيقونة التطبيق │ └── splash.png ← شاشة البداية ├── app.json ← إعدادات التطبيق ├── babel.config.js ← إعدادات المترجم ├── package.json ← الحزم والـ scripts └── package-lock.json ← إصدارات الحزم الدقيقة
قاعدة: في البداية ستعمل فقط على App.js. باقي الملفات لا تلمسها إلا عند الحاجة!

.2 ملف App.js سطراً بسطر App.js Explained

📄 App.js — شرح كل سطر
// 1️⃣ استيراد المكونات من react-native
import { StatusBar } from 'expo-status-bar';
import { StyleSheet, Text, View } from 'react-native';

// 2️⃣ المكوّن الرئيسي (اسمه App بالاتفاق)
export default function App() {

  // 3️⃣ return يُعيد JSX — واجهة التطبيق
  return (
    {/* View = صندوق/حاوية */}
    <View style={styles.container}>

      {/* Text = أي نص */}
      <Text>Open up App.js to start!</Text>

      {/* StatusBar = شريط الحالة (الوقت، البطارية) */}
      <StatusBar style="auto" />

    </View>
  );
}

// 4️⃣ StyleSheet.create = تعريف الأنماط
const styles = StyleSheet.create({
  container: {
    flex: 1,                      // اشغل كل الشاشة
    backgroundColor: '#fff',    // خلفية بيضاء
    alignItems: 'center',        // توسيط أفقي
    justifyContent: 'center',    // توسيط عمودي
  },
});
لاحظ: كل شيء في JSX يجب أن يكون داخل <View> واحدة على الأقل. View هي الحاوية الأساسية في RN مثل div في HTML.

.3 المكونات الأربعة الأساسية Core Components

كل واجهة في RN تتكون من هذه المكونات:

<View>
الحاوية الأساسية. تجمع المكونات معاً وتُطبّق Flexbox.
يشبه <div>
<Text>
لعرض أي نص. يجب أن يكون كل نص داخل Text مباشرة — لا يمكن وضع نص خارجها!
يشبه <p> و <h1>
<Image>
لعرض الصور سواء من الجهاز أو من الإنترنت. يحتاج تحديد حجم دائماً.
يشبه <img>
<ScrollView>
تتيح التمرير. استخدمها عندما المحتوى أطول من الشاشة.
يشبه overflow:scroll
🧩 استخدام المكونات الأربعة
import { View, Text, Image, ScrollView } from 'react-native';

export default function App() {
  return (
    <ScrollView>   {/* يتيح التمرير */}

      <View>
        <Text style={{ fontSize: 24, fontWeight: 'bold' }}>
          مرحباً بالعالم!
        </Text>

        <Image
          source={{ uri: 'https://picsum.photos/200' }}
          style={{ width: 200, height: 200 }}
        />
      </View>

    </ScrollView>
  );
}
خطأ شائع جداً: وضع نص مباشرة خارج <Text>
❌ <View>مرحبا</View> — خطأ!
✅ <View><Text>مرحبا</Text></View> — صح!

.4 StyleSheet — CSS في JavaScript Styling

في RN لا يوجد CSS. التنسيق يكون بـ JavaScript Objects:

CSSReact Native StyleSheet
font-size: 18pxfontSize: 18
background-color: redbackgroundColor: 'red'
margin-top: 10pxmarginTop: 10
border-radius: 8pxborderRadius: 8
font-weight: boldfontWeight: 'bold'
text-align: centertextAlign: 'center'
🎨 طريقتا كتابة الأنماط
// طريقة 1: Inline (مباشرة في JSX)
<Text style={{ color: 'blue', fontSize: 20 }}>
  نص أزرق
</Text>

// طريقة 2: StyleSheet.create (الأفضل)
const styles = StyleSheet.create({
  title: {
    color: 'blue',
    fontSize: 20,
    fontWeight: 'bold',
    textAlign: 'center',
  },
});

<Text style={styles.title}>نص أزرق</Text>
لماذا StyleSheet.create أفضل؟ يُحسّن الأداء لأن RN يُحسّب الأنماط مرة واحدة. أما Inline فتُحسَب في كل render.

.6 import و export في RN Modules

📦 أنواع الـ import
// 1. استيراد من react-native (named imports)
import { View, Text, Image, StyleSheet } from 'react-native';

// 2. استيراد من مكتبة خارجية (default import)
import React from 'react';

// 3. استيراد صورة محلية
import logo from './assets/icon.png';

// 4. export — لتصدير المكوّن
export default function App() { ... }
// أو:
const App = () => { ... };
export default App;
في React 17+ لا تحتاج import React from 'react' في كل ملف — Expo يضيفها تلقائياً!

.7 ملف app.json App Config

ملف app.json يتحكم في كيفية ظهور تطبيقك:

⚙️ app.json — الإعدادات المهمة
{
  "expo": {
    "name": "تطبيقي الأول",       // اسم التطبيق
    "slug": "my-first-app",      // معرّف فريد
    "version": "1.0.0",          // الإصدار
    "icon": "./assets/icon.png", // أيقونة التطبيق
    "splash": {
      "image": "./assets/splash.png", // شاشة البداية
      "backgroundColor": "#0ea5e9"
    },
    "android": {
      "package": "com.yourname.myapp"  // Package ID للـ Play Store
    },
    "ios": {
      "bundleIdentifier": "com.yourname.myapp"
    }
  }
}

📝 التمارين العملية Exercises

تمرين 1: المكافئات

اذكر المكوّن المقابل في RN لكل وسم HTML: <div>، <img>، وماذا تستخدم لو المحتوى أطول من الشاشة؟

الحل:

<div> → <View>
<img> → <Image>
محتوى أطول من الشاشة → <ScrollView>

تمرين 2: خطأ شائع

ما الخطأ في الكود التالي وكيف تصلحه؟

<View>مرحبا بالعالم</View>
الحل:

الخطأ: وضع نص مباشرة داخل View بدون Text.

// الصحيح:
<View>
  <Text>مرحبا بالعالم</Text>
</View>
تمرين 3: تحويل CSS إلى RN

حوّل هذا الكود CSS إلى StyleSheet في React Native:

.title { font-size: 24px; color: #0369a1; font-weight: bold; text-align: center; margin-top: 16px; }
الحل: 
const styles = StyleSheet.create({
  title: {
    fontSize: 24,
    color: '#0369a1',
    fontWeight: 'bold',
    textAlign: 'center',
    marginTop: 16,
  },
});
تمرين 4: ملف app.json

اكتب قسم expo في app.json لتطبيق اسمه "تطبيق الطقس" بـ slug: "weather-app" وإصدار 1.0.0.

الحل: 
{
  "expo": {
    "name": "تطبيق الطقس",
    "slug": "weather-app",
    "version": "1.0.0"
  }
}
تمرين 5: بناء مكوّن

اكتب مكوّن React Native يعرض نص "مرحباً [اسمك]!" بحجم 28 لون أزرق، ونص "مسار React Native" بحجم 16 لون رمادي تحته.

الحل: 
import { View, Text, StyleSheet } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.greeting}>مرحباً أحمد!</Text>
      <Text style={styles.sub}>مسار React Native</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: { flex: 1, alignItems: 'center', justifyContent: 'center' },
  greeting:  { fontSize: 28, color: '#0369a1', fontWeight: 'bold' },
  sub:       { fontSize: 16, color: '#64748b', marginTop: 8 },
});

اختبر فهمك لبنية مشروع Expo!

1يمكن وضع نص مباشرة داخل View بدون استخدام Text.
2StyleSheet.create أفضل من الـ Inline styles من ناحية الأداء.
3ملف App.js هو نقطة بداية تطبيق Expo ومنه يبدأ التحميل.
4في React Native، نكتب CSS في ملف .css منفصل مثل الويب.
5ScrollView تُستخدم عندما يكون المحتوى أطول من ارتفاع الشاشة.
