.1 ما هي تطبيقات الجوال؟ Mobile Apps

تطبيق الجوال هو برنامج يعمل مباشرة على هاتفك (Android أو iPhone). تختلف عن مواقع الويب لأنها:

  • تُثبَّت على الجهاز من Google Play أو App Store
  • تستخدم ميزات الجوال (الكاميرا، GPS، الإشعارات، البصمة...)
  • تعمل بدون اتصال إنترنت (offline mode)
  • أسرع وأكثر سلاسة من المواقع
📱 تخيل الفرق:

تطبيق Instagram على الجوال = تجربة سلسة، إشعارات، كاميرا مباشرة.
موقع instagram.com في المتصفح = أبطأ، بدون إشعارات، تجربة محدودة.

سوق تطبيقات الجوال بالأرقام:
🌍 +5 مليار مستخدم جوال
📲 +3.5 مليون تطبيق على Google Play
💰 +500 مليار $ حجم سوق التطبيقات سنوياً

هناك 3 طرق رئيسية لبناء تطبيق جوال:

الفكرة الذهبية: بدل ما تتعلم Swift للـ iPhone و Kotlin للأندرويد، تتعلم React Native مرة واحدة وتبني للاثنين! 🎯

.3 ما هو React Native؟ What is RN?

React Native هو إطار عمل (Framework) طوّرته شركة Meta (Facebook) سنة 2015، يتيح بناء تطبيقات جوال حقيقية باستخدام JavaScript و React.

🎬 تخيلها كـ "مترجم":

أنت تكتب كود JavaScript عادي بأسلوب React.
React Native يترجمه إلى مكونات أصلية (Native) على iOS و Android.
النتيجة: تطبيق حقيقي 100%، ليس مجرد موقع داخل تطبيق!

الفكرة الأساسية:
في الويب (React)في الجوال (React Native)
<div><View>
<p>   <h1><Text>
<img><Image>
<button><Pressable>
CSS منفصلStyleSheet داخل JS
الخلاصة: إذا تعرف React، فأنت على بُعد أسبوعين من إتقان React Native!

📝 التمارين العملية Exercises

تمرين 1: لماذا React Native؟

اذكر 3 أسباب رئيسية تجعل مطور JavaScript يختار React Native بدل تعلم Swift و Kotlin منفصلين.

الحل:

1. كود واحد لمنصتين: 80%+ من الكود يعمل على iOS و Android معاً
2. اللغة المعروفة: JavaScript بدلاً من تعلم لغتين جديدتين
3. سرعة التطوير: Hot Reload + مكتبات جاهزة + مجتمع ضخم

تمرين 2: المكافئ في RN

اكتب المكوّن المكافئ في React Native لكل من وسوم HTML التالية:

<div> · <p> · <img> · <button>

الحل:

<div> → <View>
<p> → <Text>
<img> → <Image>
<button> → <Pressable> أو <TouchableOpacity>

تمرين 3: شركات RN

اذكر 5 تطبيقات مشهورة من حياتك اليومية مبنية بـ React Native (كلياً أو جزئياً).

الحل:

Facebook · Instagram · Discord · Shopify · Pinterest · Uber Eats · Walmart · Coinbase · Microsoft Office · Bloomberg · Skype

تمرين 4: RN مقابل Flutter

اذكر فرقاً جوهرياً واحداً بين React Native و Flutter من ناحية اللغة المستخدمة.

الحل:

React Native يستخدم JavaScript (لغة منتشرة جداً تُستخدم في الويب أيضاً)، بينما Flutter يستخدم Dart (لغة جديدة من Google غير منتشرة خارج Flutter).

تمرين 5: جرّب Expo Snack

افتح snack.expo.dev في متصفحك، وغيّر النص "Hello world" إلى اسمك. التقط صورة للنتيجة.

الحل:

1. افتح snack.expo.dev
2. ابحث عن سطر <Text>Hello world!</Text>
3. غيّره إلى <Text>مرحباً، أنا [اسمك]!</Text>
4. شاهد النتيجة على الجوال المحاكي على اليسار 🎉
