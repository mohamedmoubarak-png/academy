.1 ما هي الدالة؟ What is a Function?

الدالة هي كتلة كود معاد الاستخدام — تعرّفها مرة واحدة وتستدعيها كلما احتجتها.

☕ تخيل ماكينة قهوة:

الماكينة مبرمجة مرة واحدة على "كيفية صنع القهوة". كل ما تضغط الزر، تنفّذ نفس التعليمات وتخرج لك قهوة. الدالة هكذا — تعرّفها مرة وتستدعيها متى شئت.

لماذا نستخدم الدوال؟
  • إعادة الاستخدام: لا تكرر نفس الكود — ضعه في دالة واستدعِها
  • التنظيم: تقسيم الكود الكبير إلى وحدات صغيرة مفهومة
  • سهولة الصيانة: تعدّل مكان واحد بدل مئة مكان
  • القابلية للاختبار: تختبر كل دالة بشكل مستقل

.2 تعريف الدوال واستدعاؤها Declaring & Calling

function greet(name) {← المعامل (parameter)
  return `مرحباً ${name}!`;← الإرجاع
}

greet('أحمد');← الاستدعاء مع الوسيطة (argument)
📋 أمثلة
// دالة بدون معاملات
function sayHello() {
  console.log('مرحباً بالعالم!');
}
sayHello();  // مرحباً بالعالم!

// دالة بمعاملين
function add(a, b) {
  return a + b;
}
console.log(add(3, 5));  // 8
Function Declaration vs Function Expression:
🔍 نوعان رئيسيان
// Declaration — يمكن استدعاؤها قبل تعريفها (Hoisting)
function square(n) { return n * n; }

// Expression — تُحفظ في متغير
const cube = function(n) { return n ** 3; };
Hoisting: Function Declarations ترتفع لأعلى الكود تلقائياً — يمكنك استدعاؤها قبل سطر تعريفها. أما Function Expressions فلا.

.3 المعاملات والقيم الافتراضية Parameters & Defaults

تستطيع إعطاء معاملات قيماً افتراضية تُستخدم لو لم يُمرَّر شيء:

⚙️ Default Parameters
function greet(name = 'زائر', lang = 'ar') {
  if (lang === 'ar') return `مرحباً ${name}!`;
  return `Hello ${name}!`;
}

greet('أحمد')        // "مرحباً أحمد!"
greet()             // "مرحباً زائر!"  ← قيمة افتراضية
greet('Sara', 'en') // "Hello Sara!"
Rest Parameters — عدد غير محدود من الوسائط:
📦 ...rest
function sum(...numbers) {
  return numbers.reduce((total, n) => total + n, 0);
}

sum(1, 2, 3)          // 6
sum(10, 20, 30, 40)   // 100

.4 return — إرجاع قيمة Return Statement

return تُنهي تنفيذ الدالة وترجع قيمة للمستدعي:

↩️ return
function max(a, b) {
  if (a > b) return a;
  return b;
}

const result = max(10, 25);
console.log(result);  // 25
الكود بعد return لا يُنفَّذ: 
function test() {
  return 'انتهيت';
  console.log('هذا لن يُطبع أبداً');  // dead code
}
دالة بدون return: ترجع undefined تلقائياً.
function double(n) { return n * 2 }
double(7)
14
double(double(3))
12

.5 Arrow Functions Arrow Functions ⇒

صيغة أحدث وأقصر لكتابة الدوال — الأكثر استخداماً في JS الحديث:

🏹 Arrow Function
// الصيغة التقليدية
function add(a, b) {
  return a + b;
}

// Arrow Function — مكافئة تماماً
const add = (a, b) => {
  return a + b;
};

// اختصار: جسم واحد بدون {} و return
const add = (a, b) => a + b;

// معامل واحد: بدون أقواس
const double = n => n * 2;

// بدون معاملات: أقواس فارغة
const hello = () => 'مرحباً!';
الحالةالصيغة
معاملان فأكثر(a, b) => a + b
معامل واحدn => n * 2
بدون معاملات() => 'hello'
جسم متعدد الأسطر(a, b) => { ... return x; }

.6 نطاق المتغيرات Scope

النطاق (Scope) يحدد أين يمكن الوصول للمتغير:

🔭 Global vs Local Scope
const global = 'أنا في كل مكان';  // Global scope

function test() {
  const local = 'أنا داخل الدالة فقط'; // Local scope
  console.log(global);  // ✅ يصل للـ global
  console.log(local);   // ✅ يصل للـ local
}

test();
console.log(global);  // ✅
console.log(local);   // ❌ ReferenceError!
القاعدة: متغير داخل دالة غير مرئي خارجها. لكن الدالة ترى المتغيرات في النطاق الأوسع منها.

.7 الدوال من الدرجة الأولى First-Class Functions

في JS، الدوال كائنات من الدرجة الأولى — يمكن تمريرها كوسائط أو إرجاعها من دوال أخرى:

🎯 تمرير دالة كوسيط
// تمرير دالة كمعامل (Callback)
function doTwice(fn) {
  fn();
  fn();
}

doTwice(() => console.log('مرحباً!'));
// مرحباً!
// مرحباً!
🔢 أمثلة عملية على Callbacks
const nums = [1, 2, 3, 4, 5];

// forEach — تنفّذ دالة على كل عنصر
nums.forEach(n => console.log(n));

// map — تحوّل كل عنصر
const doubled = nums.map(n => n * 2);
// [2, 4, 6, 8, 10]

// filter — تُصفّي العناصر
const evens = nums.filter(n => n % 2 === 0);
// [2, 4]

📝 التمارين العملية Exercises

تمرين 1: دالة المساحة

اكتب دالة area(width, height) تحسب مساحة المستطيل وتعيدها.

الحل: 
function area(width, height) {
  return width * height;
}
console.log(area(5, 10));  // 50
تمرين 2: Arrow Function

حوّل هذه الدالة إلى Arrow Function: function double(n) { return n * 2; }

الحل: 
const double = n => n * 2;
console.log(double(6));  // 12
تمرين 3: قيمة افتراضية

اكتب دالة power(base, exp = 2) تحسب القوة. لو لم يُمرَّر exp تستخدم 2.

الحل: 
function power(base, exp = 2) {
  return base ** exp;
}
console.log(power(3));      // 9  (3²)
console.log(power(2, 10));  // 1024
تمرين 4: دالة تحديد الحد الأقصى

اكتب دالة maxOf(...nums) تأخذ أي عدد من الأرقام وتعيد الأكبر بينها.

الحل: 
function maxOf(...nums) {
  return Math.max(...nums);
}
console.log(maxOf(3, 7, 1, 9, 4));  // 9
تمرين 5: دالة تصفية

استخدم filter و Arrow Function لاستخراج كل الأعداد الأكبر من 10 من المصفوفة [5, 12, 3, 18, 7, 25].

الحل: 
const nums = [5, 12, 3, 18, 7, 25];
const big  = nums.filter(n => n > 10);
console.log(big);  // [12, 18, 25]
