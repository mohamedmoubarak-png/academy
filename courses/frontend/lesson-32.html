.1 ما هو موقع Portfolio؟ Portfolio Website

موقع الـ Portfolio هو بطاقة تعريفك الرقمية — المكان الذي يرى فيه أصحاب العمل من أنت، ماذا تتقن، وماذا بنيت. إنه أقوى CV في مجال البرمجة.

💼 تخيل متجراً على الإنترنت:

موقع Portfolio هو متجرك الخاص على الويب. المنتج الذي تبيعه هو مهاراتك ومشاريعك، والزبون الأهم هو صاحب العمل أو العميل.

موقع Portfolio احترافي يضم:

  • قسم Hero — اسمك ومسماك الوظيفي وزر CTA
  • قسم عني (About) — من أنت، مهاراتك، معلوماتك
  • قسم المشاريع (Projects) — أفضل أعمالك مع روابط مباشرة
  • نموذج التواصل (Contact) — طريقة للوصول إليك

.4 Hero + Typewriter Hero Section

تأثير Typewriter يعرض مسمياتك الوظيفية بالكتابة والمسح تلقائياً — يجذب انتباه الزائر فور دخوله الموقع:

⌨️ Typewriter Animation
const roles = ["مطور Frontend", "مصمم UI/UX"];
let ri=0, ci=0, dl=false;
function type() {
  const w=roles[ri], el=document.getElementById("typed");
  if(!dl){ el.textContent=w.slice(0,++ci);
    if(ci===w.length){dl=true;setTimeout(type,1600);return;}
  } else {
    el.textContent=w.slice(0,--ci);
    if(ci===0){dl=false;ri=(ri+1)%roles.length;}
  }
  setTimeout(type, dl ? 55 : 95);
}

.6 Dark Mode CSS Variables

Dark Mode يعمل بتغيير CSS Variables عبر class واحدة على الـ body — بدون تكرار لأي لون:

🌙 Dark Mode CSS
/* Light Mode */
:root { --bg:#fff; --text:#1e293b; }

/* Dark Mode */
body.dark { --bg:#0f172a; --text:#e2e8f0; }

/* يتطبق تلقائياً في كل مكان */
body { background:var(--bg); color:var(--text); }
🌙 Dark Mode JS
let dark = localStorage.getItem("theme") === "dark";
document.getElementById("themeBtn").addEventListener("click", () => {
  dark = !dark;
  document.body.classList.toggle("dark", dark);
  localStorage.setItem("theme", dark ? "dark" : "light");
});

.7 Scroll Animations IntersectionObserver

IntersectionObserver هو الطريقة الحديثة لتشغيل animations عند ظهور العنصر في الشاشة — أفضل بكثير من scroll event للأداء:

👁️ IntersectionObserver
const io = new IntersectionObserver(entries => {
  entries.forEach(entry => {
    if(entry.isIntersecting)
      entry.target.classList.add("show");
  });
}, { threshold: 0 });

document.querySelectorAll(".fade-in").forEach(el => io.observe(el));
لماذا IntersectionObserver؟ لا يستهلك CPU عند كل pixel تمرير، بل يعمل فقط عندما يتغير وضع العنصر في viewport.

تحدي 5: أضف Animation للأرقام العداد

أضف أرقاماً متحركة في قسم "عني" مثل: "32 درس" و"3 مشاريع" تعد من 0 للرقم النهائي.

الفكرة: 
function countUp(el, target, duration = 1500) {
  let start = 0;
  const step = target / (duration / 16);
  const timer = setInterval(() => {
    start = Math.min(start + step, target);
    el.textContent = Math.floor(start);
    if (start >= target) clearInterval(timer);
  }, 16);
}
