.1 ما هو الحدث؟ What is an Event?

الحدث هو أي شيء يحدث في المتصفح — ضغطة زر، تحريك ماوس، كتابة نص، تحميل صفحة... JavaScript تستمع لهذه الأحداث وتستجيب لها.

🔔 تخيل جرس الباب:

لما حد يضغط الجرس (الحدث)، تسمعه (addEventListener) وتفتح الباب (الـ handler function). بدون الاستماع، الجرس يرن ولا أحد يسمعه.

🔔 المفهوم الأساسي
// العنصر          الحدث        الدالة المُنفَّذة
element.addEventListener('click', () => {
  console.log('تم الضغط!');
});

.2 addEventListener Adding Event Listeners

👂 addEventListener vs onclick
const btn = document.querySelector('#myBtn');

// ❌ الطريقة القديمة (تستبدل الـ handler السابق)
btn.onclick = () => console.log('مرحبا');

// ✅ addEventListener (يمكن إضافة عدة handlers)
btn.addEventListener('click', () => console.log('Handler 1'));
btn.addEventListener('click', () => console.log('Handler 2'));
// كلاهما ينفّذ عند الضغط

// إزالة الـ listener (يجب نفس الدالة المُسمّاة)
function handleClick() { console.log('click'); }
btn.addEventListener('click', handleClick);
btn.removeEventListener('click', handleClick);

.4 كائن الحدث The Event Object

عند استدعاء الـ handler، JS تمرر له تلقائياً كائناً يحتوي معلومات عن الحدث:

📦 event object
document.addEventListener('click', (event) => {
  console.log(event.type);      // 'click'
  console.log(event.target);    // العنصر الذي ضُغط عليه
  console.log(event.clientX);  // موضع الماوس X
  console.log(event.clientY);  // موضع الماوس Y
});

// حدث keyboard
document.addEventListener('keydown', (e) => {
  console.log(e.key);     // 'Enter', 'a', 'ArrowUp'...
  console.log(e.ctrlKey); // true لو Ctrl مضغوط
  console.log(e.code);    // 'KeyA', 'Enter'...
});
btn.addEventListener('click', e => console.log(e.target.textContent))

.5 preventDefault و stopPropagation Controlling Events

🚫 preventDefault
// منع السلوك الافتراضي
const form = document.querySelector('form');
form.addEventListener('submit', (e) => {
  e.preventDefault();  // منع إعادة تحميل الصفحة
  // عالج البيانات بنفسك
});

// منع فتح رابط
const link = document.querySelector('a');
link.addEventListener('click', (e) => {
  e.preventDefault();
  console.log('الرابط محظور!');
});
Event Bubbling — فقاعة الأحداث:
🫧 Bubbling و stopPropagation
// الضغط على child ينتشر لأعلى (parent ← grandparent...)
child.addEventListener('click', (e) => {
  e.stopPropagation();  // أوقف الانتشار
  console.log('child فقط');
});

parent.addEventListener('click', () => {
  console.log('لن يُطبع لو stopPropagation في الابن');
});

.6 Event Delegation Event Delegation

بدل إضافة listener لكل عنصر، أضفه للأب وتحقق من target — أقوى وأكفأ بكثير:

🎯 Event Delegation
// ❌ طريقة سيئة — listener على كل زر
document.querySelectorAll('.btn').forEach(btn => {
  btn.addEventListener('click', handler);
});

// ✅ Event Delegation — listener واحد على الأب
document.querySelector('#container')
  .addEventListener('click', (e) => {
    // تحقق من العنصر الذي ضُغط
    if (e.target.classList.contains('btn')) {
      console.log('زر:', e.target.textContent);
    }
    if (e.target.classList.contains('del')) {
      e.target.closest('.item').remove();
    }
  });
Event Delegation مفيد جداً لما تضيف عناصر ديناميكياً — الـ listener يعمل على العناصر الجديدة تلقائياً.

.7 أحداث Keyboard و Input Keyboard & Input Events

⌨️ أحداث الكتابة
const input = document.querySelector('input');

// يُطلق مع كل حرف يُكتب
input.addEventListener('input', (e) => {
  console.log(e.target.value);   // القيمة الحالية
});

// اكتشاف مفتاح Enter
input.addEventListener('keydown', (e) => {
  if (e.key === 'Enter') {
    console.log('Enter!', input.value);
  }
  if (e.key === 'Escape') {
    input.value = '';  // تفريغ عند Escape
  }
});

// اختصارات Ctrl+S
document.addEventListener('keydown', (e) => {
  if (e.ctrlKey && e.key === 's') {
    e.preventDefault();
    console.log('حفظ!');
  }
});

📝 التمارين العملية Exercises

تمرين 1: زر تغيير اللون

اكتب JS يغيّر لون خلفية الصفحة لأحمر عند الضغط على زر، ويعيدها لأبيض عند الضغط مرة أخرى.

الحل: 
const btn = document.querySelector('button');
btn.addEventListener('click', () => {
  document.body.style.background =
    document.body.style.background === 'red' ? 'white' : 'red';
});
تمرين 2: منع إرسال نموذج

اكتب JS يمنع إرسال أي نموذج ويطبع بدلاً من ذلك قيم الحقول في الـ console.

الحل: 
const form = document.querySelector('form');
form.addEventListener('submit', (e) => {
  e.preventDefault();
  const inputs = form.querySelectorAll('input');
  inputs.forEach(inp => console.log(inp.name, inp.value));
});
تمرين 3: بحث فوري

اكتب JS يُصفّي قائمة من العناصر لحظة الكتابة في حقل بحث — أخفِ العناصر التي لا تطابق النص.

الحل: 
const search = document.querySelector('#search');
const items  = document.querySelectorAll('.item');

search.addEventListener('input', (e) => {
  const q = e.target.value.toLowerCase();
  items.forEach(item => {
    const match = item.textContent.toLowerCase().includes(q);
    item.style.display = match ? '' : 'none';
  });
});
تمرين 4: Event Delegation

عندك قائمة ul بعناصر li. أضف listener واحد فقط على الـ ul يحذف أي li عند الضغط عليه.

الحل: 
document.querySelector('ul').addEventListener('click', (e) => {
  if (e.target.tagName === 'LI') {
    e.target.remove();
  }
});
تمرين 5: اختصار Ctrl+Enter

اكتب JS يستمع لحدث keydown على document ويطبع "إرسال!" فقط عند ضغط Ctrl + Enter معاً.

الحل: 
document.addEventListener('keydown', (e) => {
  if (e.ctrlKey && e.key === 'Enter') {
    console.log('إرسال!');
  }
});
