.1 ما هو ES6+؟ What is ES6+?

ES6 (ECMAScript 2015) كان أكبر تحديث في تاريخ JavaScript. أضاف عشرات المميزات الجديدة التي جعلت الكتابة أسرع وأوضح. والإصدارات بعده (ES7, ES8...) تضيف مميزات أصغر كل سنة.

📱 تخيل تحديث iOS:

ES5 كانت كالهاتف القديم — تعمل لكنها مرهقة. ES6+ هي كالنظام الحديث — نفس الجهاز لكن أسرع، أجمل، وأقل كوداً.

الدوالArrow Functions () =>
النصوصTemplate Literals `${}`
الاستخراجDestructuring
النشرSpread ... و Rest
الوحداتimport / export
هياكل بياناتMap و Set

.2 Destructuring Destructuring Assignment

استخرجنا قيماً من المصفوفات والكائنات في دروس سابقة — هنا نراجع ونعمّق:

📦 Destructuring متقدم
// Nested Destructuring
const user = {
  name: 'أحمد',
  address: { city: 'القاهرة', zip: '12345' }
};

const { name, address: { city } } = user;
console.log(city);  // 'القاهرة'

// في معاملات الدوال
function display({ name, age = 0 }) {
  return `${name} — ${age}`;
}

// Swap بدون متغير مؤقت
let a = 1, b = 2;
[a, b] = [b, a];
// a=2, b=1

.3 Spread و Rest ... Operator

نفس الرمز ... لكن بمعنيين حسب السياق:

🌊 Spread — نشر القيم
// نشر مصفوفة
const nums   = [1, 2, 3];
const more   = [0, ...nums, 4];  // [0,1,2,3,4]
const copy   = [...nums];         // نسخة

// نشر كائن
const base   = { x: 1, y: 2 };
const moved  = { ...base, x: 10 }; // {x:10, y:2}

// تمرير مصفوفة كوسائط
Math.max(...[3, 1, 9, 4])  // 9
🗃️ Rest — جمع الباقي
// في الدوال
function sum(first, ...rest) {
  return first + rest.reduce((a, b) => a + b, 0);
}
sum(1, 2, 3, 4)  // 10

// في Destructuring
const [head, ...tail] = [10, 20, 30];
// head=10, tail=[20,30]

const { name, ...rest2 } = { name:'أحمد', age:25, city:'القاهرة' };
// name='أحمد', rest2={age:25, city:'القاهرة'}

.4 Optional Chaining و Nullish Coalescing Safe Navigation

من أكثر المميزات الحديثة فائدةً في العمل اليومي:

🔗 Optional Chaining (?.) — ES2020
const user = { profile: { name: 'سارة' } };

// بدون ?. — يرمي error لو profile غير موجود
// user.address.city  ← TypeError!

// مع ?. — يعيد undefined بدلاً من error
user?.address?.city       // undefined (بدون خطأ)
user?.profile?.name       // 'سارة'

// مع المصفوفات والدوال
user?.friends?.[0]         // undefined
user?.greet?.()             // undefined (لو الدالة غير موجودة)
❓ Nullish Coalescing (??) — ES2020
// || يستخدم القيمة الثانية عند أي falsy value
0     || 'افتراضي'   // 'افتراضي' (لأن 0 falsy)
''    || 'افتراضي'   // 'افتراضي' (لأن '' falsy)

// ?? يستخدم الثانية فقط عند null/undefined
0     ?? 'افتراضي'   // 0 ✅ (لأنه ليس null/undefined)
''    ?? 'افتراضي'   // '' ✅
null  ?? 'افتراضي'   // 'افتراضي'

// مع Optional Chaining
const city = user?.address?.city ?? 'غير محدد';
قاعدة: استخدم ?? دائماً بدل || للقيم الافتراضية — أأمن وأدق.

.5 Logical Assignment &&= ||= ??=

⚡ Logical Assignment — ES2021
let a = null;
let b = 5;

// ??= — عيّن لو القيمة null/undefined
a ??= 'افتراضي';  // a = 'افتراضي'
b ??= 10;         // b = 5 (لم يتغير)

// ||= — عيّن لو القيمة falsy
let name = '';
name ||= 'زائر';   // name = 'زائر'

// &&= — عيّن لو القيمة truthy
let count = 5;
count &&= count + 1;  // count = 6

.6 Modules — import / export ES Modules

تقسيم الكود إلى ملفات منفصلة — كل ملف module مستقل:

📤 export — تصدير من الملف
// utils.js

// Named exports
export const PI = 3.14159;

export function add(a, b) {
  return a + b;
}

export const greet = name => `مرحباً ${name}`;

// Default export (واحد فقط لكل ملف)
export default class Calculator { ... }
📥 import — استيراد في ملف آخر
// app.js

// Named imports
import { PI, add, greet } from './utils.js';

// Default import
import Calculator from './utils.js';

// Import كل شيء تحت namespace
import * as Utils from './utils.js';
Utils.add(1, 2);
في المتصفح: أضف type="module" لوسم script:
<script type="module" src="app.js"></script>

.7 Map و Set Map & Set

🗺️ Map — قاموس متقدم
const map = new Map();

// إضافة (أي نوع كـ key)
map.set('name', 'أحمد');
map.set(42, 'رقم');
map.set({ id: 1 }, 'كائن');

// قراءة وبحث
map.get('name')       // 'أحمد'
map.has('name')       // true
map.size              // 3
map.delete('name')

// التجوال
for (const [key, val] of map) {
  console.log(key, '→', val);
}
🔵 Set — مجموعة بدون تكرار
const set = new Set([1, 2, 3, 2, 1]);
// Set {1, 2, 3} — حذف التكرارات تلقائياً!

set.add(4);
set.has(3);   // true
set.size;    // 4

// حيلة رائعة: إزالة التكرارات من مصفوفة
const unique = [...new Set([1,2,2,3,3])];
// [1, 2, 3]

📝 التمارين العملية Exercises

تمرين 1: Optional Chaining

عندك const data = { user: null }. اكتب سطراً يجلب data.user.name بأمان ويعيد "مجهول" لو غير موجود.

الحل: 
const data = { user: null };
const name = data?.user?.name ?? 'مجهول';
console.log(name);  // 'مجهول'
تمرين 2: إزالة التكرارات

استخدم Set لإزالة التكرارات من [5,3,5,1,3,2,1,2] وأعدها مصفوفة.

الحل: 
const unique = [...new Set([5,3,5,1,3,2,1,2])];
console.log(unique);  // [5,3,1,2]
تمرين 3: Spread لدمج كائنين

ادمج {name:'أحمد'} و {age:25, city:'القاهرة'} في كائن واحد وغيّر age إلى 26.

الحل: 
const merged = { ...{name:'أحمد'}, ...{age:25, city:'القاهرة'}, age: 26 };
// {name:'أحمد', age:26, city:'القاهرة'}
تمرين 4: Map لعدّ التكرارات

من المصفوفة ['أحمر','أزرق','أحمر','أخضر','أزرق','أحمر']، استخدم Map لعدّ كم مرة تكرر كل لون.

الحل: 
const colors = ['أحمر','أزرق','أحمر','أخضر','أزرق','أحمر'];
const count  = new Map();
colors.forEach(c => count.set(c, (count.get(c) ?? 0) + 1));
// Map { أحمر→3, أزرق→2, أخضر→1 }
تمرين 5: Rest في Destructuring

من const arr = [1,2,3,4,5]، استخرج العنصر الأول في متغير first والباقي في rest.

الحل: 
const [first, ...rest] = [1,2,3,4,5];
console.log(first); // 1
console.log(rest);  // [2,3,4,5]
