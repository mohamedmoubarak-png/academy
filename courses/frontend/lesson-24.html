.1 ما هو localStorage؟ What is localStorage?

localStorage هو مخزن بيانات في المتصفح — يحفظ البيانات على جهاز المستخدم وتبقى حتى بعد إغلاق المتصفح وإعادة تشغيله.

📒 تخيل دفتر ملاحظات شخصي:

كل مرة تفتح تطبيقك تجد ملاحظاتك موجودة. localStorage هو هذا الدفتر — يحفظ بياناتك على جهازك وتجدها في كل زيارة.

theme
"dark"
username
"أحمد"
cart
'[{"id":1,...}]'
lastVisit
"2024-01-15"
تخزّن فيه: إعدادات المستخدم، سلة التسوق، التقدم في التعلم، آخر بحث، الوضع الليلي...
لا تخزّن فيه: كلمات المرور، بيانات حساسة، بيانات ضخمة.

.2 العمليات الأساسية Basic Operations

💾 CRUD في localStorage
// ✅ حفظ (Create / Update)
localStorage.setItem('username', 'أحمد');
localStorage.setItem('age', '25');     // القيم دائماً String

// ✅ قراءة (Read)
const name = localStorage.getItem('username');
// 'أحمد'

const missing = localStorage.getItem('notExists');
// null (لو المفتاح غير موجود)

// ✅ حذف مفتاح (Delete)
localStorage.removeItem('age');

// ✅ مسح كل البيانات
localStorage.clear();

// ✅ عدد المفاتيح
localStorage.length;  // عدد العناصر المخزنة
localStorage.setItem('color', 'blue')
localStorage.getItem('color')

.3 تخزين الكائنات والمصفوفات Storing Objects & Arrays

localStorage يخزّن نصوصاً فقط. لتخزين كائنات ومصفوفات نستخدم JSON.stringify و JSON.parse:

🔄 JSON.stringify و JSON.parse
const user = { name: 'سارة', age: 22, skills: ['HTML', 'CSS'] };

// حفظ: كائن → نص
localStorage.setItem('user', JSON.stringify(user));

// قراءة: نص → كائن
const saved = JSON.parse(localStorage.getItem('user'));
console.log(saved.name);    // 'سارة'
console.log(saved.skills); // ['HTML', 'CSS']
✅ نمط آمن مع ??
// آمن لو المفتاح غير موجود
const data = JSON.parse(
  localStorage.getItem('todos') ?? '[]'
);
// لو 'todos' غير موجود → مصفوفة فارغة بدل null
JSON.parse على null يرمي خطأ! دائماً تحقق من وجود القيمة أو استخدم القيمة الافتراضية كما في المثال أعلاه.

.5 نمط مساعد لـ localStorage Storage Helper Pattern

بدل تكرار JSON.stringify/parse في كل مكان، ابنِ مساعداً صغيراً:

🛠️ Storage Helper
const storage = {
  get(key, fallback = null) {
    try {
      const val = localStorage.getItem(key);
      return val ? JSON.parse(val) : fallback;
    } catch { return fallback; }
  },
  set(key, value) {
    localStorage.setItem(key, JSON.stringify(value));
  },
  remove(key) {
    localStorage.removeItem(key);
  }
};

// استخدام نظيف
storage.set('user', { name: 'أحمد', age: 25 });
const user = storage.get('user', {});
const todos = storage.get('todos', []);  // [] كقيمة افتراضية

.6 حالات استخدام شائعة Common Use Cases

🌙 الوضع الليلي
// حفظ الوضع الليلي
function toggleDark() {
  const isDark = document.body.classList.toggle('dark');
  localStorage.setItem('darkMode', isDark);
}

// استعادة الوضع عند التحميل
if (localStorage.getItem('darkMode') === 'true') {
  document.body.classList.add('dark');
}
📝 حفظ قائمة مهام
let todos = JSON.parse(localStorage.getItem('todos') ?? '[]');

function addTodo(text) {
  todos.push({ text, done: false, id: Date.now() });
  localStorage.setItem('todos', JSON.stringify(todos));
}

function toggleTodo(id) {
  todos = todos.map(t =>
    t.id === id ? { ...t, done: !t.done } : t
  );
  localStorage.setItem('todos', JSON.stringify(todos));
}

📝 التمارين العملية Exercises

تمرين 1: حفظ وقراءة

احفظ قيمة 'dark' بالمفتاح 'theme' ثم اقرأها وأعدها.

الحل: 
localStorage.setItem('theme', 'dark');
const theme = localStorage.getItem('theme');
console.log(theme);  // 'dark'
تمرين 2: حفظ مصفوفة

احفظ مصفوفة ['HTML','CSS','JS'] في localStorage باسم 'skills' ثم استعدها كمصفوفة JS.

الحل: 
localStorage.setItem('skills', JSON.stringify(['HTML','CSS','JS']));

const skills = JSON.parse(localStorage.getItem('skills') ?? '[]');
console.log(skills);  // ['HTML', 'CSS', 'JS']
تمرين 3: عداد زيارات

اكتب كوداً يزيد عدد زيارات المستخدم بمقدار 1 في كل مرة يُشغَّل (يستخدم localStorage).

الحل: 
const visits = (parseInt(localStorage.getItem('visits')) || 0) + 1;
localStorage.setItem('visits', visits);
console.log(`زيارة رقم: ${visits}`);
تمرين 4: الوضع الليلي

اكتب دالة تُبدّل بين الوضع الليلي والعادي وتحفظ الاختيار في localStorage.

الحل: 
function toggleTheme() {
  const isDark = document.body.classList.toggle('dark');
  localStorage.setItem('theme', isDark ? 'dark' : 'light');
}

// تطبيق الثيم عند التحميل
if (localStorage.getItem('theme') === 'dark') {
  document.body.classList.add('dark');
}
تمرين 5: مسح كل البيانات

اكتب دالة logout() تحذف مفاتيح: user و token و settings من localStorage.

الحل: 
function logout() {
  ['user', 'token', 'settings'].forEach(key => {
    localStorage.removeItem(key);
  });
  console.log('تم تسجيل الخروج');
}
