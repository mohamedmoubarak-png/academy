.1 ما هو CSS Grid؟ What is CSS Grid?

CSS Grid هو نظام تخطيط ثنائي الأبعاد — يتحكم في الأعمدة (columns) والصفوف (rows) في نفس الوقت. في حين أن Flexbox يرتّب العناصر على محور واحد فقط، Grid يتحكم في الاتجاهين معاً.

🗺️ تخيل جدول Excel:

الـ Grid Container هو الجدول نفسه. الـ Grid Items هي الخلايا. بتحدد عدد الأعمدة وعدد الصفوف، وبعدين كل عنصر يأخذ مكانه — أو تحدد مكانه أنت بالضبط.

🚀 تفعيل Grid
.container {
  display: grid;
}

بمجرد ما تكتب display: grid على أي عنصر، كل أبناؤه المباشرون يصبحون Grid Items.

.2 تعريف الأعمدة والصفوف Columns & Rows

أهم خاصيتين في Grid هما grid-template-columns و grid-template-rows. بيهم تحدد عدد وأحجام الأعمدة والصفوف.

grid-template-columns
📐 أمثلة على تعريف الأعمدة
/* 3 أعمدة بعرض ثابت */
grid-template-columns: 200px 200px 200px;

/* 3 أعمدة متساوية بـ fr */
grid-template-columns: 1fr 1fr 1fr;

/* اختصار: repeat(3, 1fr) */
grid-template-columns: repeat(3, 1fr);

/* عمود ثابت + عمود مرن */
grid-template-columns: 250px 1fr;

/* 4 أعمدة غير متساوية */
grid-template-columns: 1fr 2fr 1fr 1fr;
grid-template-rows
📏 تعريف الصفوف
/* صفوف بأحجام ثابتة */
grid-template-rows: 80px 1fr 60px;

/* الصفوف تتحدد تلقائياً بالغالب */
/* لكن ممكن تتحكم فيها بـ: */
grid-auto-rows: 150px;    /* كل صف 150px */
grid-auto-rows: minmax(100px, auto); /* الحد الأدنى 100px */
grid-template-columns: 1fr 2fr 1fr — العمود الأوسط ضعف الباقين
1fr
2fr
1fr
تذكر: repeat(n, value) اختصار مريح — repeat(4, 1fr) = أربع أعمدة متساوية.

.3 وحدة fr والـ gap fr Unit & Gap

وحدة fr (Fraction) هي وحدة خاصة بـ Grid تعني "جزء من المساحة المتاحة".

🍕 تخيل بيتزا:

لو كتبت 1fr 1fr 1fr = كل عمود يأخذ ثلث البيتزا. لو كتبت 1fr 2fr = الأول يأخذ ثلث والثاني يأخذ ثلثين.

🍕 مثال على fr
/* المجموع 4 أجزاء: 1+2+1 */
grid-template-columns: 1fr 2fr 1fr;
/* النتيجة: 25% - 50% - 25% */
خاصية gap

تتحكم في المسافة بين الخلايا (بدل ما تستخدم margin).

↔️ gap
gap: 20px;              /* نفس المسافة أفقياً وعمودياً */
gap: 10px 20px;         /* صفوف 10px - أعمدة 20px */
row-gap: 10px;           /* بين الصفوف فقط */
column-gap: 20px;        /* بين الأعمدة فقط */
gap: 10px — مسافة بين الخلايا

.4 تحديد مكان العناصر Placing Items

بدل ما تترك Grid يرتب العناصر تلقائياً، تقدر تحدد مكان كل عنصر بالضبط.

📌 grid-column و grid-row
.item {
  grid-column: 1 / 3;   /* من العمود 1 للعمود 3 (يأخذ عمودين) */
  grid-row:    1 / 2;   /* من الصف 1 للصف 2 */
}

/* اختصار باستخدام span */
.item {
  grid-column: span 2;  /* يمتد على عمودين */
  grid-row:    span 2;  /* يمتد على صفين */
}
العنصر الأول يمتد على عمودين (grid-column: span 2)

.5 grid-template-areas Named Areas

من أقوى مميزات Grid — تعطي اسم لكل منطقة في التخطيط، والكود يصبح واضحاً ومقروءاً جداً.

🗺️ مثال: تخطيط موقع كامل
.layout {
  display: grid;
  grid-template-columns: 1fr 3fr;
  grid-template-rows:    80px 1fr 60px;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  gap: 12px;
  min-height: 100vh;
}

header  { grid-area: header;  }
aside   { grid-area: sidebar; }
main    { grid-area: main;    }
footer  { grid-area: footer;  }

.6 auto-fill و auto-fit Auto Columns

أقوى ميزة في Grid للتصميم المتجاوب — بدون Media Queries! الأعمدة تتكيّف تلقائياً مع حجم الشاشة.

🔄 auto-fill و auto-fit
/* أعمدة تلقائية لا تقل عن 200px */
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 16px;
}

/* الفرق: auto-fit يمدد العناصر لملء الفراغ */
.grid {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}
الخاصيةالسلوكمتى تستخدمها؟
auto-fillيملأ الصف بأعمدة فارغة لو المحتوى أقللما تريد الشبكة تحافظ على حجمها
auto-fitيمدد العناصر الموجودة لملء الصف كلهلما تريد العناصر تملأ المساحة دايماً
auto-fit, minmax(120px, 1fr) — جرب تضغط النافذة

.7 Grid vs Flexbox When to Use Which?

الاتنين قويين لكن لكل واحد مكانه المناسب.

المعيارFlexboxGrid
الأبعادأحادي — صف أو عمودثنائي — صف وعمود معاً
الاستخدام الأمثلعناصر في سطر (nav, buttons, cards صغيرة)تخطيط الصفحة كله أو قوائم cards
التحكم في المحتوىالمحتوى يتحكم في الحجمالتخطيط يتحكم في المحتوى
محاذاةسهل جداً (justify + align)مرن جداً (areas + spans)
💡 القاعدة العملية:
  • Flexbox = لما تيجي من المحتوى (كم عنصر؟ كيف يتوزعون؟)
  • Grid = لما تيجي من التخطيط (كم عمود؟ كم صف؟)

والحل الأجمل: استخدمهم معاً — Grid للتخطيط الخارجي، Flexbox للعناصر الداخلية.

📌 مثال على الاستخدام المشترك: 
/* تخطيط الصفحة بـ Grid */
.page {
  display: grid;
  grid-template-columns: 250px 1fr;
}

/* قائمة التنقل بـ Flexbox */
nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

📝 التمارين العملية Exercises

تمرين 1: شبكة بسيطة

اكتب CSS لعمل شبكة من 4 أعمدة متساوية بمسافة 16px بين كل خلية.

الحل: 
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 16px;
}
تمرين 2: عمود ثابت + محتوى

اعمل تخطيطاً بعمود sidebar ثابت عرضه 280px وباقي المساحة للمحتوى الرئيسي.

الحل: 
.layout {
  display: grid;
  grid-template-columns: 280px 1fr;
  gap: 20px;
}
تمرين 3: امتداد العناصر

في شبكة 3 أعمدة، اجعل أول عنصر يمتد على العمودين الأولين، والعنصر الأخير يمتد على الثلاثة أعمدة كلها.

الحل: 
.container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 12px; }
.first  { grid-column: span 2; }
.last   { grid-column: span 3; }
تمرين 4: تخطيط كامل بـ areas

استخدم grid-template-areas لعمل تخطيط موقع فيه: header أعلى بالعرض كله، sidebar على اليسار، main في الوسط والوسط، footer أسفل بالعرض كله.

الحل: 
.page {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: 70px 1fr 60px;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  gap: 10px;
  min-height: 100vh;
}
header { grid-area: header; }
aside  { grid-area: sidebar; }
main   { grid-area: main; }
footer { grid-area: footer; }
تمرين 5: شبكة متجاوبة بدون Media Queries

اعمل شبكة من بطاقات تتكيف تلقائياً — كل بطاقة لا تقل عن 220px، وتملأ المساحة المتاحة.

الحل: 
.cards-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 20px;
}
