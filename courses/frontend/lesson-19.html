.1 ما هو الكائن؟ What is an Object?

الكائن هو مجموعة من الخصائص (key: value) تصف شيئاً ما. يناسب تمثيل الكيانات الحقيقية كمستخدم، منتج، طلب...

🪪 تخيل بطاقة هوية:

بطاقة الهوية تحتوي على: الاسم، العمر، الجنسية، رقم البطاقة... كل معلومة لها اسم (key) وقيمة (value). هذا هو الكائن بالضبط.

{
  name: 'أحمد',← اسم الخاصية (key)
  age: 25,← قيمة رقمية
  isStudent: true,← قيمة boolean
  city: 'القاهرة'← آخر خاصية بدون فاصلة
}
المصفوفة vs الكائن: المصفوفة = قائمة مرتبة (index رقمي). الكائن = مجموعة خصائص (key نصي). استخدم المصفوفة للقوائم، والكائن لوصف كيان.

.2 إنشاء الكائنات والوصول Create & Access

🔧 إنشاء كائن
const person = {
  name:  'سارة',
  age:   22,
  city:  'الرياض',
  hobbies: ['قراءة', 'برمجة']  // مصفوفة داخل كائن

طريقتان للوصول للخصائص:
🎯 Dot vs Bracket Notation
// Dot Notation — الأشهر والأنظف
person.name      // 'سارة'
person.age       // 22
person.hobbies[0] // 'قراءة'

// Bracket Notation — لما الـ key متغير أو يحتوي مسافة
person['name']     // 'سارة'
const key = 'age';
person[key]       // 22 (مفيد مع المتغيرات)
person.name
'سارة'
person.hobbies.length

.3 إضافة وتعديل وحذف Add, Edit, Delete

✏️ تعديل الكائن
const car = { brand: 'Toyota', year: 2020 };

// تعديل خاصية موجودة
car.year = 2024;

// إضافة خاصية جديدة
car.color = 'أبيض';
car.isElectric = false;

// حذف خاصية
delete car.year;

// التحقق من وجود خاصية
'brand' in car       // true
'year'  in car        // false (محذوفة)
car.hasOwnProperty('color') // true
const والكائنات: const يمنع إعادة تعيين المتغير، لكن يسمح بتعديل محتوى الكائن. car = {} ← خطأ. car.color = 'أسود' ← مسموح.

.4 Methods — الدوال داخل الكائنات Object Methods

يمكن أن تكون قيمة الخاصية دالة — وتسمى حينها Method:

⚙️ Methods و this
const person = {
  name:  'أحمد',
  age:   25,

  // Method عادي
  greet() {
    return `مرحباً، أنا ${this.name}!`;
  },

  // Method يستخدم خصائص الكائن
  isAdult() {
    return this.age >= 18;
  }
};

person.greet()    // "مرحباً، أنا أحمد!"
person.isAdult()  // true
this داخل method يشير إلى الكائن نفسه. هو ما يجعل الدالة "تعرف" في أي كائن هي.

.5 Object Destructuring Destructuring

استخراج خصائص الكائن في متغيرات منفصلة بسطر واحد:

🎁 Destructuring
const user = { name: 'محمد', age: 30, city: 'جدة' };

// بدون Destructuring
const name = user.name;
const age  = user.age;

// مع Destructuring ✅
const { name, age, city } = user;

// إعادة تسمية
const { name: userName, age: userAge } = user;

// قيمة افتراضية
const { name, role = 'مستخدم' } = user;
// role = 'مستخدم' (لأنها غير موجودة في user)
📌 استخدام شائع جداً — في معاملات الدوال: 
// بدلاً من: function showUser(user) { user.name ... }
function showUser({ name, age, city = 'غير محدد' }) {
  return `${name} (${age}) — ${city}`;
}

showUser({ name: 'سارة', age: 25 });
// "سارة (25) — غير محدد"

.6 Spread و Object.assign Merge & Copy

📦 Spread للكائنات
const base    = { name: 'أحمد', age: 25 };
const extras  = { city: 'القاهرة', job: 'مطور' };

// نسخ كائن
const copy    = { ...base };

// دمج كائنين
const merged  = { ...base, ...extras };
// { name:'أحمد', age:25, city:'القاهرة', job:'مطور' }

// تحديث خاصية واحدة بدون تعديل الأصل
const updated = { ...base, age: 26 };
// { name:'أحمد', age:26 }
لماذا Spread بدل التعديل المباشر؟ يحافظ على الكائن الأصلي ولا يعدّله — مبدأ أساسي في البرمجة الحديثة (Immutability).

.7 دوال Object المدمجة Object Static Methods

🛠️ Object.keys, values, entries
const scores = { أحمد: 85, سارة: 92, محمد: 78 };

// الأسماء فقط
Object.keys(scores)
// ['أحمد', 'سارة', 'محمد']

// القيم فقط
Object.values(scores)
// [85, 92, 78]

// الأزواج [key, value]
Object.entries(scores)
// [['أحمد',85], ['سارة',92], ['محمد',78]]

// التجوال على الكائن بـ entries
for (const [name, score] of Object.entries(scores)) {
  console.log(`${name}: ${score}`);
}
🔒 Object.freeze — منع التعديل
const config = Object.freeze({
  API_URL: 'https://api.example.com',
  VERSION: '1.0'
});

config.API_URL = 'جديد';  // لن يتغير — مجمّد!

📝 التمارين العملية Exercises

تمرين 1: إنشاء كائن وعرضه

أنشئ كائن يمثّل سيارة بخصائص: brand، model، year، color. ثم اطبع كل خاصية.

الحل: 
const car = { brand: 'Toyota', model: 'Camry', year: 2022, color: 'أبيض' };
for (const [key, val] of Object.entries(car)) {
  console.log(`${key}: ${val}`);
}
تمرين 2: Destructuring

من الكائن {name:'نور', age:20, job:'مصممة'}، استخرج name و job فقط في متغيرات.

الحل: 
const { name, job } = { name: 'نور', age: 20, job: 'مصممة' };
console.log(name, job);  // نور مصممة
تمرين 3: تحديث بدون تعديل الأصل

لديك const user = {name:'أحمد', age:25}. أنشئ كائناً جديداً بنفس الخصائص لكن age = 26.

الحل: 
const user    = { name: 'أحمد', age: 25 };
const updated = { ...user, age: 26 };
console.log(updated);  // {name:'أحمد', age:26}
console.log(user);     // {name:'أحمد', age:25} ← لم يتغير
تمرين 4: Method

أضف method اسمها introduce لكائن person تطبع: "أنا [الاسم]، عمري [السن]".

الحل: 
const person = {
  name: 'سارة',
  age:  22,
  introduce() {
    return `أنا ${this.name}، عمري ${this.age}`;
  }
};
console.log(person.introduce());
تمرين 5: Object.values مع reduce

عندك {رياضيات:90, علوم:85, عربي:78}. احسب متوسط الدرجات باستخدام Object.values و reduce.

الحل: 
const grades = { رياضيات:90, علوم:85, عربي:78 };
const vals   = Object.values(grades);
const avg    = vals.reduce((s,n) => s+n, 0) / vals.length;
console.log(avg);  // 84.33...
