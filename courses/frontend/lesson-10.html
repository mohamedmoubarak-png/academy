.1 ما هو Flexbox؟ What is Flexbox?

Flexbox (Flexible Box) هو نظام تخطيط أحادي المحور — يرتّب العناصر أفقياً أو عمودياً بشكل مرن ومتجاوب.

📦 تخيل صف من الصناديق على رف:

الـ Container هو الرف. الـ Items هي الصناديق. Flexbox يتيح لك التحكم في: كيف تُرتَّب الصناديق على الرف؟ ماذا يحدث لو فاضت؟ كيف تتوزع المساحة بينها؟

🚀 تفعيل Flexbox
/* خطوة واحدة فقط! */
.container {
  display: flex;
}
/* الآن كل الأبناء المباشرين أصبحوا flex items */
مصطلحان أساسيان:
• Main Axis (المحور الرئيسي) — الاتجاه الذي تُرتَّب فيه العناصر (افتراضياً: أفقي)
• Cross Axis (المحور العرضي) — العمودي على الرئيسي

.2 خصائص الحاوية Container Properties

gapالمسافة بين العناصر16px | 1rem | 10px 20px

.5 flex-direction و flex-wrap Direction & Wrap

🔄 الاتجاه والتفاف
/* flex-direction: اتجاه المحور الرئيسي */
.container { flex-direction: row; }           /* أفقي (افتراضي) */
.container { flex-direction: column; }        /* عمودي ↓ */
.container { flex-direction: row-reverse; }   /* أفقي معكوس */
.container { flex-direction: column-reverse; }/* عمودي معكوس ↑ */

/* flex-wrap: ماذا يحدث لو العناصر تفيض؟ */
.container { flex-wrap: nowrap; }   /* كلها في سطر واحد (افتراضي) */
.container { flex-wrap: wrap; }     /* تنتقل للسطر التالي عند الفيضان */

/* gap: المسافة بين العناصر (أحسن من margin!) */
.container { gap: 16px; }          /* بين كل الاتجاهات */
.container { gap: 10px 20px; }     /* عمودي أفقي */
.container { row-gap: 10px; column-gap: 20px; }

/* flex-flow: اختصار لـ direction + wrap */
.container { flex-flow: row wrap; }
column للقوائم العمودية: استخدم flex-direction: column لعمل قوائم عمودية مع justify-content لتوزيع العناصر.

.6 خصائص العنصر Item Properties

📦 خصائص الـ Item
/* flex-grow: كم يأخذ من المساحة الزائدة؟ */
.item { flex-grow: 0; }    /* لا يكبر (افتراضي) */
.item { flex-grow: 1; }    /* يأخذ مساحة متساوية */
.main { flex-grow: 2; }    /* يأخذ ضعف الباقين */

/* flex-shrink: هل يصغر لو المساحة ضيقة؟ */
.item { flex-shrink: 1; }  /* يصغر (افتراضي) */
.logo { flex-shrink: 0; }  /* لا يصغر أبداً */

/* flex-basis: الحجم الابتدائي قبل الحساب */
.item { flex-basis: auto; }   /* حسب المحتوى (افتراضي) */
.item { flex-basis: 200px; }  /* ابدأ بـ 200px */
.item { flex-basis: 0; }      /* ابدأ من الصفر */

/* flex: اختصار grow + shrink + basis */
.item { flex: 1; }        /* = flex: 1 1 0 */
.item { flex: 0 0 200px; }/* عرض ثابت 200px */

/* align-self: تجاوز align-items للعنصر وحده */
.special { align-self: flex-start; }

/* order: ترتيب العرض (بدون تغيير HTML) */
.last-first { order: -1; } /* يظهر أول */
.first-last { order: 1; }  /* يظهر آخر */

.7 أنماط شائعة Common Patterns

🏗️ أنماط Flexbox عملية
/* 1. Navbar: لوجو يسار + روابط يمين */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 24px;
}

/* 2. توسيط كامل (أفقي + عمودي) */
.hero {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

/* 3. بطاقات متساوية العرض */
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}
.card { flex: 1 1 280px; }  /* ← كل بطاقة لا تقل عن 280px */

/* 4. Sidebar + Main (تخطيط جانبي) */
.layout { display: flex; gap: 20px; }
.sidebar { flex: 0 0 260px; }  /* عرض ثابت */
.main    { flex: 1; }           /* يأخذ باقي المساحة */

/* 5. Push to end (ادفع لليمين/اليسار) */
.nav-items { display: flex; }
.nav-login { margin-right: auto; } /* يُدفع لأقصى اليمين */

📝 التمارين

تمرين 1: توسيط عمودي وأفقي

اعمل div بارتفاع 300px يحتوي نص في المنتصف تماماً (أفقي وعمودي) باستخدام Flexbox.

الحل: 
.hero {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px;
  background: #f0f4ff;
  border-radius: 12px;
}

.hero h1 { color: #1e3a8a; font-size: 2rem; }
تمرين 2: Navbar كاملة

اعمل navbar فيها: لوجو يمين + روابط في الوسط + زر تسجيل يسار — كلها بـ Flexbox.

الحل: 
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 16px 24px;
  background: #1e3a8a;
  color: white;
}

.nav-logo  { font-weight: 700; font-size: 1.2rem; }
.nav-links { display: flex; gap: 20px; list-style: none; }
.nav-links a { color: white; text-decoration: none; }
.nav-btn   { background: white; color: #1e3a8a; border: none; padding: 8px 18px; border-radius: 20px; font-weight: 700; cursor: pointer; }
تمرين 3: بطاقات متساوية

اعمل صف من 3 بطاقات متساوية العرض تستخدم flex: 1. عند الشاشة الصغيرة تصبح عمودية بـ flex-wrap: wrap وflex-basis: 100%.

الحل: 
.cards { display: flex; flex-wrap: wrap; gap: 16px; }
.card  { flex: 1 1 280px; background: white; border-radius: 12px; padding: 20px; }
تمرين 4: تخطيط جانبي (Sidebar)

اعمل تخطيط فيه: sidebar عرضه 260px ثابت + main يأخذ باقي المساحة، بمسافة 20px بينهما.

الحل: 
.layout  { display: flex; gap: 20px; min-height: 100vh; }
.sidebar { flex: 0 0 260px; background: #f8fafc; padding: 20px; }
.main    { flex: 1; padding: 20px; }
تمرين 5: Footer بـ 3 أعمدة

اعمل footer بـ 3 أعمدة متساوية بـ Flexbox: "عن الموقع"، "روابط سريعة"، "تواصل معنا".

الحل: 
.footer {
  display: flex;
  gap: 40px;
  background: #1e293b;
  color: white;
  padding: 40px 24px;
  flex-wrap: wrap;
}

.footer-col { flex: 1 1 200px; }
.footer-col h3 { margin-bottom: 16px; font-size: 1rem; color: #cbd5e1; }
.footer-col ul { list-style: none; }
.footer-col li { margin-bottom: 8px; font-size: 0.9rem; opacity: 0.7; }

اختبر فهمك!

1justify-content يتحكم في المحور العرضي (Cross Axis).
2flex: 1 تجعل العناصر تتوزع بالتساوي في المساحة المتاحة.
3flex-wrap: wrap يسمح للعناصر بالانتقال للسطر التالي عند الفيضان.
4gap في Flexbox تحل محل padding للمسافات بين العناصر.
5للتوسيط الكامل (أفقي وعمودي) نحتاج justify-content: center وalign-items: center معاً.
