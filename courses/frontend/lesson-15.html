.1 ما هي الشروط؟ What are Conditions?

الشروط هي طريقة البرنامج لـاتخاذ القرار. "لو كان كذا، افعل كذا. وإلا، افعل كذا."

🚦 تخيل إشارة مرور:

لو الإشارة خضرا → امشِ.
لو الإشارة حمرا → وقِف.
لو الإشارة صفرا → تأهّب.

البرامج بتفكر بنفس الطريقة — كل قرار مبني على شرط.

هل age >= 18?
نعم ✅ ↓
اسمح بالدخول
لا ❌ ↓
ارفض الدخول

.2 if الأساسي Basic if

أبسط شكل للشرط — ينفّذ الكود فقط لو الشرط true:

✅ if بسيط
if (الشرط) {
  // كود ينفّذ لو الشرط true
}

// مثال:
const score = 85;

if (score >= 50) {
  console.log('ناجح! 🎉');
}
if (85 >= 50) { console.log('ناجح!') }
ناجح!
الأقواس الموجية {} مهمة! كل الكود بداخلها ينفّذ لو الشرط true. لو سطر واحد فقط، الأقواس اختيارية — لكن استخدمها دائماً لتجنب الأخطاء.

.3 if / else if / else

لما تريد تنفّذ كود مختلف لو الشرط false:

🔀 if / else
const age = 16;

if (age >= 18) {
  console.log('مسموح بالدخول ✅');
} else {
  console.log('ممنوع الدخول ❌');
}
// ← "ممنوع الدخول ❌"
📌 مثال: تحديد نوع العدد 
const num = 7;

if (num % 2 === 0) {
  console.log('زوجي');
} else {
  console.log('فردي');
}
// ← "فردي"

.4 else if — شروط متعددة else if

لما عندك أكثر من حالتين، تضيف else if بينهم:

📊 تقدير درجة
const score = 72;
let grade;

if (score >= 90) {
  grade = 'A — ممتاز';
} else if (score >= 80) {
  grade = 'B — جيد جداً';
} else if (score >= 70) {
  grade = 'C — جيد';
} else if (score >= 50) {
  grade = 'D — مقبول';
} else {
  grade = 'F — راسب';
}

console.log(grade);  // "C — جيد"
مهم: JS تفحص الشروط بالترتيب وتوقف عند أول شرط true. ترتيب الشروط مهم جداً.

.5 switch switch Statement

لما تقارن متغيراً بقيم ثابتة كثيرة، switch أنظف من if/else if المتراكمة:

🔀 switch/case
const day = 'الجمعة';

switch (day) {
  case 'السبت':
  case 'الأحد':
    console.log('عطلة! 🎉');
    break;

  case 'الجمعة':
    console.log('نصف يوم 😌');
    break;

  default:
    console.log('يوم عمل 💼');
}
// ← "نصف يوم 😌"
لا تنسَ break! بدون break، JS ستكمل تنفيذ كل الـ cases التالية حتى نهاية الـ switch — وهذا يسبب أخطاء غير متوقعة.

.6 Ternary Operator Shorthand Condition

اختصار لـ if/else في سطر واحد — مفيد جداً في التعيين السريع:

⚡ التعبير الثلاثي
// الشكل:
const result = الشرط ? قيمة_لو_true : قيمة_لو_false;

// مثال:
const age    = 20;
const status = age >= 18 ? 'بالغ' : 'قاصر';
console.log(status);  // "بالغ"

// مفيد في JSX / template literals:
const msg = `أنت ${age >= 18 ? 'بالغ' : 'قاصر'}`;
متى تستخدمه؟ لما الشرط بسيط وتريد تعيين قيمة. لو الكود داخل الشرط طويل أو معقد، استخدم if/else العادي.

.7 Truthy و Falsy في الشروط Truthy & Falsy

في JS، أي قيمة ممكن تستخدمها في شرط — مش لازم تكون boolean صريحة:

🔍 Truthy & Falsy
// Falsy — تعتبر false في الشروط:
if (0)         { } // لا يُنفَّذ
if ('')        { } // لا يُنفَّذ
if (null)      { } // لا يُنفَّذ
if (undefined) { } // لا يُنفَّذ
if (NaN)       { } // لا يُنفَّذ
if (false)     { } // لا يُنفَّذ

// Truthy — كل ما عدا ذلك:
if (1)          { } // يُنفَّذ
if ('hello')    { } // يُنفَّذ
if ([])         { } // يُنفَّذ (مصفوفة فارغة truthy!)
if ({})         { } // يُنفَّذ (كائن فارغ truthy!)
📌 استخدام عملي — التحقق من وجود قيمة: 
const username = '';   // فارغ

if (username) {
  console.log(`مرحباً ${username}`);
} else {
  console.log('الرجاء إدخال الاسم');
}
// ← "الرجاء إدخال الاسم" (لأن '' falsy)

📝 التمارين العملية Exercises

تمرين 1: هل الرقم موجب؟

اكتب شرطاً يطبع "موجب" لو الرقم أكبر من صفر، "سالب" لو أصغر، و"صفر" لو يساوي صفر.

الحل: 
const num = -5;

if (num > 0)      console.log('موجب');
else if (num < 0) console.log('سالب');
else               console.log('صفر');
تمرين 2: فصل السنة

استخدم switch لتحديد فصل السنة بناءً على رقم الشهر (1-12).

الحل: 
const month = 7;
let season;

switch (true) {
  case month >= 3 && month <= 5:  season = 'ربيع'; break;
  case month >= 6 && month <= 8:  season = 'صيف';  break;
  case month >= 9 && month <= 11: season = 'خريف'; break;
  default: season = 'شتاء';
}
console.log(season);  // "صيف"
تمرين 3: Ternary

اكتب سطراً واحداً يعيّن قيمة متغير label: "قصير" لو الطول أقل من 170، و"طويل" لو أكبر أو يساوي.

الحل: 
const height = 175;
const label  = height >= 170 ? 'طويل' : 'قصير';
console.log(label);  // "طويل"
تمرين 4: التحقق من الدخول

اكتب كوداً يتحقق: لو username موجود (truthy) و password يساوي '1234'، اطبع "مرحباً". وإلا اطبع "بيانات خاطئة".

الحل: 
const username = 'أحمد';
const password = '1234';

if (username && password === '1234') {
  console.log(`مرحباً ${username}!`);
} else {
  console.log('بيانات خاطئة');
}
تمرين 5: حاسبة خصم

لو مبلغ الشراء amount >= 500 → خصم 20%. لو >= 200 → خصم 10%. وإلا لا خصم. احسب السعر النهائي.

الحل: 
const amount = 350;
let discount;

if (amount >= 500)      discount = 0.20;
else if (amount >= 200) discount = 0.10;
else                     discount = 0;

const final = amount * (1 - discount);
console.log(`السعر النهائي: ${final}`);  // 315
