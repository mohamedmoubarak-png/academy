.1 ما هو DOM؟ What is the DOM?

DOM اختصار لـ Document Object Model — هو تمثيل شجري لصفحة HTML في الذاكرة. JavaScript تستخدمه للوصول لأي عنصر في الصفحة وتعديله.

🌳 تخيل شجرة عائلة:

document هو الجد الأكبر. html هو الأب. head و body أبناؤه. وكل ما بداخلهم أحفاد. JavaScript تستطيع الوصول لأي فرد في هذه العائلة وتعديله.

document
  └── <html>
      ├── <head>
      │    └── <title> "أكاديميتي"
      └── <body>
           ├── <h1 id="title"> "مرحباً"
           └── <p class="text"> "فقرة"

.2 اختيار العناصر Selecting Elements

🎯 طرق الاختيار
// بالـ ID — يعيد عنصر واحد
const title = document.getElementById('title');

// بالـ CSS selector — أول عنصر مطابق
const btn   = document.querySelector('.btn');
const h1    = document.querySelector('h1');
const inp   = document.querySelector('#myInput');

// كل العناصر المطابقة — NodeList
const items = document.querySelectorAll('.item');
const paras = document.querySelectorAll('p');

// التجوال على NodeList
items.forEach(item => console.log(item));
الدالةتعيدتستخدم مع
getElementById(id)عنصر واحدID فقط
querySelector(sel)أول عنصرأي CSS selector
querySelectorAll(sel)NodeList (كلهم)أي CSS selector
القاعدة العملية: استخدم querySelector و querySelectorAll دائماً — أقوى وأمرن من getElementById.

.3 قراءة وتعديل المحتوى Reading & Changing Content

📝 innerHTML, textContent, innerText
const el = document.querySelector('#box');

// قراءة
el.textContent  // النص فقط بدون HTML
el.innerHTML    // المحتوى كاملاً بما فيه HTML

// تعديل
el.textContent = 'نص جديد';
el.innerHTML   = '<strong>نص غامق</strong>';

// قيمة الـ input
const inp = document.querySelector('input');
inp.value        // قراءة ما كتبه المستخدم
inp.value = ''   // تفريغ الحقل
innerHTML vs textContent: استخدم textContent لو تضع نصاً عادياً — أأمن وأسرع. استخدم innerHTML فقط لو تحتاج إدراج HTML فعلياً وتأكد من سلامة البيانات (XSS).

.4 تعديل CSS والـ Classes Styles & Classes

🎨 style و classList
const box = document.querySelector('.box');

// تعديل style مباشر (للتغييرات الديناميكية)
box.style.color      = 'red';
box.style.fontSize   = '20px';     // camelCase
box.style.display    = 'none';     // إخفاء
box.style.display    = 'block';    // إظهار

// classList — الطريقة الأفضل
box.classList.add('active');      // إضافة class
box.classList.remove('hidden');   // إزالة class
box.classList.toggle('dark');    // تبديل (يضيف لو غير موجود، يحذف لو موجود)
box.classList.contains('active') // true/false
classList.toggle مفيد جداً لزر الإضاءة/الظلام أو أي عنصر بين حالتين.

.5 إنشاء وإضافة وحذف العناصر Create, Append, Remove

🏗️ createElement و append
// إنشاء عنصر جديد
const li = document.createElement('li');
li.textContent = 'عنصر جديد';
li.classList.add('item');

// إضافة للصفحة
const list = document.querySelector('ul');
list.append(li);       // في النهاية
list.prepend(li);      // في البداية

// إضافة HTML مباشرة (بدون إنشاء)
list.innerHTML += '<li>عنصر</li>';

// حذف عنصر
li.remove();

// حذف كل الأبناء
list.innerHTML = '';
📌 نمط شائع جداً — بناء قائمة من مصفوفة: 
const fruits = ['تفاح', 'موز', 'عنب'];
const ul     = document.querySelector('ul');

ul.innerHTML = fruits
  .map(f => `<li>${f}</li>`)
  .join('');

.6 الخصائص (Attributes) Attributes

🔧 getAttribute و setAttribute
const img  = document.querySelector('img');
const link = document.querySelector('a');

// قراءة attribute
img.getAttribute('src')     // 'photo.jpg'
img.getAttribute('alt')     // 'صورة'

// تعديل attribute
img.setAttribute('src', 'new.jpg');
link.setAttribute('href', 'https://example.com');

// حذف attribute
img.removeAttribute('alt');

// Shorthand للخصائص الشائعة
img.src = 'new.jpg';    // مكافئ لـ setAttribute
link.href = '...';
inp.disabled = true;   // تعطيل input

.7 التنقل في شجرة DOM DOM Traversal

🌲 التنقل بين العناصر
const el = document.querySelector('.item');

// العنصر الأب
el.parentElement

// الأبناء
el.children           // HTMLCollection
el.firstElementChild
el.lastElementChild

// الإخوة
el.nextElementSibling
el.previousElementSibling

// closest — ابحث في الآباء
el.closest('.card')    // أقرب أب بـ class="card"
closest() مفيد جداً مع Event Delegation — ستتعلمها في درس الأحداث.

📝 التمارين العملية Exercises

تمرين 1: تغيير النص

اكتب JS يغيّر نص أي عنصر بـ id="title" إلى "أهلاً بالعالم!".

الحل: 
document.getElementById('title').textContent = 'أهلاً بالعالم!';
تمرين 2: تبديل class

اكتب JS يضيف class "active" لعنصر لو لم يكن موجوداً، ويحذفه لو كان موجوداً.

الحل: 
const el = document.querySelector('.box');
el.classList.toggle('active');
تمرين 3: بناء قائمة

من المصفوفة ['HTML','CSS','JavaScript']، ابنِ قائمة ul بعناصر li وأضفها للصفحة.

الحل: 
const items = ['HTML', 'CSS', 'JavaScript'];
const ul    = document.createElement('ul');
ul.innerHTML = items.map(i => `<li>${i}</li>`).join('');
document.body.append(ul);
تمرين 4: تعديل خاصية

اكتب JS يغيّر src لأول صورة في الصفحة إلى 'new-image.jpg' ويضبط alt إلى 'صورة جديدة'.

الحل: 
const img = document.querySelector('img');
img.src = 'new-image.jpg';
img.alt = 'صورة جديدة';
تمرين 5: إخفاء وإظهار

اكتب دالة toggle(id) تخفي العنصر لو كان ظاهراً وتظهره لو كان مخفياً.

الحل: 
function toggle(id) {
  const el = document.getElementById(id);
  el.style.display = el.style.display === 'none' ? 'block' : 'none';
}
