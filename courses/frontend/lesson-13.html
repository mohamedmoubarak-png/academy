.1 ما هو JavaScript؟ What is JavaScript?

تذكر ثلاثي الويب من الدرس الأول؟ HTML = الهيكل، CSS = الشكل، وJavaScript = الحياة. JS هي لغة البرمجة الوحيدة التي يفهمها المتصفح مباشرة.

🤖 تخيل روبوت:

HTML بنى جسم الروبوت. CSS رسم شكله وألوانه. JavaScript هي البرنامج اللي بيخلي الروبوت يتحرك، يسمع، يتكلم، ويتفاعل.

JavaScript تقدر بيها:
  • تعديل المحتوى: تغيير النصوص والصور بدون إعادة تحميل الصفحة
  • الاستجابة للمستخدم: لما يضغط زر، يكتب في حقل، يحرك الماوس
  • التواصل مع السيرفر: جلب بيانات من API بدون تحديث الصفحة
  • التخزين المحلي: حفظ بيانات في المتصفح
  • بناء تطبيقات كاملة: مواقع تفاعلية، ألعاب، تطبيقات موبايل
JavaScript في كل مكان: اليوم JS تشتغل على السيرفر (Node.js)، الموبايل (React Native)، حتى الـ AI وقواعد البيانات. تعلمها = قدرة على بناء أي شيء.

.2 إزاي تكتب JS؟ How to Write JS?

ثلاث طرق لإضافة JavaScript لصفحتك:

الطريقة 1: داخل وسم script (inline)
📄 JS داخل HTML
<body>
  <h1>مرحباً</h1>

  <!-- JS في آخر الـ body دائماً -->
  <script>
    console.log('أهلاً يا عالم!');
  </script>
</body>
الطريقة 2: ملف خارجي (الأفضل)
📁 ربط ملف JS خارجي
<!-- في آخر الـ body -->
<script src="script.js"></script>
لماذا في آخر body؟ عشان المتصفح يكون حمّل HTML كلها الأول قبل ما JS تحاول تتعامل معاها. لو حطيتها في الـ head ممكن JS تطلب عنصر مش موجود بعد.
أين أشوف النتائج؟

افتح المتصفح، اضغط F12 أو Ctrl+Shift+I، واختر تبويب Console. ده مكان عمل المبرمج.

.3 المتغيرات Variables: var, let, const

المتغير هو صندوق له اسم، بتحط فيه قيمة لاستخدامها لاحقاً.

📦 تخيل صناديق مُسمّاة:

let age = 25 معناها: صندوق اسمه "age" بداخله القيمة 25. أي وقت تكتب age في الكود، JS تفتح الصندوق وتاخد القيمة منه.

📦 تعريف المتغيرات
// let — متغير قابل للتغيير (الأكثر استخداماً)
let name = 'أحمد';
name = 'محمد';    // مسموح — ممكن تغير قيمته

// const — ثابت لا يتغير
const PI = 3.14159;
// PI = 4;  ❌ خطأ — const لا يتغير

// var — القديم، تجنبه في الكود الحديث
var old = 'قديم';
الكلمةقابل للتغيير؟النطاق (Scope)متى تستخدمها؟
letنعم ✅Block scopeمعظم الوقت
constلا ❌Block scopeالقيم الثابتة
varنعم ✅Function scopeتجنبه في الكود الحديث
قواعد تسمية المتغيرات:
✅ أسماء صحيحة وخاطئة
// ✅ صح
let userName = 'أحمد';     // camelCase — المعيار
let age2 = 25;             // أرقام مسموحة (مش في البداية)
let _private = true;       // underscore مسموح

// ❌ خطأ
// let 2name = '';        رقم في البداية
// let my-var = '';       شرطة مش مسموحة
// let let = '';          كلمة محجوزة
القاعدة الذهبية: استخدم const دايماً كأول خيار. لو احتجت تغيير القيمة، حوّل لـ let. أما var فلا تستخدمه أبداً.

.4 أنواع البيانات Data Types

كل قيمة في JavaScript لها نوع. هناك 7 أنواع أساسية:

String Number Boolean null undefined Object Array
🔢 أمثلة على الأنواع
// String — نص (بين "" أو '' أو ``)
const name    = 'أحمد';
const greeting= "مرحباً!";
const msg     = `اسمك: ${name}`;   // template literal

// Number — أعداد (صحيحة وعشرية)
const age    = 25;
const price  = 9.99;
const neg    = -10;

// Boolean — صح أو غلط فقط
const isLoggedIn = true;
const isDone    = false;

// null — فارغ عن قصد
const user = null;

// undefined — لم تُعطَ قيمة بعد
let result;      // result = undefined

// Array — قائمة مرتبة
const colors = ['أحمر', 'أخضر', 'أزرق'];

// Object — مجموعة خصائص
const person = { name: 'أحمد', age: 25 };
String بالتفصيل:
📝 Template Literals — الطريقة الحديثة
const name  = 'سارة';
const score = 95;

// الطريقة القديمة (صعبة القراءة)
const msg1 = 'مرحباً ' + name + '، نتيجتك: ' + score;

// Template Literal (أفضل)
const msg2 = `مرحباً ${name}، نتيجتك: ${score}`;
// ← "مرحباً سارة، نتيجتك: 95"
الفرق بين null و undefined: null = "الصندوق موجود لكنه فاضي عن قصد". undefined = "الصندوق موجود لكن لسه ما حطناش فيه حاجة".

.6 التحويل بين الأنواع Type Conversion

أحياناً تحتاج تحوّل قيمة من نوع لآخر — مثلاً نص من المستخدم لرقم للحساب.

🔄 التحويل الصريح (Explicit)
// نص → رقم
Number('42')        // 42
parseInt('42px')    // 42  (يأخذ الجزء الرقمي)
parseFloat('3.14') // 3.14

// رقم → نص
String(42)          // "42"
(42).toString()    // "42"

// أي شيء → boolean
Boolean(0)          // false
Boolean('')         // false
Boolean('hello')    // true
Boolean(42)         // true
Falsy Values في JS — احفظها: هذه القيم تعتبر false داخل الشروط: 0 ، "" ، null ، undefined ، NaN ، false — كل ما عداها يعتبر true.

.7 console.log Your Best Friend

console.log() هي أداة المبرمج لطباعة أي قيمة في الـ Console. ستستخدمها في كل سطر كود تقريباً.

🖨️ console.log وأصدقاؤه
// طباعة قيم عادية
console.log('مرحباً');              // مرحباً
console.log(42, 'hello', true);  // 42 hello true

// طباعة مع وصف
const age = 25;
console.log('العمر:', age);          // العمر: 25

// رسائل تحذير وأخطاء
console.warn('تحذير!');            // أصفر
console.error('خطأ!');           // أحمر

// طباعة كائن
const user = { name: 'أحمد', age: 25 };
console.log(user);                  // {name: 'أحمد', age: 25}
console.log('مرحباً', 42, true)
مرحباً 42 true
const x = 10; console.log('x =', x)
x = 10

📝 التمارين العملية Exercises

تمرين 1: عرّف عن نفسك

عرّف 4 متغيرات بـ const: اسمك، عمرك، مدينتك، وهل أنت طالب (boolean). ثم اطبعهم بـ template literal.

الحل: 
const name      = 'أحمد';
const age       = 20;
const city      = 'القاهرة';
const isStudent = true;

console.log(`اسمي ${name}، عمري ${age}، من ${city}، طالب: ${isStudent}`);
تمرين 2: اكتشف الأنواع

استخدم typeof على هذه القيم واطبع النتيجة: 'مرحبا'، 100، false، null، [].

الحل: 
console.log(typeof 'مرحبا');  // string
console.log(typeof 100);      // number
console.log(typeof false);    // boolean
console.log(typeof null);     // object (bug تاريخي)
console.log(typeof []);       // object
تمرين 3: التحويل بين الأنواع

حوّل النص '42' لرقم، والرقم 99 لنص، والرقم 0 لـ boolean.

الحل: 
console.log(Number('42'));    // 42
console.log(String(99));     // "99"
console.log(Boolean(0));    // false
تمرين 4: const أم let؟

حدد أيهما تستخدم (const / let) لكل منها: اسم المستخدم، عداد يزيد في loop، قيمة PI، درجة حرارة تتغير.

الحل: 
const userName = 'أحمد';   // const — لا يتغير
let   counter  = 0;        // let — يتغير
const PI       = 3.14159;  // const — ثابت رياضي
let   temp     = 25;       // let — يتغير
تمرين 5: Template Literal

عندك متغيرين: product = 'لابتوب' و price = 5000. اطبع جملة: "سعر اللابتوب هو 5000 جنيه" باستخدام template literal.

الحل: 
const product = 'لابتوب';
const price   = 5000;
console.log(`سعر ${product} هو ${price} جنيه`);
