.1 ما هو Box Model؟ The Box Model

كل عنصر HTML هو في الحقيقة صندوق مستطيل. الـ Box Model يصف هذا الصندوق وطبقاته الأربع من الداخل للخارج:

Margin (هامش خارجي)
Border (حدود)
Padding (حشوة)
Content (المحتوى)
📱 تخيل موبايل في علبة:
  • Content = الموبايل نفسه
  • Padding = الإسفنج الداخلي في العلبة
  • Border = جدران العلبة
  • Margin = المسافة بين العلبة وعلب المنتجات الأخرى
DevTools: افتح F12 في المتصفح، اختر أي عنصر، وشوف الـ Box Model بصرياً في تبويب "Computed"!

.2 Content — المحتوى Content Area

منطقة المحتوى هي القلب — فيها النص أو الصورة. نتحكم فيها بـ:

📐 أبعاد المحتوى
.box {
  width:  300px;        /* عرض ثابت */
  height: 200px;        /* ارتفاع ثابت */

  width:  50%;          /* نصف عرض الأب */
  width:  100vw;        /* عرض الشاشة كامل */
  height: 100vh;        /* ارتفاع الشاشة كامل */

  /* حدود دنيا وعليا */
  min-width:  200px;    /* لا يصغر عن 200px */
  max-width:  800px;    /* لا يكبر عن 800px */
  min-height: 100px;
  max-height: 400px;
}
max-width للمحتوى: استخدم دايماً max-width على الـ Container الرئيسي بدل width — هيظهر جيداً على الشاشات الكبيرة والصغيرة: max-width: 1100px; margin: 0 auto;

.3 Padding — الحشوة Padding

الـ Padding هو المسافة الداخلية بين المحتوى وحدود العنصر. هو داخل العنصر ولونه لون الخلفية.

📦 طرق كتابة Padding
/* الأربعة اتجاهات: أعلى - يمين - أسفل - يسار */
padding: 10px 20px 10px 20px;

/* اختصار: أعلى/أسفل - يمين/يسار */
padding: 10px 20px;

/* اختصار: نفس القيمة للكل */
padding: 16px;

/* كل اتجاه منفرد */
padding-top:    10px;
padding-right:  20px;
padding-bottom: 10px;
padding-left:   20px;   /* في RTL: يمين */

/* المنطقية (الأحدث - تراعي RTL تلقائياً) */
padding-inline: 20px;   /* يمين ويسار */
padding-block:  10px;   /* أعلى وأسفل */
تذكّر: الـ Padding بيضيف للحجم في الوضع الافتراضي. لو عندك box عرضها 200px وضفت padding: 20px — العرض الكلي يصبح 240px. الحل في الدرس القادم!

.4 Border — الحدود Border

🖼️ الحدود
/* الاختصار: سُمك - نوع - لون */
border: 2px solid #1e3a8a;
border: 1px dashed #ccc;
border: 3px dotted red;

/* كل جانب منفرد */
border-top:    2px solid blue;
border-bottom: 1px dashed gray;
border-right:  4px solid green;  /* في RTL: اليمين */
border-left:   none;

/* أنواع الحدود */
border-style: solid;   /* خط صلب */
border-style: dashed;  /* خط متقطع */
border-style: dotted;  /* نقاط */
border-style: double;  /* خطان */
border-style: none;    /* بلا حدود */

/* تدوير الزوايا */
border-radius: 8px;     /* كل الزوايا */
border-radius: 50%;     /* دائرة! */
border-radius: 20px 0;  /* أعلى يمين/يسار ثم أسفل */

/* Outline: خارج الـ border، لا يأخذ مساحة */
outline: 2px solid blue;
outline-offset: 4px;

.5 Margin — الهامش Margin

الـ Margin هو المسافة خارج العنصر — يفصله عن باقي العناصر. هو شفاف دايماً (لا يأخذ لون الخلفية).

↔️ الـ Margin
/* نفس نمط الـ padding */
margin: 20px;             /* الكل */
margin: 10px 20px;        /* أعلى/أسفل - يمين/يسار */
margin-top:   30px;
margin-bottom: 16px;

/* توسيط عنصر block أفقياً! */
.container {
  width: 800px;
  margin: 0 auto;   /* أعلى/أسفل: 0 — يمين/يسار: auto */
}

/* margin: auto في Flexbox: يدفع للنهاية */
.nav-logo { margin-left: auto; }  /* يدفع اللوجو لليسار */

/* Margin Collapse: مشكلة شائعة! */
/* لو p1 عنده margin-bottom: 20px وp2 عنده margin-top: 30px */
/* المسافة بينهم = 30px (الأكبر) وليس 50px */
Margin Collapse: عندما يكون لعنصرين متتاليين margin عمودي، لا تُجمع القيمتان — يُطبَّق الأكبر فقط. هذا لا يحدث في Flexbox وGrid!

.6 box-sizing: border-box The Box Sizing Fix

هذه واحدة من أهم خصائص CSS — تحل مشكلة شائعة جداً:

📦 المشكلة: الصندوق "يطفح"!

عندك div عرضه 200px، وضفت padding: 20px. بالوضع الافتراضي content-box، العرض الكلي يصبح 200 + 20 + 20 = 240px. يطفح من الحاوية!

الحل: border-box يجعل العرض المحدد يشمل الـ padding والـ border تلقائياً.

✅ الإعداد الأساسي لكل مشروع
/* ضع هذا في أول ملف CSS في كل مشروع */
*, *::before, *::after {
  box-sizing: border-box;
}

/* الآن: width: 200px + padding: 20px = مازال 200px إجمالاً */
.box {
  width: 200px;
  padding: 20px;    /* المحتوى = 160px، الإجمالي = 200px */
  border: 2px solid black; /* يُحسب ضمن الـ 200px */
}

/* مقارنة: */
/* content-box (افتراضي): total = width + padding + border */
/* border-box:             total = width (يشمل كل شيء) */

.7 Overflow — المحتوى الفائض Overflow

لما المحتوى أكبر من حجم العنصر، ماذا نفعل؟ خاصية overflow تتحكم في ذلك:

overflow: visible (افتراضي)
هذا نص طويل جداً يفيض خارج الصندوق ويظهر بجانب العناصر الأخرى
overflow: hidden
هذا نص طويل جداً يُقطع عند حدود الصندوق ولا يظهر الباقي
overflow: scroll
هذا نص طويل جداً يظهر شريط تمرير دائماً حتى لو المحتوى قصير
overflow: auto
هذا نص طويل جداً يظهر شريط تمرير فقط عند الحاجة — الأفضل!
📜 overflow الكاملة
.box {
  overflow: visible; /* يظهر خارج الحدود (افتراضي) */
  overflow: hidden;  /* يُقطع، لا شريط تمرير */
  overflow: scroll;  /* شريط تمرير دائماً */
  overflow: auto;    /* شريط فقط عند الحاجة (الأفضل) */

  /* محاور منفصلة */
  overflow-x: auto;  /* أفقي فقط */
  overflow-y: hidden; /* عمودي فقط */
}

/* اقطع النص الطويل بـ ... */
.truncate {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
  max-width: 200px;
}

📝 التمارين Exercises

تمرين 1: الإعداد الأساسي

اكتب CSS الإعداد الأساسي لكل مشروع: box-sizing: border-box للكل، reset للـ margin والـ padding، وتعريف body.

الحل: 
*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

body {
  font-family: 'Cairo', Arial, sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #1e293b;
  background: #f8fafc;
}
تمرين 2: بطاقة بـ Box Model

اعمل .card: padding: 24px على الكل، border: 1px solid #e2e8f0، border-radius: 12px، margin-bottom: 16px، box-shadow خفيف.

الحل: 
.card {
  padding: 24px;
  border: 1px solid #e2e8f0;
  border-radius: 12px;
  margin-bottom: 16px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06);
  background: white;
}
تمرين 3: توسيط Container

اعمل .container بعرض أقصى 1100px، موسّط أفقياً، وبـ padding: 0 16px للجوانب.

الحل: 
.container {
  max-width: 1100px;
  margin: 0 auto;
  padding: 0 16px;
}
تمرين 4: زرار بـ Padding وBorder

اعمل .btn: padding: 12px 28px، border: none، border-radius: 30px، خلفية زرقاء، نص أبيض، وعند hover border-radius يصبح 8px.

الحل: 
.btn {
  padding: 12px 28px;
  border: none;
  border-radius: 30px;
  background: #1e3a8a;
  color: white;
  font-weight: 700;
  cursor: pointer;
  transition: border-radius 0.3s;
}

.btn:hover { border-radius: 8px; }
تمرين 5: نص مقطوع بـ ...

اعمل .truncate-text: العرض الأقصى 250px، اقطع النص بـ ... عند الفيضان.

الحل: 
.truncate-text {
  max-width: 250px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
