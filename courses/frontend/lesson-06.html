.1 ما هو CSS؟ What is CSS?

CSS اختصار لـ Cascading Style Sheets، يعني "صحائف الأنماط المتتالية". ببساطة: هو اللغة اللي بتتحكم في شكل ومظهر صفحة HTML.

🏠 تذكّر تشبيه البيت:

HTML = الطوب والخرسانة (الهيكل). CSS = الدهان، الأثاث، الديكور. نفس البيت ممكن يبقى شكله مختلف تماماً بأثاث مختلف. نفس الـ HTML ممكن تظهر بـ 1000 شكل مختلف بـ CSS مختلف!

كلمة "Cascading" (متتالية) مهمة جداً — معناها إن الأنماط بتتراكم وبتورّث من الأب للابن، والأولوية للأقرب والأكثر تحديداً.

أي صفحة ويب بتشوفها دلوقتي — مهما كانت جميلة — مبنية بـ HTML وCSS وJavaScript. افتح أي موقع واضغط F12 وشوف!

.2 طرق إضافة CSS Ways to Add CSS

External CSS

✅ الأفضل

ملف .css منفصل — كل الصفحات تستخدمه. سهل الصيانة.

Internal CSS

⚠️ مقبول

داخل <style> في الـ head. لصفحة واحدة أو للتجربة السريعة.

Inline CSS

❌ تجنّب

داخل الـ attribute style مباشرة. صعب الصيانة ويكسر الفصل بين HTML وCSS.

📁 الطرق الثلاثة
<!-- 1. External: ملف خارجي (الأفضل) -->
<head>
  <link rel="stylesheet" href="style.css">
</head>

<!-- 2. Internal: داخل <style> في الصفحة -->
<head>
  <style>
    p { color: blue; }
  </style>
</head>

<!-- 3. Inline: داخل الوسم مباشرة (تجنّب) -->
<p style="color: blue;">نص أزرق</p>

.3 تركيب قاعدة CSS CSS Rule Syntax

كل قاعدة CSS بتتكون من 3 أجزاء:

📐 تركيب القاعدة
selector {
  property: value;
  property: value;
}

/* مثال حقيقي */
h1 {
  color: #1e3a8a;          /* لون النص */
  font-size: 2rem;         /* حجم الخط */
  text-align: center;      /* توسيط النص */
  margin-bottom: 20px;     /* مسافة أسفل */
}
المصطلحالمعنىمثال
Selectorمن الذي سيتأثر؟h1, .card, #header
Propertyماذا نغيّر؟color, font-size, margin
Valueما هي القيمة الجديدة؟red, 16px, 20px
Declarationproperty + value معاًcolor: red;
Ruleselector + كل الـ declarationsالكتلة كلها
التعليقات في CSS: بتتكتب هكذا: /* هذا تعليق */ — مختلفة عن HTML التي تستخدم <!-- -->

.4 المحددات الأساسية Basic Selectors

🎯 المحددات الثلاثة الأساسية
/* 1. محدد العنصر: يستهدف كل العناصر من هذا النوع */
p {
  color: #333;
  line-height: 1.8;
}

/* 2. محدد الكلاس (.): يستهدف كل من له هذا الكلاس */
.highlight {
  background: yellow;
  font-weight: bold;
}

/* 3. محدد المعرّف (#): يستهدف عنصر واحد بعينه */
#main-title {
  font-size: 2.5rem;
  text-align: center;
}
📝 الاستخدام في HTML
<!-- العنصر: يتأثر كل p تلقائياً -->
<p>هذه فقرة تأثرت بمحدد العنصر</p>

<!-- الكلاس: أضف class="highlight" -->
<p class="highlight">هذه فقرة مُميَّزة</p>

<!-- المعرّف: استخدم id فريد في الصفحة -->
<h1 id="main-title">العنوان الرئيسي</h1>

<!-- عنصر بكلاسين في نفس الوقت! -->
<p class="highlight big-text">فقرة بكلاسين</p>
المحددالرمزالاستخداممرات الاستخدام
Elementp, h1تأثير عام على كل العناصرلا محدودية
Class.card, .btnتنسيق مشترك بين عناصر متعددةلا محدودية (الأفضل)
ID#header, #logoعنصر فريد واحد في الصفحةمرة واحدة فقط
لا تستخدم ID للتنسيق! الـ ID لازم يكون فريداً — استخدمه للـ JavaScript وروابط المرساة فقط. للتنسيق، دايماً استخدم class.

.5 المحددات المتقدمة Advanced Selectors

🔬 محددات أكثر قوة
/* الجميع: يستهدف كل العناصر */
* { box-sizing: border-box; margin: 0; padding: 0; }

/* تجميع محددات: فاصلة بينهم */
h1, h2, h3 { font-family: 'Cairo', sans-serif; }

/* السليل (Descendant): مسافة بينهم */
nav a { color: white; text-decoration: none; }
/* يستهدف كل a داخل nav (حتى لو متداخلة) */

/* الابن المباشر (Child): علامة > */
ul > li { list-style: disc; }
/* يستهدف li ابن مباشر لـ ul فقط */

/* الأخ المجاور (Adjacent Sibling): علامة + */
h2 + p { font-size: 1.1rem; }
/* أول p بعد h2 مباشرة */

/* المحددات الشبه-فئوية (Pseudo-classes) */
a:hover    { color: orange; }       /* عند التحويم */
a:visited  { color: purple; }      /* بعد الزيارة */
li:first-child { font-weight: bold; } /* أول عنصر */
li:last-child  { border: none; }     /* آخر عنصر */
li:nth-child(2) { background: #f0f4ff; } /* العنصر الثاني */

/* محدد الخاصية (Attribute Selector) */
input[type="email"] { border-color: green; }
a[target="_blank"]  { font-style: italic; }
:hover هو الأكثر استخداماً. كل زر وكل رابط تقريباً عنده :hover لإضافة تفاعل بصري بدون JavaScript.

.6 الأولوية (Specificity) Specificity

لما أكتر من قاعدة CSS بتستهدف نفس العنصر، مَن الفائز؟ الجواب هو Specificity — نظام نقاط يحدد الأولوية.

🏆 تخيل نظام نقاط:

كل نوع محدد له نقاط مختلفة. اللي عنده أكتر نقاط، أنماطه هي اللي بتطبّق.

1
0
0
→ #header (ID) = 100 نقطة
0
1
0
→ .card (Class) = 10 نقاط
0
0
1
→ p (Element) = 1 نقطة
⚖️ مثال على الأولوية
p { color: black; }         /* 1 نقطة */
.text { color: blue; }      /* 10 نقاط */
#intro { color: red; }      /* 100 نقطة */

/* <p class="text" id="intro"> → اللون: red (100 نقطة) */

/* !important يتغلب على الكل (استخدمه باحتراس) */
p { color: green !important; }  /* الأعلى على الإطلاق */
تجنّب !important قدر الإمكان. استخدامه بكثرة يجعل الكود صعب الصيانة. الحل الصح هو كتابة محددات أكثر تحديداً.

.7 جرب بنفسك! Try It Live

💻 محرر الكود الحي

📝 التمارين Exercises

تمرين 1: ألوان الفقرات

اكتب CSS يجعل: كل h2 بلون أزرق غامق، كل p بحجم 1.1rem، العناصر ذات الكلاس .important بخلفية صفراء وخط عريض.

الحل: 
h2 { color: #1e3a8a; }
p  { font-size: 1.1rem; }
.important { background: yellow; font-weight: bold; }
تمرين 2: تنسيق القائمة

اكتب CSS يستهدف: كل li داخل nav (محدد السليل) بـ display: inline، وعند الـ hover يتغير لون الـ a داخل nav.

الحل: 
nav li { display: inline; margin-left: 15px; }
nav a  { color: #1e3a8a; text-decoration: none; }
nav a:hover { color: #dc2626; text-decoration: underline; }
تمرين 3: تجميع المحددات

اكتب قاعدة CSS واحدة تجعل h1 و h2 و h3 و h4 جميعها بنفس الخط 'Cairo' و font-weight: 700.

الحل: 
h1, h2, h3, h4 {
  font-family: 'Cairo', sans-serif;
  font-weight: 700;
}
تمرين 4: Pseudo-classes

اكتب CSS يجعل: أول li في القائمة بلون أحمر، آخر li بلون أزرق، والباقي بلون رمادي. وعند hover على أي li تصبح خلفيتها صفراء.

الحل: 
li { color: gray; }
li:first-child { color: red; }
li:last-child  { color: blue; }
li:hover { background: yellow; }
تمرين 5: صفحة مُنسَّقة كاملة

اعمل صفحة HTML + CSS بها: header بخلفية زرقاء وكتابة بيضاء، nav بروابط أفقية، section بـ 3 بطاقات .card، وfooter. كل الـ CSS في <style>.

الحل: 
<style>
  * { box-sizing: border-box; margin: 0; padding: 0; }
  body { font-family: Arial; }

  header { background: #1e3a8a; color: white; padding: 20px; text-align: center; }

  nav { background: #f0f4ff; padding: 10px 20px; }
  nav a { margin-left: 15px; color: #1e3a8a; text-decoration: none; font-weight: bold; }
  nav a:hover { color: #dc2626; }

  .cards { display: flex; gap: 16px; padding: 20px; flex-wrap: wrap; }
  .card { flex: 1; min-width: 200px; background: white; border-radius: 10px; padding: 20px; box-shadow: 0 2px 8px rgba(0,0,0,0.1); }
  .card:hover { transform: translateY(-4px); transition: 0.2s; }

  footer { background: #1e3a8a; color: white; text-align: center; padding: 16px; margin-top: 20px; }
</style>
