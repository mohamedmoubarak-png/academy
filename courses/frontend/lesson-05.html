.1 هيكل الجدول Table Structure

الجدول في HTML بيتكوّن من صفوف وأعمدة. الهيكل الكامل:

📊 تخيل جدول Excel:

الـ <table> هو ورقة العمل كلها. الـ <thead> هو الصف العلوي بالعناوين (الأزرق الغامق). الـ <tbody> هو البيانات اللي في النص. الـ <tfoot> هو صف الإجمالي في الأسفل.

📋 كود الجدول الكامل
<table>

  <!-- عنوان الجدول (اختياري) -->
  <caption>نتائج الطلاب</caption>

  <!-- رأس الجدول -->
  <thead>
    <tr>
      <th>الاسم</th>
      <th>الدرجة</th>
    </tr>
  </thead>

  <!-- جسم الجدول -->
  <tbody>
    <tr>

  </tfoot>

</table>
الوسموظيفته

<tbody>قسم جسم الجدول (البيانات)
<tfoot>قسم ذيل الجدول (الإجماليات)
<tr>صف (Table Row)
<th>خلية رأسية عريضة (Table Header)
<td>خلية بيانات عادية (Table Data)

.2 colspan و rowspan Merging Cells

colspan يدمج خلايا أفقياً (عبر أعمدة). rowspan يدمج خلايا عمودياً (عبر صفوف).

📦 تخيل خلايا الجدول كغرف:

colspan="2" = إزالة الحائط بين غرفتين جنب بعض (توسيع أفقي). rowspan="2" = إزالة الحائط بين طابقين (توسيع عمودي).

🔀 دمج الخلايا
<table border="1">
  <tr>
    <!-- colspan: تمتد على عمودين -->
    <th colspan="2">المرحلة الأولى</th>
    <th>المرحلة الثانية</th>
  </tr>
  <tr>
    <td>HTML</td>
    <td>CSS</td>
    <td>JavaScript</td>
  </tr>
  <tr>
    <!-- rowspan: تمتد على صفين -->
    <td rowspan="2">مجموعة أ</td>
    <td>محمد</td>
    <td>95</td>
  </tr>
  <tr>
    <!-- مش محتاج تكتب مجموعة أ تاني -->
    <td>سارة</td>
    <td>88</td>
  </tr>
</table>
تحذير شائع: لما تستخدم rowspan="2"، الصف التاني هيكون فيه خلية أقل من الصفوف العادية. لو نسيت ده، الجدول هيختل.

.3 الفورم HTML Forms

الفورم (<form>) هو الطريقة الوحيدة لإرسال بيانات المستخدم للسيرفر. تسجيل الدخول، البحث، نماذج التواصل — كلها forms.

📬 تخيل طلب وظيفة:

الـ <form> هو الورقة كلها. الـ inputs هي الحقول اللي بتملأها (الاسم، الإيميل، ...). الـ button هو زر "إرسال الطلب".

📝 هيكل الفورم الأساسي
<form action="/submit" method="POST">

  <!-- كل حقل فيه label + input -->
  <label for="username">اسم المستخدم:</label>
  <input
    type="text"
    id="username"
    name="username"
    placeholder="أدخل اسمك"
    required
  >

  <button type="submit">إرسال</button>

</form>
الخاصيةوظيفتهاالقيم
actionأين تُرسل البيانات (URL)/submit, contact.php
methodطريقة الإرسالGET (في الـ URL) | POST (مخفي)
nameاسم الحقل في البيانات المُرسلةأي نص بدون مسافات
requiredالحقل إلزامي — لا يمكن إرسال بدونهلا قيمة لها
placeholderنص تلميح داخل الحقلأي نص
GET مقابل POST: استخدم GET للبحث (البيانات تظهر في الـ URL). استخدم POST للبيانات الحساسة (كلمة مرور، بيانات شخصية).

.4 أنواع الـ input Input Types

أكتر ما يميّز الـ HTML5 هو أنواع الـ input المتعددة — المتصفح بيعرف كيف يتعامل مع كل نوع:

type="text"
type="email"
type="password"
type="number"
type="date"
type="range"
type="color"
type="file"
🎛️ أنواع input بالكود
<input type="text"     placeholder="نص عادي">
<input type="email"    placeholder="me@example.com">
<input type="password" placeholder="••••••••">
<input type="number"  min="1" max="100">
<input type="date">
<input type="range"   min="0" max="100">
<input type="color">
<input type="file"    accept=".pdf,.jpg">
<input type="checkbox" id="agree">
<input type="radio"    name="gender" value="male">
Radio vs Checkbox: الـ radio للاختيار الواحد من مجموعة (ذكر / أنثى). الـ checkbox للاختيار المتعدد (الهوايات: كرة + سباحة + قراءة). Radio buttons في نفس المجموعة لازم يكون لهم نفس الـ name.

.5 label و textarea و select Form Elements

وسم label — ربط النص بالحقل

الـ <label> بيربط نص وصفي بحقل input. لما المستخدم يضغط على النص، الحقل ينشّط تلقائياً. الربط يتم عبر for في الـ label و id في الـ input:

🏷️ label صح ولأ
<!-- ✅ الطريقة الصحيحة -->
<label for="name">الاسم الكامل:</label>
<input type="text" id="name" name="fullname">

<!-- ✅ أو التضمين: label حاوي input -->
<label>
  الاسم الكامل:
  <input type="text" name="fullname">
</label>
textarea — حقل نص متعدد الأسطر
📄 textarea
<label for="msg">رسالتك:</label>
<textarea
  id="msg"
  name="message"
  rows="5"
  cols="40"
  placeholder="اكتب رسالتك هنا..."
  required
></textarea>
select — قائمة منسدلة
🔽 select و option
<label for="course">اختر المسار:</label>
<select id="course" name="course">
  <option value="">-- اختر --</option>
  <option value="frontend">الفرونت إند</option>
  <option value="python" selected>Python</option>
  <option value="backend" disabled>الباك إند (قريباً)</option>
</select>

.6 أزرار الفورم Buttons

🔘 أنواع الأزرار
<!-- إرسال الفورم -->
<button type="submit">إرسال ✉️</button>

<!-- إعادة تعيين الحقول -->
<button type="reset">مسح 🗑️</button>

<!-- زر عادي لا يرسل الفورم -->
<button type="button" onclick="doSomething()">معاينة 👁️</button>
مهم: لو استخدمت <button> داخل <form> بدون type، القيمة الافتراضية هي submit وده ممكن يرسل الفورم بالغلط. دايماً حدد الـ type.

📝 التمارين Exercises

تمرين 1: جدول جدول حصص

اعمل جدول حصص أسبوعي فيه: thead بأيام الأسبوع (السبت للأربعاء)، tbody بـ 3 حصص، caption بعنوان.

الحل: 
<table border="1">
  <caption>جدول حصصي الأسبوعي</caption>
  <thead>
    <tr>
      <th>الحصة</th>
      <th>السبت</th>
      <th>الأحد</th>
      <th>الاثنين</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>1</td><td>HTML</td><td>CSS</td><td>JS</td></tr>
    <tr><td>2</td><td>Python</td><td>HTML</td><td>CSS</td></tr>
    <tr><td>3</td><td>مشروع</td><td>مشروع</td><td>مشروع</td></tr>
  </tbody>
</table>
تمرين 2: جدول بدمج خلايا

اعمل جدول فيه: الخلية الأولى تمتد على عمودين (colspan="2")، وخلية في العمود الأول تمتد على صفين (rowspan="2").

الحل: 
<table border="1">
  <tr>
    <th colspan="2">المسارات التعليمية</th>
    <th>المدة</th>
  </tr>
  <tr>
    <td rowspan="2">الفرونت إند</td>
    <td>HTML وCSS</td>
    <td>4 أسابيع</td>
  </tr>
  <tr>
    <td>JavaScript</td>
    <td>6 أسابيع</td>
  </tr>
</table>
تمرين 3: نموذج تسجيل الدخول

اعمل فورم تسجيل دخول فيه: حقل إيميل، حقل كلمة مرور، checkbox "تذكرني"، وزر submit.

الحل: 
<form action="/login" method="POST">

  <label for="email">البريد الإلكتروني:</label>
  <input type="email" id="email" name="email"
         placeholder="me@example.com" required>

  <label for="pass">كلمة المرور:</label>
  <input type="password" id="pass" name="password"
         placeholder="••••••••" required>

  <label>
    <input type="checkbox" name="remember">
    تذكرني
  </label>

  <button type="submit">تسجيل الدخول</button>

</form>
تمرين 4: استطلاع رأي

اعمل استطلاع رأي فيه: سؤال بـ radio (كيف تعلمت البرمجة؟ — دورات، يوتيوب، كتب)، سؤال بـ checkbox (ما اللغات التي تعرفها؟ — HTML, CSS, JS, Python).

الحل: 
<form>
  <p><strong>كيف تعلمت البرمجة؟</strong></p>
  <label><input type="radio" name="how" value="courses"> دورات تدريبية</label><br>
  <label><input type="radio" name="how" value="youtube"> يوتيوب</label><br>
  <label><input type="radio" name="how" value="books"> كتب</label>

  <p><strong>ما اللغات التي تعرفها؟</strong></p>
  <label><input type="checkbox" name="lang" value="html"> HTML</label>
  <label><input type="checkbox" name="lang" value="css"> CSS</label>
  <label><input type="checkbox" name="lang" value="js"> JavaScript</label>
  <label><input type="checkbox" name="lang" value="py"> Python</label>

  <br><button type="submit">إرسال الاستطلاع</button>
</form>
تمرين 5: نموذج تواصل كامل

اعمل نموذج تواصل كامل فيه: الاسم (text)، الإيميل (email)، الموضوع (select بـ 3 خيارات)، الرسالة (textarea)، أولوية الرد (radio: عادي/عاجل)، موافقة على الشروط (checkbox required)، وأزرار submit و reset.

الحل: 
<form action="/contact" method="POST">
  <label for="n">الاسم:</label>
  <input type="text" id="n" name="name" required>

  <label for="e">الإيميل:</label>
  <input type="email" id="e" name="email" required>

  <label for="sub">الموضوع:</label>
  <select id="sub" name="subject">
    <option value="support">الدعم التقني</option>
    <option value="billing">الاشتراك</option>
    <option value="other">أخرى</option>
  </select>

  <label for="m">الرسالة:</label>
  <textarea id="m" name="message" rows="4" required></textarea>

  <p>الأولوية:</p>
  <label><input type="radio" name="priority" value="normal" checked> عادي</label>
  <label><input type="radio" name="priority" value="urgent"> عاجل</label>

  <label>
    <input type="checkbox" name="terms" required>
    أوافق على الشروط والأحكام
  </label>

  <button type="submit">إرسال ✉️</button>
  <button type="reset">مسح 🗑️</button>
</form>
