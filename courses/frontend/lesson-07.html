🎨 خصائص اللون
p {

  background-color: #f9f9f9;      /* لون الخلفية */

.2 صيغ الألوان Color Formats

🌈 كل الصيغ
/* 1. أسماء (Color Names) — 140 اسم */
color: red;  color: blue;  color: tomato;  color: steelblue;

/* 2. Hex — الأكثر شيوعاً */
color: #ff0000;      /* أحمر كامل */
color: #f00;         /* اختصار لـ #ff0000 */
color: #1e3a8a;      /* أزرق داكن */

/* 3. RGB — أحمر، أخضر، أزرق (0-255) */
color: rgb(255, 0, 0);        /* أحمر */
color: rgb(30, 58, 138);      /* أزرق داكن */

/* 4. RGBA — مع شفافية (0-1) */
color: rgba(30, 58, 138, 0.5);   /* نصف شفاف */
background: rgba(0, 0, 0, 0.3);  /* أسود شفاف */

/* 5. HSL — تدرج الألوان، الإشباع، الإضاءة */
color: hsl(225, 64%, 33%);    /* نفس الأزرق الداكن */
color: hsla(225, 64%, 33%, 0.7); /* مع شفافية */

/* 6. Gradient — تدرج لوني */
background: linear-gradient(to right, #1e3a8a, #7c3aed);
background: radial-gradient(circle, #fff, #1e3a8a);
متى تستخدم كل صيغة؟
• Hex: في معظم الحالات — أشهر وأسرع
• RGBA: لما تحتاج شفافية على لون محدد
• HSL: لما تعمل ثيمات — سهل تغيير الإضاءة
• Opacity: لما تريد تشفيف العنصر كله (نص + خلفية)

.3 CSS Variables Custom Properties

المتغيرات في CSS (Custom Properties) تسمح بتعريف قيم مرة واحدة واستخدامها في كل مكان. هذا يجعل تغيير الثيم أمراً سهلاً للغاية!

📦 تخيل صندوق ألوان:

بدل ما تكتب #1e3a8a في كل مكان، تعرّف متغير --primary: #1e3a8a مرة واحدة. لو غيّرت اللون، كل الموقع يتغير تلقائياً!

📦 CSS Variables
/* تعريف المتغيرات في :root (متاحة في كل الصفحة) */
:root {
  --primary:   #1e3a8a;
  --secondary: #7c3aed;
  --accent:    #ea580c;
  --bg:        #f9fafb;
  --text:      #1e293b;
  --radius:    10px;
  --spacing:   16px;
}

/* استخدام المتغيرات بدالة var() */
h1 { color: var(--primary); }
.card {
  background: var(--bg);
  border-radius: var(--radius);
  padding: var(--spacing);
}
button { background: var(--accent); }

/* قيمة افتراضية لو المتغير غير معرّف */
color: var(--missing, red);

.4 خصائص الخطوط Font Properties

🔤 خصائص الخط
p {
  /* نوع الخط — قائمة fallback */
  font-family: 'Cairo', 'Arial', sans-serif;

  /* الحجم */
  font-size: 16px;      /* بكسل */
  font-size: 1rem;      /* نسبة للجذر (16px افتراضي) */
  font-size: 1.5em;     /* نسبة للأب */
  font-size: clamp(1rem, 2.5vw, 1.5rem); /* متجاوب! */

  /* السُمك */
  font-weight: normal;  /* = 400 */
  font-weight: bold;    /* = 700 */
  font-weight: 900;     /* الأثقل */

  /* الأسلوب */
  font-style: normal;
  font-style: italic;   /* مائل */

  /* اختصار لكل الخصائص مرة واحدة */
  font: italic bold 1.2rem/1.8 'Cairo', sans-serif;
  /*    style weight size/line-height family */
}
معاينة أنواع الخطوط:
font-family: 'Cairo', sans-serif (الخط المستخدم في هذه الأكاديمية)
دفتر البرمجة — نتعلم معاً 🚀
font-family: 'Aref Ruqaa' (العناوين الخطية)
دفتر البرمجة
font-family: 'Fira Code' (الكود البرمجي)
const academy = "برمجة بالعربي";

.5 Google Fonts Google Fonts

Google Fonts مكتبة مجانية بمئات الخطوط. طريقة الاستخدام بسيطة جداً:

🔗 استخدام Google Fonts
<!-- الخطوة 1: أضف في <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link
  href="https://fonts.googleapis.com/css2?family=Cairo:wght@400;700&display=swap"
  rel="stylesheet"
>

/* الخطوة 2: استخدم في CSS */
body {
  font-family: 'Cairo', sans-serif;
}

/* للعربية: خطوط مُوصى بها */
/* Cairo    — واضح وحديث للنصوص */
/* Aref Ruqaa — خطي جميل للعناوين */
/* Tajawal  — خفيف ومريح للقراءة */
/* Reem Kufi — هندسي للعناوين */
للأداء: لا تحمّل أكثر من 2-3 خطوط ومتغيراتهم. كل خط إضافي يبطئ تحميل الصفحة.

ظل النص

✏️ خصائص النصوص كاملة
h1 {
  text-align: center;            /* left | right | center | justify */
  text-decoration: none;         /* none | underline | line-through */
  text-transform: uppercase;     /* none | lowercase | uppercase | capitalize */
  line-height: 1.8;              /* ارتفاع السطر (بدون وحدة = نسبة) */
  letter-spacing: 1px;           /* مسافة بين الحروف */
  word-spacing: 4px;             /* مسافة بين الكلمات */
  white-space: nowrap;           /* منع كسر السطر */
  text-overflow: ellipsis;       /* ... عند القص (مع overflow:hidden) */
  text-shadow: 2px 2px 4px rgba(0,0,0,0.2); /* ظل */
}

📝 التمارين Exercises

تمرين 1: نظام ألوان بـ CSS Variables

عرّف في :root متغيرات لـ: لون رئيسي، ثانوي، خلفية، نص. ثم استخدمهم في تنسيق h1 وp وbutton.

الحل: 
:root {
  --primary:  #1e3a8a;
  --secondary: #7c3aed;
  --bg:        #f8fafc;
  --text:      #1e293b;
}

h1     { color: var(--primary); }
p      { color: var(--text); background: var(--bg); }
button { background: var(--secondary); color: white; }
تمرين 2: تنسيق النصوص

اكتب CSS يجعل: عنوان الصفحة بالوسط وبحجم 2.5rem، الفقرات بـ line-height: 1.9 وfont-size: 1.05rem، والروابط بدون تسطير وتظهر باللون الأزرق عند hover.

الحل: 
h1 { text-align: center; font-size: 2.5rem; }
p  { line-height: 1.9; font-size: 1.05rem; }
a  { text-decoration: none; color: inherit; }
a:hover { color: #1e3a8a; }
تمرين 3: ألوان الخلفية والشفافية

اعمل بطاقة .card بخلفية بيضاء وظل. داخلها عنوان بخلفية rgba زرقاء شفافة (0.15)، وفقرة بلون رمادي.

الحل: 
.card {
  background: white;
  border-radius: 12px;
  padding: 20px;
  box-shadow: 0 4px 16px rgba(0,0,0,0.08);
}

.card h2 {
  background: rgba(30, 58, 138, 0.15);
  color: #1e3a8a;
  padding: 8px 14px;
  border-radius: 6px;
  margin-bottom: 12px;
}

.card p { color: #64748b; line-height: 1.8; }
تمرين 4: typography كاملة

اكتب CSS لـ body يحدد: font-family بـ 3 خطوط fallback، font-size: 16px، color للنص، background للصفحة، line-height، وdirection: rtl.

الحل: 
body {
  font-family: 'Cairo', 'Segoe UI', Arial, sans-serif;
  font-size: 16px;
  color: #1e293b;
  background-color: #f8fafc;
  line-height: 1.8;
  direction: rtl;
}
تمرين 5: بطاقة أسعار منسّقة

اعمل بطاقة سعر فيها: السعر القديم بـ text-decoration: line-through وlون رمادي، السعر الجديد بحجم كبير ولون أخضر، وبادج "خصم 25%".

الحل: 
.price-card { background: white; border-radius: 12px; padding: 24px; text-align: center; }
.old-price  { text-decoration: line-through; color: #94a3b8; font-size: 1rem; }
.new-price  { font-size: 2.5rem; font-weight: 900; color: #15803d; margin: 8px 0; }
.badge      {
  display: inline-block;
  background: #dcfce7;
  color: #15803d;
  font-size: 0.85rem;
  font-weight: 700;
  padding: 4px 14px;
  border-radius: 20px;
  letter-spacing: 0.5px;
}
