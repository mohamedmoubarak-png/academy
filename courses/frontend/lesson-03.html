.1 وسوم النصوص Text Tags

قبل ما ندخل للقوائم والروابط، خلينا نعرف الوسوم اللي بتتعامل مع النصوص العادية:

الوسموظيفتهمثال
<p>فقرة نصية — بتضيف مسافة قبلها وبعدهاأكثر وسم بتستخدمه
<br>سطر جديد داخل نفس الفقرة (مفرد)للعناوين والأشعار
<hr>خط أفقي فاصل (مفرد)للفصل بين أقسام
<pre>نص محافظ على المسافات والأسطر (Preformatted)للكود والشعر
<blockquote>اقتباس طويل من مصدر آخرللأقوال والاقتباسات
<q>اقتباس قصير مضمّنداخل فقرة
<abbr>اختصار — يعرض شرحه عند الـ hoverHTML, CSS, JS
📝 أمثلة وسوم النصوص
<!-- فقرة عادية -->
<p>هذه فقرة نصية عادية.</p>

<!-- سطر جديد بدون فقرة جديدة -->
<p>السطر الأول<br>السطر الثاني في نفس الفقرة</p>

<!-- خط فاصل -->
<hr>

<!-- اقتباس -->
<blockquote>
  العلم نور والجهل ظلام.
</blockquote>

<!-- اختصار -->
<p>
  <abbr title="HyperText Markup Language">HTML</abbr>
  هي لغة الويب الأساسية.
</p>
استخدم <br> بحكمة: لا تستخدم <br> لإضافة مسافات بين الفقرات — ده شغل CSS مش HTML. <br> بس لما تحتاج سطر جديد داخل نفس العنصر.

.2 القوائم غير المرتبة Unordered Lists (ul)

القائمة غير المرتبة بتعرض عناصر بنقاط (Bullets) من غير ترتيب معين.

🛒 تخيل قائمة تسوق:

لما بتكتب قائمة المشتريات، مش مهم ترتيبها — خبز، لبن، بيض. ده بالظبط <ul>: قائمة عناصر مش محتاجة ترتيب.

🔵 القائمة غير المرتبة
<ul>
  <li>خبز</li>
  <li>لبن</li>
  <li>بيض</li>
</ul>

النتيجة:

  • خبز
  • لبن
  • بيض
قاعدة: <li> (List Item) لازم يكون دايماً داخل <ul> أو <ol>. ما تحطه لوحده أبداً.

.3 القوائم المرتبة Ordered Lists (ol)

القائمة المرتبة بتعرض عناصر بأرقام أو حروف — الترتيب مهم هنا.

📋 تخيل وصفة طبخ:

وصفة الكيكة لازم تتبع بالترتيب: أولاً اخلط الدقيق، تانياً ضيف البيض، تالثاً... ده هو <ol>: ترتيب مهم!

🔢 القائمة المرتبة
<!-- أرقام عادية (افتراضي) -->
<ol>
  <li>اخلط الدقيق والسكر</li>
  <li>ضيف البيض والزبدة</li>
  <li>اخبز على 180 درجة</li>
</ol>

<!-- بحروف إنجليزية -->
<ol type="A">
  <li>الخيار الأول</li>
  <li>الخيار الثاني</li>
</ol>

<!-- ابدأ من رقم معين -->
<ol start="5">
  <li>يبدأ من 5</li>
  <li>ثم 6</li>
</ol>
خصائص <ol>:
• type="1" أرقام (افتراضي) | type="A" حروف كبيرة | type="a" حروف صغيرة | type="I" رومان
• start="5" لتبدأ من رقم معين
• reversed لترتيب تنازلي

.4 القوائم الوصفية Description Lists (dl)

القائمة الوصفية بتربط مصطلح بـ تعريفه. مثالية للقواميس والمصطلحات والـ FAQs.

📖 القائمة الوصفية
<dl>
  <dt>HTML</dt>
  <dd>لغة ترميز النصوص الفائقة — لبناء هيكل الصفحات</dd>

  <dt>CSS</dt>
  <dd>أوراق الأنماط المتتالية — للتنسيق والتصميم</dd>

  <dt>JavaScript</dt>
  <dd>لغة البرمجة للتفاعلية والحركة</dd>
</dl>
الفرق بين الثلاث: <dl> = القائمة كلها | <dt> (Description Term) = المصطلح | <dd> (Description Details) = التعريف

.5 القوائم المتداخلة Nested Lists

ممكن تحط قائمة جوا قائمة لعمل تصنيفات فرعية. مثال: قائمة مواد الدراسة والفصول الفرعية.

🗂️ قوائم متداخلة
<ul>
  <li>مسار الفرونت إند
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>مسار Python
    <ul>
      <li>الأساسيات</li>
      <li>Django</li>
    </ul>
  </li>
</ul>
انتبه: القائمة الداخلية لازم تتحط داخل <li> مش بعده. الخطأ الشائع هو وضعها بعد </li>.

.6 الروابط بالتفصيل Links (a tag)

وسم <a> (Anchor) هو أكتر وسم مميّز في HTML — ده اللي بيربط صفحات الويب ببعض. بدونه مفيش "ويب"!

🌉 تخيل كباري بين جزر:

كل صفحة ويب هي جزيرة. الـ Anchor هو الكوبري اللي بيربطها بالجزيرة التانية. من غير كباري، كل جزيرة معزولة.

الخصائص المهمة للرابط:
الخاصيةوظيفتهامثال
hrefالعنوان الذي يوديك إليه الرابطhref="https://google.com"
targetأين يفتح الرابط؟target="_blank" (تبويب جديد)
relالعلاقة بالصفحة المرتبطة (للـ SEO والأمان)rel="noopener noreferrer"
titleنص تلميح يظهر عند الـ Hovertitle="اذهب لجوجل"
downloadلتحميل الملف بدل فتحهdownload="resume.pdf"
أمان مهم: لما تفتح رابط خارجي في تبويب جديد (target="_blank")، دايماً ضيف rel="noopener noreferrer" عشان تمنع ثغرات أمنية.

.7 أنواع الروابط Types of Links

في 5 أنواع روابط ممكن تعملها بـ <a>:

🔗 أنواع الروابط الخمسة
<!-- 1. رابط خارجي: لموقع تاني -->
<a href="https://google.com" target="_blank" rel="noopener noreferrer">
  جوجل
</a>

<!-- 2. رابط داخلي: لصفحة في نفس موقعك -->
<a href="about.html">عن الموقع</a>
<a href="../index.html">الرئيسية (مجلد أعلى)</a>

<!-- 3. رابط مرساة: للانتقال لجزء من نفس الصفحة -->
<a href="#contact">اذهب لقسم التواصل</a>
<!-- الوجهة: عنصر بنفس الـ id -->
<section id="contact">...</section>

<!-- 4. رابط إيميل -->
<a href="mailto:hello@example.com">راسلني</a>

<!-- 5. رابط هاتف -->
<a href="tel:+201234567890">اتصل بي</a>
📌 الروابط المتداخلة بالأيقونات:

ممكن تحط أي عنصر جوا الرابط مش بس نص — صورة، أيقونة، حتى div:

<!-- رابط بصورة -->
<a href="https://google.com">
  <img src="logo.png" alt="جوجل">
</a>
لا تستخدم روابط فاضية: href="#" كرابط مؤقت مقبول لكن في الإنتاج، كل رابط لازم له وجهة حقيقية.

📝 التمارين Exercises

تمرين 1: قائمة خطوات

اكتب قائمة مرتبة بخطوات تنزيل تطبيق على الموبايل (5 خطوات على الأقل).

الحل: 
<h2>خطوات تنزيل التطبيق</h2>
<ol>
  <li>افتح متجر التطبيقات</li>
  <li>ابحث عن اسم التطبيق</li>
  <li>اضغط على "تنزيل"</li>
  <li>انتظر حتى يكتمل التنزيل</li>
  <li>افتح التطبيق وسجّل دخولك</li>
</ol>
تمرين 2: قائمة وصفية للمصطلحات

اعمل قائمة وصفية تشرح 3 مصطلحات: Browser (المتصفح)، Server (الخادم)، URL.

الحل: 
<dl>
  <dt>Browser (المتصفح)</dt>
  <dd>برنامج يستخدمه المستخدم لتصفح الويب مثل Chrome وFirefox</dd>

  <dt>Server (الخادم)</dt>
  <dd>جهاز كمبيوتر يخزن الملفات ويرسلها عند الطلب</dd>

  <dt>URL</dt>
  <dd>عنوان فريد لكل صفحة ويب مثل https://google.com</dd>
</dl>
تمرين 3: قائمة تنقل (Navigation)

اعمل قائمة تنقل بـ nav وul وli وa — لموقع فيه 5 صفحات: الرئيسية، عن الموقع، المدونة، المشاريع، تواصل.

الحل: 
<nav>
  <ul>
    <li><a href="index.html">الرئيسية</a></li>
    <li><a href="about.html">عن الموقع</a></li>
    <li><a href="blog.html">المدونة</a></li>
    <li><a href="projects.html">المشاريع</a></li>
    <li><a href="contact.html">تواصل</a></li>
  </ul>
</nav>
تمرين 4: روابط متنوعة

اعمل قسم "تواصل معي" فيه: رابط خارجي لـ LinkedIn، رابط إيميل، رابط هاتف، ورابط لتحميل ملف CV.

الحل: 
<section id="contact">
  <h2>تواصل معي</h2>
  <ul>
    <li>
      <a href="https://linkedin.com" target="_blank" rel="noopener">
        LinkedIn
      </a>
    </li>
    <li><a href="mailto:me@example.com">راسلني بالإيميل</a></li>
    <li><a href="tel:+201234567890">اتصل بي</a></li>
    <li><a href="cv.pdf" download="Mohamed-CV">حمّل السيرة الذاتية</a></li>
  </ul>
</section>
تمرين 5: صفحة موقع شخصي متكاملة

اعمل صفحة شخصية فيها: nav بروابط مرساة، h2 "عني" بفقرة، h2 "مهاراتي" بقائمة ul متداخلة، h2 "تواصل" بروابط إيميل وهاتف.

الحل: 
<header><h1>محمد أحمد</h1></header>
<nav>
  <a href="#about">عني</a>
  <a href="#skills">مهاراتي</a>
  <a href="#contact">تواصل</a>
</nav>
<main>
  <section id="about">
    <h2>عني</h2>
    <p>مطور ويب شغوف بالتعلم والبناء.</p>
  </section>
  <section id="skills">
    <h2>مهاراتي</h2>
    <ul>
      <li>الفرونت إند
        <ul>
          <li>HTML</li>
          <li>CSS</li>
        </ul>
      </li>
    </ul>
  </section>
  <section id="contact">
    <h2>تواصل</h2>
    <p><a href="mailto:me@example.com">إيميل</a> | <a href="tel:+20100">هاتف</a></p>
  </section>
</main>
