.1 إيه هو الويب أصلاً؟ What is the Web?

قبل ما نتعلم نبني مواقع، خلينا نفهم الأول إحنا بنبني فين وعلى إيه. لما تفتح المتصفح بتاعك (Chrome, Firefox, Safari, Edge) وتدخل على Facebook أو YouTube أو Google، إنت بتكون "على الويب". الويب هو شبكة عملاقة من الصفحات والمواقع المربوطة ببعضها عبر الإنترنت.

🤔 تشبيه يقرب الفكرة:

تخيل إن الإنترنت هو شبكة الطرق والكباري في مدينة كبيرة. أما الويب فهو المباني والمحلات والبيوت اللي على الطرق دي. الإنترنت موجود من 1969، لكن الويب اختُرع سنة 1989 على إيد عالم اسمه Tim Berners-Lee.

إيه الفرق بين الإنترنت والويب؟
الإنترنتالويب
شبكة عالمية من الكومبيوترات المرتبطة ببعضمجموعة من الصفحات والمواقع اللي بتشتغل على الإنترنت
زي طريق سريعزي المحلات والمعالم على الطريق
بيشمل: واتساب، إيميل، تورنت، ألعاب أونلاين...بيشمل: المواقع اللي بتفتحها بالمتصفح بس
معلومة مهمة: الويب جزء من الإنترنت. كل ويب هو إنترنت، لكن مش كل إنترنت هو ويب.

.2 إزاي الويب بيشتغل؟ How the Web Works

دلوقتي خلينا نفهم إيه اللي بيحصل لما تكتب google.com في المتصفح وتضغط Enter:

الخطوات بالتفصيل:
  1. إنت بتطلب: بتكتب google.com في المتصفح وتضغط Enter
  2. المتصفح بيحول الاسم لرقم: كل موقع له عنوان رقمي اسمه IP Address زي 172.217.16.142
  3. المتصفح بيبعت طلب (Request): رسالة تقول "ابعتلي صفحة Google من فضلك"
  4. الطلب بيعدي عبر الإنترنت: بيسافر من بلدك لحد ما يوصل لسيرفرات Google
  5. السيرفر بيرد (Response): بيبعت لك ملفات الصفحة (HTML, CSS, JS, صور...)
  6. المتصفح بيركّب الصفحة: بيقرا الملفات ويعرضلك الصفحة جاهزة على شاشتك
كل ده بيحصل في أقل من ثانية! لما تفتح أي موقع، فعلياً جهازك بيرسل طلب لجهاز تاني في الجانب التاني من العالم، والرد بيرجعلك في لحظات.
مصطلحات مهمة:
  • Client (العميل): الجهاز اللي بيطلب، يعني جهازك إنت
  • Server (الخادم): الجهاز اللي بيرد ويقدم البيانات
  • Browser (المتصفح): البرنامج اللي بيعرض الصفحات (Chrome, Firefox...)
  • URL: عنوان الصفحة، زي https://google.com
  • HTTP/HTTPS: اللغة اللي بيها المتصفح والسيرفر بيتكلموا. الـ S في HTTPS معناها Secure (مشفّر وآمن)
  • Domain: اسم الموقع، زي google.com أو facebook.com

.3 ثلاثي الويب HTML + CSS + JavaScript

أي موقع في الدنيا، من بسيط لمعقد، بيتكوّن من ثلاث لغات أساسية. لو فهمت الثلاثة دول، تقدر تبني أي حاجة تتخيلها على الويب.

🏠 تخيل إنك بتبني بيت:
  • HTML = الطوب والخرسانة (الهيكل والمحتوى)
  • CSS = الدهان والديكور والأثاث (الشكل والتنسيق)
  • JavaScript = الكهرباء والمكيف والريموت (الحركة والتفاعل)

من غير الطوب، البيت مش هيقوم. من غير الدهان، البيت هيبقى وحش. من غير الكهرباء، هيبقى ميت.

اللغةالدورمثال
HTMLالمحتوى والهيكل (نصوص، صور، أزرار...)الكلام اللي بتقراه دلوقتي
CSSالتنسيق (ألوان، أحجام، خطوط، توزيع)الألوان والخطوط اللي شايفها
JavaScriptالتفاعل والحركة والمنطقزرار "عرض الحل" لما تضغط عليه
إحنا في الدرس ده هنركز على HTML بس. CSS و JavaScript ليهم دروس منفصلة جاية.

.4 إيه هو HTML؟ What is HTML?

HTML اختصار لـ HyperText Markup Language، يعني "لغة الترميز للنصوص الفائقة". الاسم معقد، بس الفكرة بسيطة جداً.

📄 تخيل ورقة Word:

لما بتكتب وثيقة في Word، بتختار: العنوان الرئيسي بخط كبير، العنوان الفرعي أصغر، وفقرة عادية. بعدين بتضيف صور، جداول، قوائم...

HTML بتعمل نفس الفكرة بالظبط، بس بدل ما تستخدم أزرار، بتكتب وسوم (Tags) تقول للمتصفح: "ده عنوان"، "دي فقرة"، "دي صورة"، "ده زر"... وهكذا.

خصائص HTML:
  • مش لغة برمجة: هي "لغة ترميز" بس. مش بتعمل عمليات حسابية أو شروط أو حلقات
  • سهلة جداً: في يوم أو يومين تقدر تعمل أي صفحة محتوى
  • أساس كل موقع: مفيش موقع في الدنيا من غير HTML
  • بيقراها المتصفح: بتفتح في أي متصفح بدون أي إعدادات
  • الملفات بتنتهي بـ .html
أهم حاجة تفهمها: HTML بتقول "إيه" - يعني إيه المحتوى. CSS بتقول "إزاي" - يعني إزاي يطلع الشكل. JavaScript بتقول "إمتى" - يعني إمتى يحصل التفاعل.

.5 هيكل صفحة HTML HTML Structure

كل صفحة HTML بتبدأ بنفس الهيكل الأساسي. خليني أوريك إياه وبعدين نفصل كل جزء:

📄 الهيكل الأساسي لصفحة HTML
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>عنوان الصفحة</title>
</head>
<body>
    <!-- محتوى الصفحة هنا -->
    <h1>أهلاً بالعالم!</h1>
</body>
</html>
شرح كل سطر:
السطر 1: <!DOCTYPE html>

دي بتقول للمتصفح: "الصفحة دي HTML بآخر إصدار (HTML5)". لازم تبقى في أول سطر، وملهاش وسم إغلاق.

السطر 2: <html lang="ar" dir="rtl">

هو الوسم الرئيسي اللي بيغلف كل الصفحة. lang="ar" معناها اللغة عربي، و dir="rtl" معناها الكتابة من اليمين لليسار.

السطر 3-6: قسم الـ <head>

ده القسم الخفي من الصفحة. مش بيظهر للمستخدم، بس بيحتوي على معلومات مهمة:

  • <meta charset="UTF-8">: لدعم اللغة العربية وكل لغات العالم
  • <title>: عنوان الصفحة اللي بيظهر في تبويب المتصفح (Tab)
السطر 7-10: قسم الـ <body>

ده القسم اللي بيشوفه المستخدم. كل المحتوى من نصوص وصور وأزرار وفيديوهات بيتكتب هنا.

الفرق المهم:
  • <head> = إعدادات الصفحة (مش بتظهر)
  • <body> = محتوى الصفحة (بيظهر للمستخدم)
📌 مثال يوضح الفرق:

تخيل كتاب. الـ <head> هو زي صفحة الغلاف والمعلومات (اسم الكاتب، دار النشر، الفهرس). أما الـ <body> فهو الصفحات اللي بتقراها فعلياً.

.6 الوسوم (Tags) HTML Tags

الـ Tag أو "الوسم" هو وحدة البناء الأساسية في HTML. معظم الوسوم بتكون زوجين: وسم فتح ووسم إغلاق، والمحتوى بينهم.

🏷️ شكل الوسم
<tagname>المحتوى</tagname>

// مثال حقيقي:
<h1>أنا عنوان رئيسي</h1>
<p>أنا فقرة عادية</p>
أشهر الوسوم اللي هتستخدمها:
الوسموظيفتهمثال
<h1> إلى <h6>عناوين بأحجام مختلفة (h1 الأكبر، h6 الأصغر)<h1>عنوان رئيسي</h1>
<p>فقرة نصية<p>ده كلام عادي</p>
<a>رابط (لينك)<a href="...">اضغط</a>
<img>صورة (وسم مفرد)<img src="pic.jpg">
<ul> و <li>قائمة بنقاط<ul><li>عنصر</li></ul>
<button>زر<button>اضغطني</button>
<div>حاوية لتجميع عناصر<div>...</div>
<br>سطر جديد (وسم مفرد)سطر<br>تاني
نوعين من الوسوم:
  1. وسوم زوجية: بتحتاج فتح وإغلاق، زي <p>...</p>
  2. وسوم مفردة: ملهاش إغلاق، زي <img> و <br>
قاعدة ذهبية: كل وسم تفتحه، لازم تقفله. النسيان ده أكتر خطأ شائع عند المبتدئين.
📌 مثال كامل: 
<body>
    <h1>اسمي محمد</h1>
    <h2>أنا مبرمج</h2>
    <p>أتعلم HTML من دفتر البرمجة.</p>
    <button>اضغط هنا</button>
</body>

.7 الخصائص (Attributes) HTML Attributes

الـ Attribute أو "الخاصية" هي معلومة إضافية بتديها للوسم عشان تتحكم في سلوكه أو شكله. بتتكتب جوه وسم الفتح.

🎯 شكل الخصائص
<tagname attribute="value">محتوى</tagname>

// مثال: رابط بيوديك لجوجل
<a href="https://google.com">روح جوجل</a>

// مثال: صورة
<img src="cat.jpg" alt="قطة لطيفة">
أشهر الخصائص:
الخاصيةوظيفتهاتُستخدم مع
hrefعنوان الرابط<a>
srcمصدر الصورة أو الفيديو<img>, <video>
altنص بديل لو الصورة ما حملتش<img>
idمعرّف فريد للعنصرأي وسم
classتصنيف العنصر (للتنسيق بـ CSS)أي وسم
styleتنسيق مباشر بـ CSSأي وسم
titleنص يظهر عند الـ hoverأي وسم
📌 مثال متكامل: 
<!-- رابط بيفتح في تبويب جديد -->
<a href="https://youtube.com" target="_blank">
    يوتيوب
</a>

<!-- صورة بعرض 300 بكسل -->
<img src="profile.jpg" alt="صورتي" width="300">

<!-- زر بلون خلفية أحمر -->
<button style="background: red; color: white;">
    احذف
</button>
القاعدة: الخصائص دايماً بتتكتب بالشكل ده: اسم_الخاصية="القيمة". القيمة بين علامتي تنصيص.

.8 أول صفحة لك! Your First Page

دلوقتي وقت العملي! خليني أوريك إزاي تعمل أول صفحة HTML بنفسك:

طريقة 1: على الكمبيوتر (للحفظ والمراجعة)
  1. افتح Notepad (على ويندوز) أو أي محرر نصوص
  2. اكتب الكود اللي تحت
  3. احفظ الملف باسم index.html (لازم بامتداد .html)
  4. افتح الملف بأي متصفح
  5. تهانينا! إنت عملت أول صفحة ويب 🎉
طريقة 2: جرب هنا مباشرة!

اكتب أي HTML في المربع، واضغط "تشغيل" تشوف النتيجة:

💻 جرب الكود
عدّل الكود وشوف النتيجة! غيّر النصوص، ضيف فقرات تانية، جرب لون مختلف. ده أفضل طريقة للتعلم.

📝 التمارين العملية Exercises

دلوقتي وقت تطبق اللي اتعلمته. حاول تحل التمرين بنفسك الأول، وبعدين شوف الحل.

تمرين 1: تعريف بنفسك

اكتب صفحة HTML كاملة فيها: عنوان رئيسي باسمك، عنوان فرعي بوظيفتك أو هوايتك، فقرة قصيرة عن نفسك.

الحل: 
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>عن أحمد</title>
</head>
<body>
    <h1>أحمد محمد</h1>
    <h2>مطور ويب</h2>
    <p>أنا شغوف بتعلم البرمجة وبناء مواقع.</p>
</body>
</html>
تمرين 2: قائمة هواياتك

اكتب كود HTML فيه قائمة بـ 5 هوايات بتحبها. استخدم <ul> و <li>.

الحل: 
<h2>هواياتي</h2>
<ul>
    <li>قراءة الكتب</li>
    <li>لعب كرة القدم</li>
    <li>السباحة</li>
    <li>البرمجة</li>
    <li>السفر</li>
</ul>
تمرين 3: روابط مفيدة

اكتب 3 روابط لمواقع بتحبها، كل رابط يفتح في تبويب جديد. استخدم target="_blank".

الحل: 
<a href="https://google.com" target="_blank">جوجل</a>
<br>
<a href="https://youtube.com" target="_blank">يوتيوب</a>
<br>
<a href="https://github.com" target="_blank">جيت هاب</a>
تمرين 4: العناوين بأحجام مختلفة

استخدم العناوين من h1 لـ h6 واكتب جملة في كل واحد فيهم. شوف الفرق في الحجم!

الحل: 
<h1>العنوان رقم 1 - الأكبر</h1>
<h2>العنوان رقم 2</h2>
<h3>العنوان رقم 3</h3>
<h4>العنوان رقم 4</h4>
<h5>العنوان رقم 5</h5>
<h6>العنوان رقم 6 - الأصغر</h6>
تمرين 5: صفحة عن مدينة

اعمل صفحة عن مدينتك المفضلة، فيها: عنوان رئيسي (اسم المدينة)، فقرة وصف قصيرة، قائمة بـ 3 معالم سياحية، رابط لخريطة المدينة على Google Maps.

الحل: 
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>القاهرة</title>
</head>
<body>
    <h1>القاهرة - أم الدنيا</h1>
    <p>القاهرة عاصمة مصر، وأكبر مدنها وأكثرها سكاناً.</p>
    <h2>أشهر معالمها</h2>
    <ul>
        <li>الأهرامات</li>
        <li>برج القاهرة</li>
        <li>المتحف المصري</li>
    </ul>
    <a href="https://maps.google.com/?q=Cairo" target="_blank">
        شاهد القاهرة على الخريطة
    </a>
</body>
</html>
