.1 ما هي DevTools؟ Browser DevTools

DevTools هي مجموعة أدوات مدمجة في المتصفح تتيح لك فحص وتعديل وتصحيح المواقع مباشرة.

🔬 تخيلها مجهراً للمواقع:

كل موقع تزوره يمكنك فتح "غطاء المحرك" ورؤية كل شيء بداخله — HTML، CSS، JS، الشبكة، الأداء. وتقدر تعدّل وتجرب بدون أي خطر على الموقع الأصلي!

.2 لوحة Elements Inspect HTML/CSS

الأكثر استخداماً. تعرض شجرة HTML وتتيح تعديل CSS في الوقت الفعلي.

Elements
Styles
Computed
<body>
<header class="navbar"> <h1>أكاديمية البرمجة</h1> </header>
.navbar { background: #1e3a8a; padding: 16px; }
ما تقدر تفعله في Elements:
  • فحص عنصر: انقر يمين → Inspect لتحديده مباشرة
  • تعديل HTML: دبل-كليك على أي نص أو وسم لتعديله
  • تجربة CSS: في قسم Styles، أضف أو عدّل خصائص مباشرة
  • Box Model: قسم Computed يعرض المسافات والحجم بدقة
  • تعطيل CSS: تيك بجانب كل خاصية لتفعيل/تعطيل
نصيحة ذهبية: عند تصميم CSS، عدّل في DevTools أولاً حتى تحصل على النتيجة المثالية، ثم انقل التعديلات لملفاتك. أسرع بكثير من إعادة رفع الصفحة كل مرة!

🖥️ console API
// الأشهر: طباعة أي شيء
console.log('مرحبا', { name: 'أحمد', age: 25 });

// تحذير (أصفر)
console.warn('هذه الدالة قديمة');

console.error('فشل تحميل البيانات');

// جدول جميل للمصفوفات والكائنات

// تجميع السجلات
console.group('بيانات المستخدم');
console.log('الاسم: أحمد');
console.log('العمر: 25');
console.groupEnd();

.4 Sources والـ Debugger Debugging

أداة تتبع الأخطاء — تتيح إيقاف الكود في أي نقطة ومراقبة المتغيرات.

🐛 Breakpoints والـ Debugger
function calculateTotal(items) {
  let total = 0;

  // ↓ هذا السطر يوقف المتصفح هنا
  debugger;

  items.forEach(item => {
    total += item.price;
  });

  return total;
}
أدوات التنقل في الـ Debugger:
الزرالاختصارالوظيفة
▶️ ResumeF8استمر حتى Breakpoint التالي
⤵️ Step OverF10نفّذ السطر الحالي
⬇️ Step IntoF11ادخل داخل الدالة
⬆️ Step OutShift+F11اخرج من الدالة الحالية
Breakpoint بدون كود: في لوحة Sources، انقر على رقم السطر مباشرة لإضافة Breakpoint بدون تعديل الكود. أفضل من debugger!

.5 لوحة Network Network Monitor

ترصد كل طلب HTTP يقوم به موقعك — ملفات، صور، APIs، كل شيء.

Network
NameStatusTypeTime
index.html200document45ms style.css200stylesheet12ms api/users200fetch320ms logo.png304image8ms api/data404fetch89ms
ما تراقبه في Network:
  • Status Codes: 200 نجاح، 404 غير موجود، 500 خطأ سيرفر
  • Response: شوف البيانات الراجعة من الـ API
  • Headers: معلومات الطلب والاستجابة
  • Timing: كم أخذ كل طلب — تحديد ما هو بطيء
  • Throttling: محاكاة اتصال بطيء (3G) لاختبار تجربة الجوال
XHR و Fetch فيلتر: عند تعامل مع APIs، فيلتر Fetch/XHR في Network يعرض فقط الطلبات البرمجية. أنظف بكثير!

.7 أدوات إضافية مفيدة Extra Tools

وضع الجوال (Device Mode):
📱 اختصارات مهمة
// فتح DevTools
F12  أو  Ctrl+Shift+I

// وضع الجوال (Device Toolbar)
Ctrl+Shift+M

// فتح Console مباشرة
Ctrl+Shift+J

// البحث في كل الملفات
Ctrl+Shift+F

// الانتقال لملف معين
Ctrl+P  (في Sources)
نصائح احترافية:
  • $0 في Console = آخر عنصر حددته في Elements
  • $(selector) = document.querySelector() مختصرة
  • copy(object) = ينسخ الكائن للـ clipboard
  • monitorEvents(el) = يراقب كل أحداث عنصر معين
  • Local Overrides: تعديل ملفات الموقع محلياً بشكل دائم
🔮 Console Shortcuts
// في Console مباشرة:

$('h1')          // أول h1 في الصفحة
$$('p')          // كل الفقرات (مصفوفة)
$0             // آخر عنصر محدد في Elements
copy($0)       // نسخ العنصر للـ clipboard
clear()        // مسح الـ console
keys({a:1,b:2}) // ['a','b']

📝 التمارين العملية Exercises

تمرين 1: فتح DevTools

ما هي ثلاث طرق لفتح DevTools في Chrome؟

الحل:

1. الضغط على F12
2. Ctrl+Shift+I
3. نقر يمين على أي عنصر في الصفحة → Inspect

تمرين 2: console.table

اكتب كوداً يستخدم console.table لعرض مصفوفة من 3 طلاب لكل منهم اسم ودرجة.

الحل: 
const students = [
  { name: 'أحمد',  score: 92 },
  { name: 'سارة',  score: 88 },
  { name: 'محمد',  score: 75 },
];
console.table(students);
تمرين 3: قياس الوقت

اكتب كوداً يقيس وقت تنفيذ حلقة تضرب الأرقام من 1 إلى 1000.

الحل: 
console.time('الضرب');
let result = 1;
for (let i = 1; i <= 1000; i++) {
  result *= i;
}
console.timeEnd('الضرب');
تمرين 4: Debugger

أضف debugger لهذه الدالة لتوقفها عند حساب الـ total:

function getTotal(prices) {
  return prices.reduce((a, b) => a + b, 0);
}
الحل: 
function getTotal(prices) {
  debugger;  ← يوقف التنفيذ هنا
  return prices.reduce((a, b) => a + b, 0);
}
تمرين 5: اختصارات DevTools

ما الاختصار لـ: (أ) تبديل وضع الجوال، (ب) البحث في كل الملفات، (ج) فتح Console مباشرة؟

الحل:

(أ) وضع الجوال: Ctrl+Shift+M
(ب) البحث في الملفات: Ctrl+Shift+F
(ج) فتح Console: Ctrl+Shift+J

اختبر فهمك لـ DevTools!
