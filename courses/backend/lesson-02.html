.1 ما هو Node.js؟ What is Node.js?

Node.js هو بيئة تشغيل JavaScript خارج المتصفح — يعمل مباشرة على نظام التشغيل.

💡
القصة الكاملة

في البداية JavaScript كانت حبيسة المتصفح فقط. ثم في 2009 أخذ Ryan Dahl محرك V8 (من Chrome) وبنى حوله Node.js — فأصبح بإمكان JS أن تقرأ الملفات، تُنشئ خوادم، وتتحدث مع قواعد البيانات.

مميزات Node.js الأساسية:

  • Single-threaded + Event Loop: لا يُجمّد عند انتظار I/O — يُكمل ما يستطيع ثم يعود
  • Non-blocking I/O: قراءة الملف/قاعدة البيانات لا تُوقف تنفيذ الكود الآخر
  • npm Ecosystem: أكبر مستودع حزم برمجية في العالم (+2 مليون حزمة)
  • نفس اللغة: JavaScript للفرونت والباك — مطور واحد يُنجز المشروع كاملاً

.4 Node.js vs المتصفح Node vs Browser

JS هي نفسها — لكن البيئة مختلفة. ما يتوفر في المتصفح ليس بالضرورة في Node وبالعكس:

🌐 المتصفح
✓ window, document, DOM
✓ localStorage, sessionStorage
✓ fetch (Fetch API)
✓ alert, confirm
✗ قراءة ملفات النظام
✗ __dirname, __filename
✗ أوامر نظام التشغيل
🟢 Node.js
✓ require / module.exports
✓ __dirname, __filename
✓ fs (قراءة الملفات)
✓ process, os, path
✓ http (إنشاء خوادم)
✗ window, document, DOM
✗ alert, localStorage
🌐 متغيرات Node.js العالمية
// __dirname: مسار المجلد الحالي
console.log(__dirname);
// → /home/user/myproject

// __filename: مسار الملف الحالي
console.log(__filename);
// → /home/user/myproject/app.js

// process: معلومات البيئة
process.env.NODE_ENV;    // 'development' أو 'production'
process.argv;            // معاملات سطر الأوامر
process.exit(0);         // إنهاء السكريبت

.5 نظام الوحدات (Modules) Module System

Node.js يُقسّم الكود إلى وحدات قابلة للاستيراد — كل ملف وحدة مستقلة:

📦 CommonJS — النظام الافتراضي في Node
// ── math.js ──────────────────────────
function add(a, b) { return a + b; }
function multiply(a, b) { return a * b; }

module.exports = { add, multiply };

// ── app.js ────────────────────────────
const { add, multiply } = require('./math');
console.log(add(5, 3));      // 8
console.log(multiply(5, 3)); // 15
🔷 ESM — النظام الحديث (في package.json أضف "type": "module")
// ── math.mjs ──────────────────────────
export function add(a, b) { return a + b; }
export const PI = 3.14159;

// ── app.mjs ───────────────────────────
import { add, PI } from './math.mjs';
console.log(add(1, PI)); // 4.14159

الوحدات المدمجة (Built-in) — تأتي مع Node.js بدون تثبيت:

console.log(fullPath); // /project/data/users.json

.6 npm و package.json npm & package.json

npm (Node Package Manager) هو مدير الحزم لـ Node.js. package.json هو "بطاقة هوية" مشروعك:

🚀 إنشاء مشروع جديد
# أنشئ مجلداً وافتح الطرفية فيه
mkdir my-backend
cd my-backend

# ابدأ مشروع npm (يُنشئ package.json)
npm init -y
# ↑ -y تعني "نعم لكل الأسئلة" — يُنشئ package.json فوراً

الـ package.json الناتج:

{
  "name": "my-backend",              ← اسم المشروع
  "version": "1.0.0",            ← إصدار المشروع
  "description": "",
  "main": "index.js",             ← نقطة الدخول
  "scripts": {
    "start": "node index.js",       ← npm start
    "dev": "nodemon index.js"      ← npm run dev
  },
  "dependencies": {
    "express": "^4.18.2"          ← حزم الإنتاج
  },
  "devDependencies": {
    "nodemon": "^3.0.2"          ← حزم التطوير فقط
  },
  "author": "",
  "license": "ISC"
}
npm scripts: أوامر مختصرة — بدل كتابة node index.js كل مرة، اكتب npm start. يمكن تعريف أي أمر تحت أي اسم تريده.

.8 nodemon — التطوير السريع nodemon

المشكلة: كل مرة تُعدّل الكود يجب إيقاف الخادم وإعادة تشغيله. الحل: nodemon يراقب التغييرات ويُعيد التشغيل تلقائياً.

⚡ إعداد nodemon
# تثبيت nodemon كـ devDependency
npm install nodemon --save-dev

# في package.json أضف script:
{
  "scripts": {
    "start": "node index.js",
    "dev":   "nodemon index.js"
  }
}

# الآن شغّل المشروع بـ:
npm run dev
# → يُعيد التشغيل تلقائياً عند أي تعديل!

# أو مباشرة بدون تثبيت عالمي:
npx nodemon index.js
⚙️ إعداد nodemon.json (اختياري)
{
  "watch": ["src/"],       // راقب مجلد src فقط
  "ext": "js,json",        // امتدادات تُفعّل الإعادة
  "ignore": ["*.test.js"], // تجاهل ملفات الاختبار
  "delay": "1000"          // انتظر 1s قبل الإعادة
}
في Deployment (الإنتاج) لا تستخدم nodemon — استخدم PM2 الذي يُديرالعمليات ويُعيد تشغيلها عند الأعطال.

.9 تجربة حية في الطرفية Live Terminal Demo

هذا ما تبدو عليه جلسة عمل Node.js كاملة — من إنشاء المشروع لتشغيله:

🟢 جلسة Node.js — من الصفر للتشغيل Terminal 
// index.js
const http = require('http');

const server = http.createServer(
  (req, res) => {
    res.writeHead(
      200,
      { 'Content-Type': 'text/plain' }
    );
    res.end('مرحباً من Node.js! 🚀');
  }
);

server.listen(3000, () => {
  console.log(
    '✓ Server on port 3000'
  );
});

// package.json
{
  "name": "my-server",
  "scripts": {
    "dev": "nodemon index.js"
  },
  "devDependencies": {
    "nodemon": "^3.0.2"
  }
}

تمرين 2: وحدة مخصصة

أنشئ ملف calculator.js يُصدّر الدوال: add، subtract، multiply، divide. ثم استخدمها في app.js وتحقق من عمل كل دالة.

الحل: 
// calculator.js
function add(a, b) { return a + b; }
function subtract(a, b) { return a - b; }
function multiply(a, b) { return a * b; }
function divide(a, b) {
  if (b === 0) throw new Error('القسمة على صفر!');
  return a / b;
}
module.exports = { add, subtract, multiply, divide };

// app.js
const calc = require('./calculator');
console.log(calc.add(10, 5));       // 15
console.log(calc.subtract(10, 5));  // 5
console.log(calc.multiply(10, 5));  // 50
console.log(calc.divide(10, 5));    // 2
تمرين 3: قراءة ملف

أنشئ ملف names.txt يحتوي على أسماء (كل اسم في سطر). ثم اكتب read.js يقرأ الملف ويطبع عدد الأسماء وكل اسم بشكل منفصل.

الحل: 
// read.js
const fs = require('fs');

const content = fs.readFileSync('names.txt', 'utf8');
const names = content.trim().split('\n');

console.log('عدد الأسماء:', names.length);
names.forEach((name, i) => {
  console.log(`${i + 1}. ${name}`);
});

تمرين 5: أول HTTP Server

أنشئ server.js بوحدة http المدمجة يُعيد:
- على المسار /: رد نصي "الصفحة الرئيسية"
- على المسار /about: رد نصي "صفحة عنا"
- على أي مسار آخر: كود 404 ورسالة "الصفحة غير موجودة"

الحل: 
const http = require('http');

const server = http.createServer((req, res) => {
  res.setHeader('Content-Type', 'text/plain; charset=utf-8');

  if (req.url === '/') {
    res.writeHead(200);
    res.end('الصفحة الرئيسية 🏠');
  } else if (req.url === '/about') {
    res.writeHead(200);
    res.end('صفحة عنا ℹ️');
  } else {
    res.writeHead(404);
    res.end('الصفحة غير موجودة 404');
  }
});

server.listen(3000, () => {
  console.log('✓ Server on http://localhost:3000');
});
